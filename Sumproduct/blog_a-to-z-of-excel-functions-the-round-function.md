_Welcome back to our regular A to Z of Excel Functions blog. Today we look at the **ROUND** function._

**The ROUND function**

![](https://sumproduct.com/wp-content/uploads/2025/05/8a13f1f5e893e3e6d9fa7e64f36457ea-1.jpg)

If only my brother knew what a **ROUND** was… The **ROUND** function rounds a number to a specified number of digits. It employs the following syntax to operate:

**ROUND(number, num\_digits)**

The **ROUND** function has the following arguments:

*   **number**: this is required and represents the value to be rounded
*   **num\_digits**: also required, this is the number of digits to which you wish to round the **number**.

It should be further noted that:

*   **ROUND** rounds to the nearest number, but be careful of midpoints. For example, **ROUND(3.5,0)** equals 4, but **ROUND(-3.5,0)** equals -4, not -3 _(see image, below)_

![](https://sumproduct.com/wp-content/uploads/2025/05/2c491a8d47d3ff2f4b2609064882b8eb-1.jpg)

*   the **number** can be a number, a cell reference or a named range
*   this function operates on the actual value in the cell and rounds it so it is consistent with the value that is displayed
*   using this function will change the underlying value and not just the value that is displayed
*   you can use the **[ROUNDDOWN](https://www.sumproduct.com/blog/article/a-to-z-of-excel-functions/the-rounddown-function)
    ** function if you always want the **number** rounded down to the nearest integer
*   you can use the **[ROUNDUP](https://www.sumproduct.com/blog/article/a-to-z-of-excel-functions/the-roundup-function)
    ** function if you always want the **number** rounded up to the nearest integer
*   you can use the **[FLOOR.MATH](https://www.sumproduct.com/blog/article/a-to-z-of-excel-functions/the-floor-dot-math-function)
    ** function to round down to the nearest integer or significant figure
*   you can use the **[CEILING.MATH](https://www.sumproduct.com/blog/article/a-to-z-of-excel-functions/the-ceiling-function)
    ** function to round up to the nearest integer or significant figure
*   you can use the **[MROUND](https://www.sumproduct.com/blog/article/a-to-z-of-excel-functions/the-mround-function)
    ** function to round to a desired multiple
*   you can use the **[EVEN](https://www.sumproduct.com/blog/article/a-to-z-of-excel-functions/the-even-function)
    ** function to round up to the nearest even integer
*   you can use the **[ODD](https://www.sumproduct.com/blog/article/a-to-z-of-excel-functions/the-odd-function)
    ** function to round up to the nearest odd integer
*   the equivalent VBA function is **VBA.ROUND**.
