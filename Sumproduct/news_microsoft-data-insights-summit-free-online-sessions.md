Microsoft Data Insights Summit – Free Online Sessions
=====================================================

18 March 2016

Microsoft’s Data Insight Summit, the inaugural Power BI and Excel user conference, takes place on Tue 22 March to Wed 23 March in Redmond (US West Coast time). We’ll actually be there – but if you don’t already have a ticket, you can still attend.

Microsoft has announced keynote and select sessions will be streamed live for free allowing the rest of us participate virtually in this two-day event aimed at business analysts around the world.

We will be reporting back about the event in April’s newsletter, but if you want to be involved check out the [Microsoft website](https://blogs.office.com/2016/03/10/data-insights-summit-free-online-excel-and-power-bi-sessions/)
 which provides more information on how to become involved.
