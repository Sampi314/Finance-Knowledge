New Features for Excel February 2024
====================================

28 February 2024

![](<Base64-Image-Removed>)

This month sees the ‘Chart data’ task pane and the ability to use images and Data Types in your PivotTables both now available in Excel for the web. The new default theme for Office is Generally Available in Excel for Windows and Mac too.

You can check out the full list here:

**_Excel for the web_**

*   ‘Chart data’ task pane
*   Images and Data Types in PivotTables

**_Excel for Windows_**

*   New default theme for Office

**_Excel for Mac_**

*   New default theme for Office.
