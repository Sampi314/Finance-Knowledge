New Features for Excel July 2025
================================

![](<Base64-Image-Removed>)

July sees several new features in Excel that enhance user productivity and collaboration across platforms.  Here is this month’s list:

**_Excel for Windows, Mac and web_**

*   Compatibility versions

**_Excel for Windows and Mac_**

*   PivotTable Auto Refresh (Insiders)

**_Excel for Windows_**

*   ‘Get Data’ dialog and OneLake catalog (Insiders)

**_Excel for Mac_**

*   View multiple panes, sheets or workbooks side by side

**_Excel for web_**

*   Power Query refresh and data source settings for authenticated data sources
*   Request more access for Enterprise customers.
