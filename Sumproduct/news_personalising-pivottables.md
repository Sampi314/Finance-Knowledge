Personalising PivotTables
=========================

16 May 2017

The latest Office 365 update provides a ‘pivotal’ new Excel feature. Now, you can finally personalise the default layout for your PivotTables. It’s been a popular request over at [Excel UserVoice](https://excel.uservoice.com/forums/304921-excel-for-windows-desktop-application/suggestions/10535790-allow-users-to-set-choose-defaults-via-settings)
 for some time and now it’s finally happened.

There are two ways you can adjust the layout settings for the PivotTable defaults, either via the newly added Edit Default Layout button under the File menu, or else through the PivotTable Options dialog box (pictured).

We’ll provide more details on this oft-asked for feature in June’s newsletter. If you haven’t already subscribed, feel free to add your name in the link at the bottom of any **SumProduct** web page.
