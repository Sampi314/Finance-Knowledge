Final Friday Fix: November 2019 Challenge
=========================================

29 November 2019

_On the final Friday of each month, we’re going to set an Excel challenge for you to puzzle over so that you can get your “Excel fix”. Challenge your office colleagues to see who can solve the puzzle quickest. There are no prizes at this stage: you’re playing for your bragging rights only!_

This month’s Final Friday Fix is about creating a universal option list for the whole workbook. For this challenge, we want to create a dashboard to track the accommodation and catering services for one of our clients:

*   A Dashboard worksheet shows which customer group is staying in which room and on what date
*   A Catering sheet which displays the number of guests in each room and, by date, who will be using catering services. (This is simply generated by an **INDEX MATCH** formula from their data.

The client wants to be able to view by whole month, 14-day or 7-day in both reports at any time. Similar to the screenshot below, we just need to input the ‘Start Date’, and click on the option to view as ‘7 days’, so that the list will be generated for the whole week to see which group of customers will be coming:

![](https://sumproduct.com/wp-content/uploads/2025/05/e774d10cbbb9450fc45efbe51abdf434-317.jpg)

whilst in the Catering sheet, the list is also updated accordingly, so that we may see how many we need to prepare catering for:

![](https://sumproduct.com/wp-content/uploads/2025/05/f32e5a15e2cf9c3e4d2d058458ce054d-321-1.jpg)

If we choose ’14 days’, the list becomes longer:

![](https://sumproduct.com/wp-content/uploads/2025/05/f1140ff857fc3b6f5f97a6a24f4a6fc7-306.jpg)

with a corresponding effect in the Catering sheet:

![](https://sumproduct.com/wp-content/uploads/2025/05/72aa864d2854c6fefb1083fba0ab5792-281.jpg)

Similarly, the list will be updated for the whole month if we choose to view ‘Whole Month’:

![](https://sumproduct.com/wp-content/uploads/2025/05/36776d1da4d05b45bb5a5d09375f407c-241.jpg)

This kind of viewing option list is like a filter, which can be applied to various problems when you work with Excel. You can use this [Excel file](https://sumproduct.com/assets/blog-pictures/2019/challenges-fff-mmm/nov-19/fff/final-friday-fix-nov-2019-start.xlsx)
 as a starter.

Sound easy? Then why not have a go? We’ll publish one solution in Monday’s blog. Have a great weekend in the meantime!
