New Features for Excel September 2023
=====================================

27 September 2023

![](<Base64-Image-Removed>)

This month sees Microsoft share that Python in Excel is now available to all Enterprise, Business, Education, Family and Personal users running Beta Channel on Windows, starting with build 16.0.16818.20000 – and then to the other platforms at a later date. Also, the Python Editor from Excel Labs and Copilot support for Python in Excel are formally announced too. I feel almost AutoComplete…

You can check out the full list here:

**_Excel for Windows_**

*   Python in Excel (Insiders Beta)
*   Python Editor from Excel Labs (Insiders Beta)
*   Copilot Support for Python in Excel (available in Preview later this year)
*   Data Validation AutoComplete

**_Excel for Mac_**

*   Data Validation AutoComplete (Insiders).
