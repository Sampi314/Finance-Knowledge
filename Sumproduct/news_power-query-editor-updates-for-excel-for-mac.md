Power Query Editor Updates for Excel for Mac
============================================

1 March 2023

You can now author queries from local files, SharePoint, OData and various other sources with the Power Query Editor in Excel for Mac.

![](https://sumproduct.com/wp-content/uploads/2025/05/4e201cb42fedfbadd8c6b7344b877a8e.jpg)

The ability to refresh data was introduced into Excel for Mac back in 2020, with the ability to import (some) data in 2021. But now, you can clean / shape the data with numerous transformations now available in Apple Land.

To access the new features, on the Data tab, click the ‘Get Data (Power Query)’ button:

![](https://sumproduct.com/wp-content/uploads/2025/05/48f06068b5e8adcc2a9e161b0e16ef5a.jpg)

Then, click on ‘Launch Power Query Editor’ to open the Query Editor.

![](https://sumproduct.com/wp-content/uploads/2025/05/c3204803685e9129132139e22e0394b0.jpg)

You can shape and transform your data using the Query Editor as you would in Excel for Windows. When you’re done, simply click the ‘Close & Load’ button on the Home tab.

![](https://sumproduct.com/wp-content/uploads/2025/05/20200bcf4b95de918385e48f819f3008.jpg)

And that’s it!

![](https://sumproduct.com/wp-content/uploads/2025/05/c8e6b7616f875f70430602bec6132ada.jpg)

Supported data sources include Blank Query, Blank Table, CSV, JSON files, local tables and ranges, Microsoft SQL Server, OData, SharePoint, SharePoint Folders, SharePoint Lists, text, XLSX and XML.

You should note that you may also access the Query Editor from the data import flow by clicking the ‘Get Data (Power Query)’ button, choosing a data source, and clicking the ‘Transform Data’ button:

![](https://sumproduct.com/wp-content/uploads/2025/05/e871093cd3454af323a3bee7614ee5ad.jpg)

This feature is Generally Available in Excel for Microsoft 365 for Mac version 16.69 (23010700) or later.
