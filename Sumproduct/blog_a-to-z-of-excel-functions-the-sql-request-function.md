A to Z of Excel Functions: The SQL.REQUEST Function
===================================================

_Welcome back to our regular A to Z of Excel Functions blog.  Today we look at the **SQL.REQUEST** function._

**The SQL.REQUEST function**

This function is mentioned for completeness.  **SQL.REQUEST** connects with an external data source and runs a query from within the Excel worksheet.  The result is then returned as an array without the requirement for VBA code. 

This function was available in Excel 2003 and earlier versions after installing the XLODBC.xla add-in, but it has not been supported in Excel since Excel 2010 (no confirmation is available for Excel 2007 at this time).  This add-in was available from the Microsoft Office website.  It does not appear to have been officially deprecated.

If this function is not available in a supported version of Excel, you must install the Microsoft Excel ODBC add-in program (XLODBC.xla).  In the past, you could install this add-in from the Microsoft Office Web site.  Nowadays, it has been replaced with **Data -> Get External Data** on the Ribbon.
