A to Z of Excel Functions: The SUBSTITUTE Function
==================================================

_Welcome back to our regular A to Z of Excel Functions blog.  Today we look at the **SUBSTITUTE** function._

**The SUBSTITUTE function**

![](https://sumproduct.com/wp-content/uploads/2026/02/image-73.png)

The **SUBSTITUTE** function quite literally substitutes new text for old text in a text string.  You may use the **SUBSTITUTE** function when you want to replace specific text in a text string; alternatively, you can use **[REPLACE](https://sumproduct.com/blog/a-to-z-of-excel-functions-the-replace-function/)
** when you wish to replace any text that occurs in a specific location in a text string.

**SUBSTITUTE** has the following syntax:

> **SUBSTITUTE(text, old\_text, new\_text\[, instance\_number\])**

The **SUBSTITUTE** function has the following arguments:

*   **text:** this is required and is the text or reference to a cell containing text where you wish to substitute characters
*   **old\_text:** this is also required and represents thetextyou wish to replace
*   **new\_text:** again required, this is the text that you want to replace the **old\_text** with
*   **instance\_number:** this argument is optional.  It specifies which occurrence of **old\_text** you want to replace with **new\_text**.  If you specify **instance\_number**, only that instance of **old\_text** is
*   replaced.  Otherwise, every occurrence of **old\_text** in **text** is changed to **new\_text**.

Whilst text substrings should be in quotation marks, values do not need to be.
