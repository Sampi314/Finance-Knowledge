Snakes on a Plane
=================

Well, a spreadsheet plane anyway…

![](<Base64-Image-Removed>)

SumProduct is proud to announce the publication of yet another new book, this time under the Microsoft Press banner.  _Python in Excel_ not only explains how to use the fully integrated Python in Excel, it also demonstrates through examples how to enlist the help of AI and Power Query to conquer your data.  There is also an extensive guide to Python code from an Excel perspective, if only to make sure you keep an eye on those AI outputs!

Written by our very own resident Python and Power Query expert **Kathryn Newitt** with contributions from a cast of many including MVPs **Liam Bastick** and **Ilgar Zarbaliyev**, our latest offering is available now in eBook and printed formats from all good bookshops (including Amazon and Microsoft Press) and all the other ones too.
