Power BI Blog: Microsoft Business Applications Summit 2019
==========================================================

13 June 2019

_This week, we’ve got a monster tip for you – if you’re not aware, we’ve just had the Microsoft Business Applications Summit 2019, held in Georgia, Atlanta!_

While we weren’t able to attend, with the power of the internet, we’re able to catch up on all the latest going on, and you are too! Make sure you check out the official [Microsoft Power BI YouTube channel](https://www.youtube.com/user/mspowerbi/featured)
, where you can see the latest updates available from MBAS 2019 with key sessions being recorded and uploaded for the public to see.

![](<Base64-Image-Removed>)

At the time of writing, there have been 37 new videos from MBAS that have been posted up on the channel, likely with more to come as the event wraps to a close. Make sure you check it out to catch all the new news. If YouTube isn’t your thing, we’ll be looking through and reporting back on key updates in the following weeks, so check back here every Thursday to find out what’s new and what’s up. See you next week!
