New Features for Excel April 2025
=================================

![](<Base64-Image-Removed>)

This month introduces several new features across platforms, including value tokens, that infernal Dark Mode and the ability to create Tables with Copilot in Excel (yes, **CTRL** + **T** was too difficult).

Here is this month’s list:

**_Excel for Windows_**

*   Introducing value tokens in Excel (Insiders)
*   Dark Mode in Excel
*   ActiveX disabled by default (Insiders)

**_Excel for Windows and web_**

*   Table creation with Copilot in Excel
*   Upload phone images with Copilot

**_Excel for Windows and Mac_**

*   View multiple worksheets side by side (Insiders).
