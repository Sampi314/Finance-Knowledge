2 September 2022

_Welcome back to this week’s Charts and Dashboards blog series. This week, we revisit creating attention grabbing chart titles._

One way to make the chart more unique is to emphasize it with a text box chart title which is straightforward to create a text box in Excel. We covered this in Chart Titles as a Text Box.

In our [original blog](https://sumproduct.com/blog/charts-and-dashboards-chart-titles-as-a-text-box/)
, we had a line chart with a chart title in cell **D29**:

![](https://sumproduct.com/wp-content/uploads/2025/05/1a1c6c21ffc0b76081701df6465b8f84.jpg)

We created a text box by inserting a rectangle and linking it to **D29**. To attach the chart title text box to the chart, we clicked on the text box, held the **CTRL** key and clicked on the chart, then right-clicked, to Group.

![](https://sumproduct.com/wp-content/uploads/2025/05/87ea9cadf81ec868a3a17e6d9a96a52a.jpg)

This ensured that the chart and the text box stuck to each other and moved together.

![](https://sumproduct.com/wp-content/uploads/2025/05/d8733ea666e15c519f95d315fb19b2ae.jpg)

We could have approached this another way. We will start with the same chart. This time however, we select the chart _first_. As before, we select a rectangle from the Shapes dropdown in the Illustrations section of the Insert tab:

![](https://sumproduct.com/wp-content/uploads/2025/05/1f6f90567b0f5361dd1cfdf10db2f1db.jpg)

This way, we create the rectangle inside the chart.

![](https://sumproduct.com/wp-content/uploads/2025/05/83c3e163865cc87eafda40751babe2b0.jpg)

We can then link the rectangle to the cell with the title:

![](https://sumproduct.com/wp-content/uploads/2025/05/729cd5a9e64d5a2816770df8e6306d87.jpg)

We can then resize the rectangle. When we move the chart, the rectangle moves with it, with no requirement to group the objects because it is already part of the chart.

![](https://sumproduct.com/wp-content/uploads/2025/05/8b6303336cdf39855b2579cd06ac63e4.jpg)

That’s it for this week. Come back next week for more Charts and Dashboards tips.
