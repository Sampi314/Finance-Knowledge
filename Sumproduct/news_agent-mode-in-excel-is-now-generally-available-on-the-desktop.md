Agent Mode in Excel is Now Generally Available on the Desktop
=============================================================

It’s been a great start to 2026.  After the recent news that Power Query is now “full on” for the web, Agent Mode in Excel, part of Microsoft 365 Copilot, is now Generally Available on Windows, with Mac rolling out very shortly.  This extends access beyond Excel for the web, which was launched just last month[](https://techcommunity.microsoft.com/blog/excelblog/agent-mode-in-excel-is-now-generally-available-on-excel-for-web/4476092)
.

Since the initial public Preview, Microsoft has expanded its availability, added web-grounded search and introduced a new multi-model reasoning system that allows customers to choose between OpenAI and Anthropic models.  They claim the improvements don’t stop there.  Microsoft states that they have significantly improved task success, performance and reliability across core Excel scenarios, including workbook creation, formula repair and chart and PivotTable generation.

The aim of Agent Mode is to take AI to the next level in Excel.  With this release, that capability is now available in the desktop versions of Excel customers.

![](https://sumproduct.com/wp-content/uploads/2026/01/image-109.png)

Agent Mode in Excel is Generally Available today across Excel for web, Windows and Mac for commercial Microsoft 365 Copilot licences and Microsoft 365 Personal, Family and Premium subscribers.

It should be noted that Personal and Family subscriptions use an “AI credit model” and Agent Mode in Excel is not yet available to customers in the EU or UK.
