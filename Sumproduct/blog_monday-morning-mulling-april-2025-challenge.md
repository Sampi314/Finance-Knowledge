Monday Morning Mulling: April 2025 Challenge
============================================

_On the final Friday of each month, we set an Excel / Power Pivot / Power Query / Power BI problem for you to puzzle over for the weekend.  On the Monday, we publish a solution.  If you think there is an alternative answer, feel free to email us.  We’ll feel free to ignore you._

**_The Challenge_**

Last Friday’s challenge invited to create a searchable data validated drop down box using a method that would work in a perpetual licence version (_e.g_. Excel 2019) of Excel. 

A list was provided, containing a set of values:

![](https://sumproduct.com/wp-content/uploads/2025/06/image-71.png)

You could download the original question file [here](https://www.sumproduct.com/assets/blog-pictures/2022/fff-mmm/2025/Apr/sp-searchable-dropdown-box---challenge.xlsx)
. 

Remember, these were the guidelines:

*   the solution must work in a perpetual licence version of Excel (rather than Excel 365 or Excel online)
*   you cannot use dynamic arrays
*   you cannot use VBA, Office Scripts or any other programming code.

**_Suggested Solution_**

You can find our Excel file [here](https://www.sumproduct.com/assets/blog-pictures/2022/fff-mmm/2025/Apr/sp-searchable-dropdown-box---suggested-solution.xlsx)
. which shows our suggested solution.  The steps are detailed below.

This Final Friday Fix is not our hardest, but sometimes simple solutions do not occur to us.  It’s a simple, yet valuable, addition to your Excel repertoire.  To start, select any cell on the list:

![](https://sumproduct.com/wp-content/uploads/2025/06/image-72.png)

Go to the ribbon, select ‘Insert’ tab and click the ‘PivotTable’ icon:

![](https://sumproduct.com/wp-content/uploads/2025/06/image-73.png)

You may also click on the ‘PivotTable’ dropdown menu (blue) and then select ‘From Table/Range’ to arrive at the same next step:

![](https://sumproduct.com/wp-content/uploads/2025/06/image-74.png)

This will open the ‘PivotTable from table or range’ dialog.  Click ‘OK’:

![](https://sumproduct.com/wp-content/uploads/2025/06/image-75.png)

Focusing on the ‘PivotTable Fields’ pane, drag the **List**field down into the Filters area:

![](<Base64-Image-Removed>)

Then, apply proper formatting.  It will create this, which is the successful completion of this month’s challenge:

![](<Base64-Image-Removed>)

As an addition, you may utilise this trick to create a searchable box for a slicer to enhance user experience:

![](<Base64-Image-Removed>)

The dropdown menu will open in front of the slicer:

![](<Base64-Image-Removed>)

This concludes the Final Friday Fix for this month.  How did you do?
