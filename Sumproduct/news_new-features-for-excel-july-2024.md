New Features for Excel July 2024
================================

New Features for Excel July 2024
================================

31 July 2024

![](https://sumproduct.com/wp-content/uploads/2025/05/93379fce9d336d3e7879563f17cbc0af.jpg)

This month sees you able to use Copilot in Excel to unlock insights from data, work on data ranges resembling tables and prompt for step-by-step instructions including formula examples. But that’s not all. Pretty much half of our August newsletter is reproduced below (if you haven’t subscribed already, perhaps you should)!

The full list is as follows:

**_Excel for the web_**

*   Copilot in Excel: unlocking insights from data
*   Copilot in Excel: expanded data structure support and comprehensive answers
*   Modernised Excel grid
*   Synchronise Forms data to Excel

**_Excel for Windows_**

*   Copilot in Excel: unlocking insights from data
*   Copilot in Excel: expanded data structure support and comprehensive answers
*   Checkboxes in cells
*   **TRANSLATE** and **DETECTLANGUAGE** functions (Insiders)
*   Python Editor (Insiders)
*   Python in Excel: additional library support (Insiders)

**_Excel for Mac_**

*   Copilot in Excel: unlocking insights from data
*   Copilot in Excel: expanded data structure support and comprehensive answers
*   Checkboxes in cells
*   **TRANSLATE** and **DETECTLANGUAGE** functions (Insiders)
*   Refresh data from Power Query’s ‘From Folder’ connector.

As always, we’ll be detailing these updates in our September newsletter. Please remember we have virtual / online training in Excel which you can find out more about [here](https://www.sumproduct.com/courses/excel-tips-tricks)
