Power BI Blog: Power Query Editing in the Web
=============================================

_Welcome back to this week’s edition of the Power BI blog series.  This week, we review Power Query editing in the web for import models with Power BI._

At last – one of the most requested capabilities for semantic model editing on the web: Power Query editing for import models.  It is now possible to get data, transform data and refresh your import models entirely on the web, with no need for Power BI Desktop.

You will now be able to add new import tables to your semantic models directly on the web using the familiar Power Query ‘Get Data’ experience.  Simply select ‘Get data from the Home’ tab of the ribbon to choose your connector and bring in new data to your semantic model.

![](<Base64-Image-Removed>)

You will also be able to shape your data with the full Power Query editor, now available on the web.  Simply select ‘Transform data from the Home’ tab of the Ribbon to launch the modern Power Query user interface.

![](<Base64-Image-Removed>)

You may refresh both schema and data right from the web editing experience.  Simply select ‘Refresh’ from the ‘Home’ tab while editing your semantic model.
