_Welcome back to our regular A to Z of Excel Functions blog. Today, we look at the **QUOTIENT** function._

**The QUOTIENT function**

![](https://sumproduct.com/wp-content/uploads/2025/05/b0acebabc48fbe720b952e5d7e0c8c84-1.jpg)

This function returns the integer portion of a division and should be used when you wish to discard the remainder of a division. This is not the same as the **[INT](https://www.sumproduct.com/blog/article/a-to-z-of-excel-functions/the-int-function)
**function which rounds a number down to the nearest integer. **QUOTIENT** rounds towards zero, _viz._

![](https://sumproduct.com/wp-content/uploads/2025/05/b80d280b7b6fd684317e5936d98315ca-1.jpg)

If you want to divide numerical values, you should use the “**/**” operator as there isn’t a **DIVIDE** function in Excel (although there is in **[DAX](https://www.sumproduct.com/blog/article/power-pivot-principles/the-a-to-z-of-dax-functions-divide)
**). For example, to divide 5 by 2, you would type =5/2 into a cell, which returns 2.5. The **QUOTIENT** function for these same numbers, **\=QUOTIENT(5,2)**, returns 2 since **QUOTIENT** doesn’t return a remainder.

**QUOTIENT** employs the following syntax to operate:

**QUOTIENT(numerator, denominator)**

The **QUOTIENT**function has the following arguments:

*   **numerator:** this is required and represents the dividend
*   **denominator:** this is also required and denotes the divisor.

It should be noted that if either argument is non-numerical, **QUOTIENT** returns the _#NUM!_ error value.
