2017’s First Updates for Get & Transform / Power Query
======================================================

10 February 2017

After a quiet time of late, Microsoft has finally got busy again. Excel 2016’s [Get & Transform](https://support.office.com/en-US/article/Get-Transform-in-Excel-2016-881c63c6-37c5-4ca2-b616-59e18d75b4de)
 / Excel 2010’s or Excel 2013’s [Power Query for Excel](https://www.microsoft.com/en-us/download/details.aspx?id=39379)
 add-in have been updated.

In total, six new data transformation and connectivity features have been announced. For Excel 2016, they are available as part of an [Office 365 subscription](https://products.office.com/en-us/buy/compare-microsoft-office-products)
. For earlier versions, you simply need to download the latest version of the add-in (see link above).

These updates include the following new or improved data connectivity and transformation features:

*   New OLE DB connector
*   Enhanced “Combine Binaries” experience when importing from any folder
*   Maximize / Restore buttons in the Navigator and Query Dependencies dialogs
*   Support for percentage data type
*   Improved “Function Authoring” experience
*   Improved performance for OData connector.

You can read more [here](https://blogs.office.com/2017/02/08/january-2017-updates-for-get-transform-in-excel-2016-and-the-power-query-add-in/)
. We’ll also detail these updates in our next newsletter too (if you aren’t already a subscriber, feel free to join by entering your details at the foot of any SumProduct web page). If you want to find out even more about Power Query, check out our [Wednesday blogs](https://www.sumproduct.com/blog/article/power-query-blogs/power-query-models-have-relationship-issues-too)
 and / or consider one of our [\>upcoming training courses](https://www.sumproduct.com/courses)
 too.
