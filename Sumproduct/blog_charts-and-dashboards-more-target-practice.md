Charts and Dashboards: More Target Practice
===========================================

18 June 2021

_Welcome back to our Charts and Dashboards blog series. This week, I look at another way to create a presentation of actual sales vs. target sales._

In [last week’s blog](https://sumproduct.com/blog/charts-and-dashboards-target-practice/)
, I looked at a way to represent the actual and target sales on a chart.

![](<Base64-Image-Removed>)

In April, my imaginary salespeople did much better (on the whole).

![](<Base64-Image-Removed>)

This time I am going to present the data in a slightly different way. I will start the same way as last week, by selecting my data and pressing **ALT + F1.**

![](<Base64-Image-Removed>)

A column chart is inserted in the sheet, and I can click on a bar from either of the data series and right click to access the menu. This time, I select ‘Change Series Chart Type’.

![](<Base64-Image-Removed>)

A dialog box appears:

![](<Base64-Image-Removed>)

I change the **Target** data series to ‘Line Chart with Markers’.

![](<Base64-Image-Removed>)

So far, this doesn’t appear to make anything clearer, but I have more changes to make. I click ‘OK’ and select the **Target** line, and right-click to access the ‘Format Data Series’ pane.

![](<Base64-Image-Removed>)

In the ‘Fill and Line’ tab, I select ‘No Line’.

![](<Base64-Image-Removed>)

This replaces the line with markers.

![](<Base64-Image-Removed>)

Next, I need to format the Markers on the ‘Format Data Series’ pane. I select ‘Built-in’ and pick the long horizontal line. I set the size of the line to 20.

![](<Base64-Image-Removed>)

This changes the look of my chart.

![](<Base64-Image-Removed>)

I can now change the colours in the ‘Format Data Series’ pane for each series and edit the chart title.

![](<Base64-Image-Removed>)

Now I can quickly see who is exceeding their targets. Time for a chat with Newbie!

That’s it for this week. Come back next week for more Charts and Dashboards tips.
