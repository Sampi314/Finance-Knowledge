Power BI Blog: Category Enhancements for New Card

24 April 2025

_Welcome back to this week’s edition of the Power BI blog series. This week, we look at category enhancements for the new card._

The new card visual is steadily progressing towards becoming the official card, which will replace both the current card and multi-row card. This update introduces additional enhancements for categories, previously known as small multiples.

Power BI is also introducing two \[2\] new styles for categories: table style and cards style. The new cards style will split each category into a separate card, providing a more organised and visually appealing layout. Each style option comes with its own settings, which you can find under **small multiples layout -> Layout -> Styles**.

![](<Base64-Image-Removed>)

Small multiple headers for the new card visual can be placed on the left, top, right or bottom.

![](<Base64-Image-Removed>)

The titles of the small multiple headers can be oriented to the left, top, right or bottom.

![](<Base64-Image-Removed>)

You can now apply conditional formatting to elements of your new card based on the logic of each category.

![](<Base64-Image-Removed>)

Remember, the new card visual is still in preview, to enable this new visual, follow these steps:

*   Go to **File -> Options and settings -> Options**
*   Navigate to Preview features
*   Check the option for new card visual.
