Power BI Blog: Using Display Folders
====================================

17 December 2020

_Welcome back to this week’s edition of the Power BI blog series. This week,_ _we look_ _at how to set display folders to organise your fields and measures._

This week, we look at how to use display folders. This is a quality-of-life tool that allows you to organise your fields and measures into folders. Display folders are especially useful when the report has many measures or fields. Segregating them into folders makes them much easier to find and manage.

To create a display folder, navigate to Model tab on the left side of the report.

![](https://sumproduct.com/wp-content/uploads/2025/05/e774d10cbbb9450fc45efbe51abdf434-88-1.jpg)

This is where you will see all the tables and their relationships.

![](https://sumproduct.com/wp-content/uploads/2025/05/f32e5a15e2cf9c3e4d2d058458ce054d-76-1.jpg)

Navigate to the Fields list on the right side of the report:

![](https://sumproduct.com/wp-content/uploads/2025/05/f1140ff857fc3b6f5f97a6a24f4a6fc7-69-1.jpg)

Our **CountryName** table current contains a mix of measures and fields. When users look at this table, it may appear a little messy with measures and fields displayed together. Let’s organise some of them into display folders.

To create a display folder, first select one measure or field that you want to group. We also must expand the Properties side panel:

![](https://sumproduct.com/wp-content/uploads/2025/05/72aa864d2854c6fefb1083fba0ab5792-61-1.jpg)

This will display several general properties for this measure. We create a display folder by giving it a name in the ‘Display folder’ dialog. In this case, we have named the folder ‘**Sales Measures’**. Yes, this blog has won several trophies for originality!

![](https://sumproduct.com/wp-content/uploads/2025/05/36776d1da4d05b45bb5a5d09375f407c-51-1.jpg)

To add more measures into this folder, we click and drag the measures into it.

![](https://sumproduct.com/wp-content/uploads/2025/05/23912d3b1671861e02bebcd5183f1607-45-1.jpg)

We can create several display folders in a single table, simply select another measure or field (in this case) and give it a ‘Display folder’ name.

![](https://sumproduct.com/wp-content/uploads/2025/05/6f49c288a0d88a66b427eaf4ece923d6-40-1.jpg)

Navigating back to the Report view, we can see that the fields and measures in the **CountryName** table are now nicely organised into folders:

![](<Base64-Image-Removed>)

Click on the down arrows next to the Display Folder icons to expand their contents:

![](<Base64-Image-Removed>)

There we have it, a simple guide on how to create display folders.
