11 Excel Trackers & Templates to help you Rock 2011
===================================================

[excel apps](https://chandoo.org/wp/category/excel-apps/)
 - 8 comments

  

We are just a few days away from 2011. New year always brings hope, cheer, joy and revitalizes us. So naturally many of us embark on journeys with new goals, resolutions, things to do.

Naturally, Excel can help us better manage the new year. In this post, I am featuring 11 templates so that you can have a rocking 2011.

1\. New Year Resolution Tracker Template
----------------------------------------

[![New Year Resolution Tracker Template - snapshot](https://chandoo.org/img/c/new-year-resolutions-excel.png)](http://chandoo.org/wp/2010/01/07/new-year-resolutions-template/ "New Year Resolution Tracker Template")

New year means new hopes, ambitions and resolutions. This handy template helps you monitor your new year resolutions and shows progress as you check items.

[Download This Template](http://chandoo.org/wp/2010/01/07/new-year-resolutions-template/ "New Year Resolution Tracker Template")

2\. Annual Goal Tracker Worksheet
---------------------------------

[![Annual Goal Tracker Worksheet - snapshot](https://chandoo.org/img/c/annual-goals-tracker-detail-th.png)](http://chandoo.org/wp/2010/03/01/employee-goal-tracker-excel/ "Annual Goal Tracker Worksheet")

An Excel tracker to help you keep track of annual goals at work. Shows progress as you mark goals as completed.

[Download This Template](http://chandoo.org/wp/2010/03/01/employee-goal-tracker-excel/ "Annual Goal Tracker Worksheet")

3\. Simple To-do List using Excel
---------------------------------

[![Simple To-do List using Excel - snapshot](https://chandoo.org/wp/wp-content/uploads/2008/10/removing-done-items-to-do-list-excel.png)](http://chandoo.org/wp/2008/10/01/simple-todo-list-using-excel/ "Simple To-do List using Excel")

A simple to-do list template to help you keep lists of items to be done and them remove them as you finish working.

[Download This Template](http://chandoo.org/wp/2008/10/01/simple-todo-list-using-excel/ "Simple To-do List using Excel")

4\. Holiday Gift List Tracker
-----------------------------

[![Holiday Gift List Tracker - snapshot](https://chandoo.org/img/a/christmas-gift-list-th.png)](http://chandoo.org/wp/2009/12/07/christmas-gift-list/ "Holiday Gift List Tracker")

Using this holiday gift list, you can keep track of overall budget, gifts bought and see where you are overspending. Pretty cool.

[Download This Template](http://chandoo.org/wp/2009/12/07/christmas-gift-list/ "Holiday Gift List Tracker")

5\. 2011 Yearly Calendar
------------------------

[![2011 Yearly Calendar - snapshot](https://img.chandoo.org/c/2011-calendar-month-january-excel.png)](http://chandoo.org/wp/2010/12/17/download-free-2011-calendar/ "2011 Yearly Calendar")

This calendar comes in 6 flavors – 1 mini calendar, 1 monthly calendar, 4 yearly calendars. All of which look great, work well. Go ahead and download now.

[Download This Template](http://chandoo.org/wp/2010/12/17/download-free-2011-calendar/ "2011 Yearly Calendar")

6\. Expense Tracker Templates
-----------------------------

[![7 Expense Tracker Templates - snapshot](https://chandoo.org/img/trackers/personal-expense-trackers-excel-downloads.png)](http://chandoo.org/wp/2010/07/16/download-expense-trackers/ "7 Expense Tracker Templates")

Our readers designed 7 beautiful Expenses trackers to help you monitor your household expenses. Each tracker has several valuable features. Go ahead and download.

[Download This Template](http://chandoo.org/wp/2010/07/16/download-expense-trackers/ "7 Expense Tracker Templates")

7\. Household Budget Tracker Template
-------------------------------------

[![Household Budget Tracker Template - snapshot](https://chandoo.org/wp/wp-content/uploads/2009/11/household-budget-spreadsheet-th.png)](http://chandoo.org/wp/2009/11/23/household-budget-spreadsheets/ "Household Budget Tracker Template")

Another great household budget spreadsheet. As you can guess, I love keeping track of expenses and spending as little as possible. Excel obviously helps me a better saver, investor and family-man.

[Download This Template](http://chandoo.org/wp/2009/11/23/household-budget-spreadsheets/ "Household Budget Tracker Template")

8\. Expense Tracker using Google Spreadsheet Forms
--------------------------------------------------

[![Expense Tracker using Google Spreadsheet Forms - snapshot](https://chandoo.org/wp/wp-content/uploads/2008/07/9-spending_by_category1.gif)](http://chandoo.org/wp/2008/07/14/use-google-docs-to-track-your-bucks-spreadsheet-forms-idea/ "Expense Tracker using Google Spreadsheet Forms")

You can simplify the process of expense tracking even more using Google forms. You can record expenses from anywhere using the interactive form and then monitor the expenses or analyze them right from Google spreadsheets.

[Download This Template](http://chandoo.org/wp/2008/07/14/use-google-docs-to-track-your-bucks-spreadsheet-forms-idea/ "Expense Tracker using Google Spreadsheet Forms")

9\. Calculate Expense Breakups using this Excel Template
--------------------------------------------------------

[![Calculate Expense Breakups using this Excel Template - snapshot](https://phd.blog.googlepages.com/sharing_trip_expenses_excel_sheet2.jpg)](http://chandoo.org/wp/2008/01/25/how-to-sharing-trip-expenses-using-excel/ "Calculate Expense Breakups using this Excel Template")

Whenever you do a group activity, the boring part of expense sharing comes up eventually. You can use this simple Excel template to find out expense breakup and how much each person owes to others.

[Download This Template](http://chandoo.org/wp/2008/01/25/how-to-sharing-trip-expenses-using-excel/ "Calculate Expense Breakups using this Excel Template")

----------------------------------

[![Mutual Fund Portfolio Tracker - snapshot](https://chandoo.org/img/c/mutual-fund-portfolio-tracker-excel-th.png)](http://chandoo.org/wp/2009/12/28/mutual-fund-tracker-excel/ "Mutual Fund Portfolio Tracker")

\[India Only\]. Mutual Funds are one of Indians’ favorite investment option. This excel tracker helps you monitor NAVs of your units.

[Download This Template](http://chandoo.org/wp/2009/12/28/mutual-fund-tracker-excel/ "Mutual Fund Portfolio Tracker")

11\. Stock Portfolio Tracker Template
-------------------------------------

[![Stock Portfolio Tracker Template - snapshot](https://chandoo.org/img/vba/stock-quotes-in-excel-macro.png)](http://chandoo.org/wp/2010/06/02/excel-stock-quotes/ "Stock Portfolio Tracker Template")

Many of us keep track of investment portfolios using Excel. Using this workbook, you can track latest prices of your stocks. This workbook uses macros to search web for the latest price of your company share.

[Download This Template](http://chandoo.org/wp/2010/06/02/excel-stock-quotes/ "Stock Portfolio Tracker Template")
