**Slope line** is very useful for spotting which values have changed from two sets. You can add a slope line to XY chart (scatter plot) using simple techniques. In this post, learn how to add them.

![add slope line to xy charts](https://chandoo.org/wp/wp-content/uploads/2020/04/add-slope-lines-to-excel-charts.png)

Steps for adding slope line to Excel XY charts
----------------------------------------------

Imagine you are analyzing app downloads for two weeks. Your data looks like this. You want to quickly **visualize which apps have been downloaded more _this week_**.

![sample data for excel chart with slope line](https://chandoo.org/wp/wp-content/uploads/2020/04/sample-data-slope-line.png)

1.  Select Last week & this week columns and create XY chart (scatter plot). You will get this.

![chart from last week & this week downloads data](https://chandoo.org/wp/wp-content/uploads/2020/04/chart-with-last-week-vs.-this-week-downloads.png)

2.  In a blank range, create slope series like this. The starting values will be 0,0. Ending values can be manually typed or you can use MAX() formula to generate them from data.

![data for slope series](https://chandoo.org/wp/wp-content/uploads/2020/04/dummy-data-for-slope-series.png)

2.  (continued). Now add this **slope** **series** to the scatter plot. We will end up with this chart.

![scatter plot after adding slope series](https://chandoo.org/wp/wp-content/uploads/2020/04/chart-after-adding-slope-series.png)

3.  Change the slope series to “Scatter with straight line” type of chart. To do this, right click on the orange dots (slope series) and select “Change series chart type”.

![changing slope series to line](https://chandoo.org/wp/wp-content/uploads/2020/04/changing-slope-dots-to-line.png)

4.  **Format everything.** Add a chart title, space out the grid lines and format the slope line. Your chart will be _almost ready._

![xy chart with slope line](https://chandoo.org/wp/wp-content/uploads/2020/04/almost-done-slope-line-in-xy-chart.png)

5.  **Bonus step:** You can add a text box to the chart and type in “more this week” in it and place it above the line. This will help in reading the data better. Here is our final XY chart with slope line.

![completed xy chart with slope line](https://chandoo.org/wp/wp-content/uploads/2020/04/slope-line-xy-chart.png)

Slope line in XY charts – When to use them & tips
-------------------------------------------------

Slope line is a great way to enhance the readability of _certain types of_ scatter plots. Consider adding them whenever you are visualizing data from two time periods or two similar situations. Examples are,

*   Last year vs. This year sales
*   Male vs. Female customers by stores
*   Failure rate in machinery before and after maintenance
*   Test scores of students in two different subjects

**Tips for working with slope line:**

*   **Always annotate the axis** (use either axis titles or put text boxes near them)
*   **Consider highlighting few of the points** on charts based on the value. For example, we can highlight apps that have significantly change in downloads. Like this:

![slope line chart with important points highlighted](https://chandoo.org/wp/wp-content/uploads/2020/04/slope-line-chart-with-important-points-highlighted.png)

[Here is a tip on how to highlight important points on a chart](https://chandoo.org/wp/how-to-highlight-maximum-value-in-excel-charts/)
