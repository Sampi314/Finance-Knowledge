### How to use this template?

Simple. Download the file first.

Set up month & year using the selection options on top. Enter details of your activities. As you make progress, mark things as done. The calendar view refreshes instantly so you can see what’s going on.

Column P & Q contain the calculations necessary to set up the calendar view.

Highlight calendar or list view ranges & check out **[conditional formatting](http://chandoo.org/wp/2009/03/13/excel-conditional-formatting-basics/)
** rules to understand how they work.
