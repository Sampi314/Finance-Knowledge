Last week we discussed a fun and easy way to [compare two lists of data in excel using conditional formatting](http://chandoo.org/wp/2010/06/17/compare-2-lists-in-excel/)
. In that post, [_**Artem**_ commented](http://chandoo.org/wp/2010/06/17/compare-2-lists-in-excel/#comment-114082)
,

> The quickest way to find all about two lists is to select them both and them click on Conditional Formatting -> Highlight cells rules -> Duplicate Values (Excel 2007). The result is that it highlights in both lists the values that ARE the same. Then in one list non-highlighted are values that are not present in the second list, and opposite for the second list. I think it is sell “geeky”, but it gets job done very very quickly when you don’t want to mess around.

_**Artem**_ must be an Excel Yoda. I somehow missed this beautiful and dead-simple way to compare lists in Excel. So here, I am documenting that technique so we all remember it and use it.

A Ridiculously easy and fun way to compare 2 lists
--------------------------------------------------

![Compare 2 lists of Data in Excel - Conditional Formatting Tip](https://chandoo.org/img/cf/compare-2-lists-conditional-formatting-qt.gif)\[works only Excel 2007+, use the [above technique](http://chandoo.org/wp/2010/06/17/compare-2-lists-in-excel/)\
 if you are on excel 2003 or earlier\]

1.  Select cells in both lists (select first list, then hold CTRL key and then select the second)
2.  Go to Conditional Formatting > Highlight Cells Rules > Duplicate Values
3.  Press ok.
4.  There is nothing do here. Go out and play!

See the screencast aside to see how this works ([click here for a detailed demo](http://chandoo.org/img/cf/compare-2-lists-cf-qt.gif)
).
