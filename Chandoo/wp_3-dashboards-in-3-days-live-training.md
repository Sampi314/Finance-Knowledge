What is this course?
--------------------

**Dashboards** are one of the most sought after skills for data professionals. In this live & interactive class,

**You will learn:**

*   How to think-up & design a dashboard from scratch
*   How to use Excel (or Power BI) to create dashboards to help your business
*   Common mistakes people make when creating dashboards.
*   How to make your dashboards interactive & _dynamic._
*   Advanced Excel & Power BI features to create dashboards

Who is this for?
----------------

This course is designed for people already using Excel and / or Power BI. 

**For Excel option:**

*   Intermediate level Excel skills
*   _Ideally_ Have created some dashboards & reports and looking for FRESH ideas & inspiration.

**For Power BI option:**

*   Intermediate level Power BI skills
*   _Ideally_ Have created some dashboards & reports and looking for FRESH ideas & inspiration.

Topics Covered
--------------

EXCEL

Day 1️⃣

On this day, we will build a **SALES Dashboard**

You will learn:

*   How to design a dashboard?
*   Using Pivot Tables with Data Model in Excel
*   Calculating values in Pivots using Power Pivot feature of Excel
*   Creating interactive charts with Slicers

EXCEL

Day 2️⃣

On this day, we will build a **Project Management Dashboard**. 

You will learn:

*   Using conditional formatting to make an interactive GANTT Chart
*   Generating custom traffic light chart in Excel
*   Adding drill-down feature to dashboard with slicers & data validation.
*   Making the dashboard dynamic

EXCEL

Day 3️⃣

On this day, we will create a **Finance KPI Dashboard.**

You will learn:

*   Calculating KPIs using Power Pivot.
*   Creating a cool variance chart with Excel
*   Adding _**alerts**_ to the dashboard using conditional formats & tables.
*   Adding SORT options to dashboard with Dynamic Array functions.
*   Maps to show geographical performance

POWER BI

Day 1️⃣

On this day, we will build a **HR Dashboard**

You will learn:

*   Gathering & cleaning employee data using Power Query
*   Modeling the data with Power Pivot & creating necessary measures.
*   Creating a dashboard with layouts in Power BI.
*   Adding bookmarks to show / hide features of the dashboard.

POWER BI

Day 2️⃣

On this day, we will build a **Sales Dashboard**

You will learn:

*   Mocking up dashboard for Power BI
*   Enhanced KPI CARD Visuals for dashboard
*   Add sales forecasts using “analytics” feature of Power BI
*   Using custom tool-tips to bring additional insight
*   Building DAX measures to add narrative to the dashboard

POWER BI

Day 3️⃣

On this day, we will build a **Multi-page CXO Dashboard**

You will learn:

*   How to build a multi-page dashboard report
*   Using drillthrough feature to navigate between pages.
*   Applying sparklines to tables in Power BI
*   One chart vs. Small-multiples for dashboard
*   Effectively using visual customization options
*   Working with Smart Narratives

Frequently Asked Questions
--------------------------

**Who is this training for?**

This training is perfect for intermediate level users of Excel or Power BI. If you work in a data profession (data analyst, reporting analyst or similar) and need to make dashboards, you will find this training _extremely_ valuable.

**How this is training is run?**

This training will be thru a Zoom video session. You will be able to see my presentation & explanations as well as share your audio (and screen if you want) when asking questions.

**What software do I need?**

**For Excel:** Excel 365 preferred. Excel 2016 / 2019 is ok.

**For Power BI:** Latest version of Power BI Desktop. _Optional: Power BI Online Account._

You can watch the recordings on any video player.

**\[Excel option\] Do I need any prior skills?**

Yes. You need to know how to create pivot tables, how to write formulas (lookups, sumifs is ok) and how to create simple charts. 

**\[Power BI option\] Do I need any prior skills?**

Yes. You need to know how to use Power BI for simple, everyday stuff. If you are completely new to Power BI, you can still enroll, but you must know the basics before we start our sessions in August.

**Can I ask questions?**

The session will be highly interactive. You will have lots of opportunities to share your experiences or ask questions.

**What if I can’t attend part of the training?**

It is _highly_ recommended that you plan to attend all 3 days. While you can watch the recording, you will miss out on the interaction and break-out sessions.
