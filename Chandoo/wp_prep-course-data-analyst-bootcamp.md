Prep Course: Data Analyst Bootcamp
==================================

![dabc PREP COURSE](https://chandoo.org/wp/wp-content/uploads/2023/09/SNAG-2978.png)

Congratulations on enrolling in my **DATA ANALYST BOOTCAMP**. You must complete this prep-course (takes 3 hours) before attending the bootcamp. This course will give you the foundation concepts of,

*   Excel
*   Power BI

Software Requirements
---------------------

Make sure you have the below software on your computer to get best results out of the bootcamp.

*   Microsoft Excel (365 preferred, else 2016+)
*   Microsoft Power BI Desktop (**[download free here](https://powerbi.microsoft.com/en-us/desktop/)
    **)
*   SQL Server Management Console ([**download free here**](https://learn.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver16)
    )
*   Zoom (for attending the meetings)
*   _Optional: SQL Server Express ([download free here](https://go.microsoft.com/fwlink/p/?linkid=2216019&clcid=0x409&culture=en-us&country=us)
    )_
