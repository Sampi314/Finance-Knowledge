Advanced Formulas & Functions
-----------------------------

Formulas make Excel smart. With out them, Excel is just a data keeping tool. But by using formulas, you can crunch data, analyze it and get answers to most complex questions. While anyone can use a simple SUM or IF formula, an advanced user of it would be able to seamlessly write & combine formulas like SUMIFS, FILTER, XLOOKUP, IFS and other functions.

Apart from knowing the formulas, advanced Excel users know how to debug them, audit them and how to use which formula for which occasion (and they also know few alternatives for any given formula problem).

Conditional Formatting
----------------------

If there is one Excel feature that can turn a boring spreadsheet into an awesome productive tool, it has to be conditional formatting.

[![Conditional Formatting in Excel - an Example](https://chandoo.org/wp/wp-content/uploads/2021/02/databar-and-icon-in-the-same-cell.png)](https://chandoo.org/wp/5-useful-conditional-formatting-tricks/)

By using conditional formatting, you can tell Excel to highlight portions of your data that meet any given condition. For example,

*   Highlighting top 10 customers
*   below average performing employees
*   All upcoming events in next 90 days

While anyone can set up simple conditional formatting rules, an advanced Excel user can do a lot more. They can **combine formulas with conditional formatting** to highlight data that meets almost any condition.

*   Make and format statistical charts like histograms, box plots, distribution curves with ease.
*   Use features like in-cell charts & conditional formatting charts

*   Power Query for data automation

*   Office scripts for desktop + Excel online automation
*   Power Automate (Flows) to automate Excel with other office apps like Outlook, SharePoint or OneDrive.

Financial Modeling with Excel
-----------------------------

The main use of Excel is to solve business problems. So you need to know how to model a business or financial situation with Excel and how to simulate real-world scenarios.

That is why advanced Excel users with modelling, business domain and statisical understanding get paid big bucks.

In order to build such models, you need below advanced Excel skills.

*   How to build the classic 3 statement financial model in Excel (Cash flow, Balance sheet and P&L)
*   Understanding of financial and business KPIs and how to calculate them with Excel formulas
*   How to use Solver to find optimum solutions
*   Doing what-if analysis with data tables in Excel
*   Ability to build forecasting models using features like LINEST, multiple regression, forecasting and trend lines.
*   Know how and when to use statistical functions of Excel.
