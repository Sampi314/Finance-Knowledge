Gmail Labs 2 new useful features – Mark as Read, Label Auto Suggest
===================================================================

[technology](https://chandoo.org/wp/category/computer/)
 - 1 comment

  

![gmail-labs-useful-features-mark-as-read-label-auto-suggest](https://chandoo.org/wp/wp-content/uploads/2008/09/gmail-labs-useful-features-mark-as-read-label-auto-suggest.png "gmail-labs-useful-features-mark-as-read-label-auto-suggest")[Gmail Labs](http://mail.google.com/support/bin/answer.py?hl=en&ctx=mail&answer=29418)
 constantly introduces new features that make using Gmail much more productive. Recently they introduced 2 new features – Mark as Read button and Auto suggest for label search which I find extremely useful.

_Mark as read_ is really useful since I use it often to get to inbox zero. I am obsessed with reading / replying to all relevant mails.

Just enable these cool features by clicking on “settings” on your gmail page and then go to “Labs” tab. Enable any of the cool labs features to make your mailbox much more productive.

![](https://chandoo.org/wp/wp-content/uploads/2008/09/gmail-enable-labs-features-howto.png "gmail-enable-labs-features-howto")

Btw, if you are experiencing problems with labs features you can always visit your gmail inbox without labs features by visiting **[http://mail.google.com/mail/?labs=0](http://mail.google.com/mail/?labs=0)
