### Excel Formula to Count Number of Unique Values

![count-unique-values-from-range](https://chandoo.org/wp/wp-content/uploads/2009/08/count-unique-values-from-range.png "count-unique-values-from-range")Assuming we have a list of values in the range:B5:B15 and we want to know how many unique values are there,

you can write the oh so wonderful [sumproduct formula](http://chandoo.org/wp/?s=sumproduct)
 like this:

`SUMPRODUCT(1/COUNTIF(B5:B15,B5:B15))`.

### How this formula works?

The portion `COUNTIF(B5:B15,B5:B15)` of the formula returns the individual item counts. So in our example range, the values would be: `5,1,2,5,1,1,1,2,5,5,5`. When you add up the reciprocals of these values, you will get the count of unique values in the range.
