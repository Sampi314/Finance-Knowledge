FREE 4 Hours Complete Excel Course
==================================

Here is a FREE, complete end-to-end Excel course to help you learn all the key and essential aspects of Microsoft Excel for Data Analysis work.

![](https://chandoo.org/wp/wp-content/uploads/2024/12/SNAG-3920.png)

What you will learn:
--------------------

*   How to use MS Excel
*   Working with Power Query to automatically clean data
    *   Data load from Web pages, CSV files & Excel files
    *   Adding columns with new calculations
    *   Replacing “blank” or “null” values
    *   Data quality checks
    *   Splitting names into first & last names
    *   Working with dates in Power Query
*   Using Important Excel Functions & Formulas
    *   Using references in Excel
    *   SUMIFS, COUNTIFS formulas
    *   Statistical Analysis of Data with Excel formulas
    *   FILTER function
    *   Lookup functions – VLOOKUP, INDEX+MATCH, XLOOKUP
    *   Combining functions (nesting) to solve problems
*   Working with Excel Pivot Tables
    *   Creating Pivot tables
    *   Sorting & filtering pivots
    *   Slicers & timelines with Pivots
    *   Grouping data with Pivots
    *   Pivot charts, Interactive charts
*   Complete Excel Project
    *   Creating a call center dashboard
