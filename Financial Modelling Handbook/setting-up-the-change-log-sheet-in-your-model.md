# How to set up a Change Log sheet in your model

**Source:** https://www.financialmodellinghandbook.org/setting-up-the-change-log-sheet-in-your-model

---

The Change Log sheet gives you a process for structured tracking of changes to your model. Hopefully, in the future, this will become redundant when Microsoft implements proper change tracking in Excel.

This Chapter covers setting up the Change Log sheet. Once the sheet is set up, read the Change Logging process guidance.

Start model
-----------

If you're starting your model build process using [this start model](https://www.financialmodellinghandbook.org/using-a-start-model/)
, there's nothing you need to do. The Change Log sheet is included and ready to go. Make sure you have [the latest version of the Productivity Pack](https://www.financialmodellinghandbook.org/financial-modelling-productivity-pack/)
 to use the Change Log shortcuts.
