# What to do if Control+Shift+0 is not working

**Source:** https://www.financialmodellinghandbook.org/what-do-if-controlshift0-is-not-working

---

We like to be able to hide unused columns at the end of the model timeline. If we need to change the timeline we want to be able to easily unhide columns again.

The Excel keystroke to unhide columns is Ctrl+Shift+0. Many users find that this does not work.

Here’s how to change your keyboard settings to fix that. Note - the screenshots below are from Windows 11.

### 1\. In Windows settings, select “Time & Language”

![](https://www.financialmodellinghandbook.org/content/images/2022/11/1-8.jpg)

### 2 From the Time & Language settings, select Typing:

![](https://www.financialmodellinghandbook.org/content/images/2022/11/2-9.jpg)

3 From the Typing dialogue, select Advanced keyboard settings
-------------------------------------------------------------

![](https://www.financialmodellinghandbook.org/content/images/2022/11/3-8.jpg)

4 From the Advanced keyboard setting dialogue, Select Language Bar Options
--------------------------------------------------------------------------

![](https://www.financialmodellinghandbook.org/content/images/2022/11/4-8.jpg)

A new dialogue will then open. I say new. It looks like it’s leftover from older versions of windows!

### 5 Select Advanced Key Settings

![](https://www.financialmodellinghandbook.org/content/images/2022/11/5-8.jpg)

### 6 With “Between input languages” selected, click on Change Key Sequence

![](https://www.financialmodellinghandbook.org/content/images/2022/11/6-6.jpg)

### 7 Select “Not assigned” under “Switch Keyboard Layout”

![](https://www.financialmodellinghandbook.org/content/images/2022/11/7-3.jpg)

Your Control+Shift+0 problems should now be a thing of the past.
