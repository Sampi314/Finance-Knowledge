We've already seen how to create a new link.

We don't always have to create links from scratch. If we already have a link to the line item, we can copy that link instead.

In our example calculation block for Electricity generation revenue, the last ingredient we need is the Operating Period flag.

We have already used this to calculate the Seasonality adjustment in row 22.

![](https://www.financialmodellinghandbook.org/content/images/public/images/95431477-0194-455f-be74-cd2c7c819d3e_2236x822.jpeg)

Rather than create a new link from scratch to the relevant calculation on the timesheet, we can copy the link from Row 22.

Step 1: Select the row
----------------------

Shift + Spacebar (Native excel keystroke)

![](https://www.financialmodellinghandbook.org/content/images/public/images/d64331f7-201d-482a-956f-2af712f26879_2268x936.jpeg)

Step 2: Copy the row
--------------------

Ctrl+c (Excel native keystroke)

Step 3: Paste the row in the desired location
---------------------------------------------

Shift+spacebar, Enter (Excel native keystroke)

![](https://www.financialmodellinghandbook.org/content/images/public/images/a44ba3d9-66c2-4a98-b3d5-57733ae25e2f_2252x944.jpeg)

Copying links between worksheets.
---------------------------------

There are two kinds of links in our model. Local links are links to line items on the same sheet. Imported links are links to line items coming from other sheets. Note that you can only copy local links to another location on the same sheet. You can copy imported links to any other sheet in the model. Imported links have the sheet name in the address, allowing us to copy them to any other sheet. This is one of the reasons we mark imported links with a different font.
