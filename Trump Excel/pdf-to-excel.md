Extract All the Tables from the PDF to Excel
--------------------------------------------

Another useful feature of [Able2Extract](https://www.investintech.com/able2extract.html?invt=68407c141)
 is to extract all the tables from a PDF into Excel.

This is especially useful if you’re extracting data from a financial report (10K) as these are full of text as well as tables.

Here are the steps to extract all the tables from a PDF:

1.  Open Able2Extract. If you have the trial version, click on the Continue button.
2.  Click the Open button.
3.  In the Open dialog box, locate and open the PDF file that you want to convert to Excel. This will open the file within the Able2Extract window.
4.  Click on the All button.
5.  Click the Excel option.
6.  In the Convert to Excel dialog box, click on the blue gear icon at the bottom left of the dialog box.![Extract All the tables from PDF to Excel](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20484%20311'%3E%3C/svg%3E)
7.  In the Settings dialog box, check the ‘Extract Tables Only’ option.![Extract Table only checked](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20444%20270'%3E%3C/svg%3E)
8.  Click OK.
9.  Click on Convert.
10.  In the Save As dialog box, save the file.

The above steps would extract all the tables in a worksheet in Excel.

While this feature is amazing, I wish there was a way to tell the tool to give the tables on different sheets. Having all the tables in a single worksheet messes up the column widths. But this still does help a lot as you can easily copy-paste these tables into different sheets manually.
