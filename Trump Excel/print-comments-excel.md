If you often need to print your Excel worksheets, it would be useful to know how to print comments in it.

By default, this setting is disabled, which means that when you print a worksheet, the comments are not printed.

While Excel provides a couple of options to print comments, it is not readily available in the ribbon.

This Tutorial Covers:

[Toggle](https://trumpexcel.com/print-comments-excel/#)

Print Comments in Excel using the Page Setup from Ribbon
--------------------------------------------------------

If you have a lot of comments in a worksheet and you print it, by default the comments would not get printed. You need to make a few changes in the settings to get this done.

There are two ways you can print comments in Excel:

*   Print Comments at the End of the Sheet.
*   Print Comments as Displayed on the Sheet.

Let’s tackle these options one by one.

### Print Comments at the End of the Sheet

Here are the steps:

*   Go to the Page Layout tab.![How to Print Comments in Excel - pageLayout](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20399%20129'%3E%3C/svg%3E)
*   In the Sheet Options group, click on the small arrow icon at the bottom right of the group (called the dialogue box launcher). It’ll open the Page Setup Dialogue box.![How to Print Comments in Excel - PageSetup](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20612%20142'%3E%3C/svg%3E)
*   In the Page Setup dialogue box, within the Sheet tab, go to the Comments drop-down and select **At end of sheet**.![How to Print Comments in Excel - At end of sheet](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20445%20461'%3E%3C/svg%3E)
*   Click OK. (You can also click on Print Preview to see how it would look when printed).![How to Print Comments in Excel - Print Preview](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20450%20462'%3E%3C/svg%3E)

Now when you print the worksheet, all the comments would be printed at the end of the worksheet.

Something as shown below:

![How to Print Comments in Excel - Cell Comments at end](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20637%20339'%3E%3C/svg%3E)

The benefit of using this method is that it gives you all the comments in one place along with the [cell address](https://trumpexcel.com/return-cell-address-instead-of-value-excel/)
 and the commenter’s name. This could be especially helpful if your worksheet gets reviewed by multiple people and you have different commenters.

_Using this method, you can get a list of all the comments, when printed, but you can’t get a list of the comments in the workbook itself. [Here is a tutorial](https://trumpexcel.com/get-list-of-comments-in-a-worksheet-excel/)
 that will show you how to quickly get a list of all the comments in a worksheet._

### Print Comments as Displayed in the Worksheet

If you want to see the comments as it is displayed in the worksheet, here are the steps:

*   The first thing you need to make sure is that all the comments that you want to print are visible on the screen. To do this, go to Review Tab –> Comments –> Show All Comments. This would instantly make all the comments show up close to the cell that holds that comment.![How to Print Comments in Excel - Show all Comments](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20332%20127'%3E%3C/svg%3E)
    *   If you want to show some of the comments (and not all), select the cell which has the comment, go to Review tab –> Comments –> Show/Hide Comment. This is a toggle button. If a comment is visible, it hides it, and if it is hidden, then this makes it visible.
*   Go to the Page Layout tab.![How to Print Comments in Excel - pageLayout](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20399%20129'%3E%3C/svg%3E)
*   In the Sheet Options group, click on the small arrow icon at the bottom right of the group (called the dialogue box launcher). It’ll open the Page Setup Dialogue box.![How to Print Comments in Excel - PageSetup](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20612%20142'%3E%3C/svg%3E)
*   In the Page Setup dialogue box, within the Sheet tab, go to the Comments drop-down and select As displayed on sheet.![How to Print Comments in Excel - As displayed](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20447%20461'%3E%3C/svg%3E)
*   Click OK. (You can also click on Print Preview to see how it would look when printed).

It will print the comments and you’ll get exactly what you see on your screen. Something as shown below:

![How to Print Comments in Excel - As displayed Output](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20612%20441'%3E%3C/svg%3E)

Print Comments in Excel using the Print Preview
-----------------------------------------------

There is another way to access the Page Setup dialogue box options.

*   Go to File –> Print (or use the [keyboard shortcut](https://trumpexcel.com/excel-keyboard-shortcuts/)
    : Ctrl + P).
*   In the Print Preview screen, at the bottom of the screen, there is a Page Setup option. Click on it.![How to Print Comments in Excel - Print Preview PageSetup](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20176%20456'%3E%3C/svg%3E)
*   It opens the Page Setup dialogue box. Select the sheet tab and you’ll find the Print comments options in the drop-down.![How to Print Comments in Excel - Comments Dropdown](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20445%20462'%3E%3C/svg%3E)

These are the same options as discussed above:

*   Select **None** – If you don’t want to print any comments.
*   Select **At end of sheet** – if you want to print comments at the end of the worksheet.
*   Select **As displayed on sheet** – if you want to print comments as shown in the worksheet.
