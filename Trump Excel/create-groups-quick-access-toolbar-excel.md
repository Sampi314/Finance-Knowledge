If you are addicted to using the [Excel Quick Access toolbar](https://trumpexcel.com/excel-quick-access-toolbar-options/)
 (QAT), this tip is for you.

As you add many icons to the Excel QAT, there is also a way to group related icons together. Here is a snapshot of my Quick Access Toolbar:

![Trump Excel Quick Access Toolbar](https://trumpexcel.com/wp-content/uploads/2014/11/Trump-Excel-Quick-Access-Toolbar.png)

These vertical line separators can be used to group similar icons together. For example, here I have all the object alignment icons together. Similarly, I have the data related icons ([data validation](https://trumpexcel.com/learn-all-about-data-validation-in-excel/)
 and [sorting](https://trumpexcel.com/sort-excel/)
) together at the end.

#### Creating Groups in Quick Access Toolbar

*   Right-click on the QAT icons and select Customize Quick Access Toolbar..![Customize Quick Access Toolbar](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20452%20170'%3E%3C/svg%3E)
*   In the Excel Option dialogue box, there are 2 sections. On the left, you have all the icons that can be added to QAT, and on the right, you have existing QAT icons. To add a separator, click on <Separator> and then click Add. You can then set the position of the Separator by using the up and down arrow keys.![Quick Access Toolbar Adding Separator](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20665%20600'%3E%3C/svg%3E)
*   Click Ok. That’s it. Now your QAT icons would be separated by a grey vertical line (which gives an illusion that you have created groups).
