### Tile chart

 

|     |     |
| --- | --- |
|     | ![](https://www.andypope.info/charts/tilechart.gif)<br>--------------------------------------------------- |
|     | ##### The chart displays a matrix. The colour of the tile represents the value at each intersection. <br><br>##### The chart is constructed from a combination of Bar, Column and XY-Scatter. The Bar and Column chart are used to provide category axis labels. The tiles are derived from the XY-scatter chart.  <br>The chart uses data from conditional formula in order to display a single tile in each position.<br><br>##### The [workbook](https://www.andypope.info/charts/tilechart.zip)<br> contains a step by step explanation on how to construct the chart. |
