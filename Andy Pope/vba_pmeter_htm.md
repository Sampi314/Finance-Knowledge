### Progress Meters

   

##### A Progress bar is a good way of letting you know that your code is still working, especially if you have disabled screen updating.

##### This [workbook](https://www.andypope.info/vba/progressmeters.zip)
 contains a collection of different progress meters.

The images were all created use the Autoshapes in Excel.
