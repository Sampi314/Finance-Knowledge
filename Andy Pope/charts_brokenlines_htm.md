### Broken Lines for formula linked data

![](https://www.andypope.info/charts/brokenlines.gif)  
 

 

##### Non zero data returned from a formula is treated as zero.  
The use of NA() allows the line to be joined between numeric points.

##### The mask method can be used to create breaks for missing data.

|     |     |     |     |     |     |
| --- | --- | --- | --- | --- | --- |     
| ##### Original Data |     | ##### Linked Data "" | ##### Linked Data #N/A | ##### Linked Data Masked | ##### Mask |
| ##### 0.28 | ##### A | ##### 1.28 | ##### 2.28 | ##### 3.28 | ##### #N/A |
| ##### 0.25 | ##### B | ##### 1.25 | ##### 2.25 | ##### 3.25 | ##### 3.25 |
| ##### 0.01 | ##### C |     | ##### #N/A | ##### #N/A | ##### #N/A |
