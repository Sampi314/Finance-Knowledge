### Create Axis Labels using Textboxes

 

|     |     |
| --- | --- |
|     | ![](https://www.andypope.info/vba/tal2.gif)<br>--------------------------------------------- |
|     | The control you have over the alignment of the axis labels is very limited and you can not control the gap between the left of the chart and the left of the plotarea that is used by the labels.  <br>I have written VBA code that replaces the built-in axis labels with textboxes.  <br>The [example workbook](https://www.andypope.info/vba/Axis_Textboxes.zip)<br> contains 2 charts. 1 with built-in alignment and one that will use textboxes.  <br>The required alignment can be chosen from 2 dropdown lists. |
|     | The code works with xl2007 charts and text boxes. One thing to note is that not all the alignment options are available in xl2007. |
|     |
