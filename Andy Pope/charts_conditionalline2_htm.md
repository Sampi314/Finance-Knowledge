### Formula Conditional line chart

 

|     |     |
| --- | --- |
|     | ![](https://www.andypope.info/charts/condline2.gif)<br>--------------------------------------------------- |
|     | ##### This chart displays a line that changes colour depending on whether it is above or below a control line. It is based on a xy-scatter chart.<br><br>##### The formula calculates all the intersections and then the sorted values are plotted using 2 series.  <br>  <br>Full credit goes to Dave Curtis, a forum member at [OzGrid](http://www.ozgrid.com/forum/)<br>. who provided the formula to calculate the intersections.<br><br>##### I was able to add the formula to dynamically sort the resulting values.<br><br>##### **Update 16th February 2008:**  <br>Apollo Huang, a fellow MVP from China, pointed out a flaw in my sorting formula, which was picking up incorrect cell values. I have incorporated his formula revision in to the download.  <br>  <br>The [download](https://www.andypope.info/charts/ConditionalLine2.zip)<br> contains data based on the RAND function so will generate new values automatically. |
