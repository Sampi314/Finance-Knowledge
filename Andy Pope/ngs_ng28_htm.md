|     | Re: Stacked cluster charts in PowerPoint<br>---------------------------------------- |
|     | ##### Approach 1 |
| ##### The chart is a combination of normal stacked column chart. The values in the data table have been staggered<br><br>![](https://www.andypope.info/ngs/ng28.h1.gif)  <br>![](https://www.andypope.info/ngs/ng28.h2.jpg) <br><br>##### An additional data series is used to display more usable axis labels<br><br>##### ![](https://www.andypope.info/ngs/ng28.h3.gif)<br><br>##### Add a dummy set of data for axis label.  <br>Move the new data to the secondary axis.  <br>Set the secondary value axis to a maximum of 2.  <br>Enable the secondary x axis, use the chart options dialog setting it to date type.  <br>Change the max scale value to 4.  <br>Enable data label.  <br>Suppress tickmark labels on the secondary x axis.  <br>Set the secondary y axis to have value in reverse then suppress tickmarks and labels.  <br>Delete extra legend entry |
| ##### Approach 2 |
| ##### ![](https://www.andypope.info/ngs/ng28.h4.gif)<br><br>##### The chart is a normal clustered column chart with East and South data on the secondary axis.  <br>The values in the data table have to be altered in order to make the back columns, West & North, taller than the original data would have mage them  <br>![](https://www.andypope.info/ngs/ng28.h5.jpg)  <br><br>##### Both examples are in this [example](https://www.andypope.info/ngs/ng28.zip)<br> presentation. |
