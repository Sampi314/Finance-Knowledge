### Invert if negative

  

![](https://www.andypope.info/charts/invert_9.gif)
--------------------------------------------------

 

|     |
| --- |
| ##### For column, bar, cylinder, cone and pyramid charts there is an option to automatically  display an alternative colour if the value is negative. This feature can be very handy but configuring it is awkward to say the least. |
| ##### How To Use 'Invert if negative' |
|     |
| ##### Lets start with a normal column chart. Negative bars are currently coloured the same as positive.<br><br>##### ![](https://www.andypope.info/charts/invert_10.gif) |
|     |
| ##### From the Format Data Series dialog we can enable the 'Invert if negative' option.![](https://www.andypope.info/charts/invert_2.gif) |
| #####   <br>This is how the Invert if negative will work according to the help.![](https://www.andypope.info/charts/invert_11.gif)  <br>_(Note: pie charts do not plot negatives and the option is not available for either pie or area)_ |
|     |
| ##### The result is a different colour for the negative values.   <br>So now we want to select a suitable negative colour.![](https://www.andypope.info/charts/invert_1.gif) |
|     |
| ##### If we once again display the Format Data Series dialog and press Fill Effects we will see the dialog shown below. The Foreground colour matches the positive bars and the Background colour matches with the negative bars.  <br>So we change the background colour to the desired colour for our negative bars.<br><br>##### Now you may have noticed that all the Patterns contain a mixture of Foreground and Background. As we want solid colours we will have to select the pattern with the least intrusive pattern, which is the 5% pattern.  <br>![](https://www.andypope.info/charts/invert_4.gif) |
|     |
| ##### So now the bars are have different colours for positive and negative values.<br><br>##### BUT wait, they are the complete opposite of what we want!![](https://www.andypope.info/charts/invert_5.gif) |
|     |
| ##### Lets go back to the Fill Effects dialog and swap the colour selection.![](https://www.andypope.info/charts/invert_6.gif) |
|     |
| ##### That's better, we now have the right colours for the positive and negative bars. Although our chart looks as though it has measles.<br><br>##### ![](https://www.andypope.info/charts/invert_7.gif) |
|     |
| ##### To get rid of those spots go back to the Format Data Series dialog and select the required colour for the positive bars.<br><br>##### ![](https://www.andypope.info/charts/invert_8.gif) |
|     |
| ##### There we go a chart with solid positive and negative colours.<br><br>![](https://www.andypope.info/charts/invert_9.gif) |
|     |
| ##### See the tip on how to '[Make X axis labels display below negative data](https://www.andypope.info/tips/tip002.htm)<br>'<br><br>This technique does not work in xl2007. The changing of the colour in the final step causes the chart to revert back to default invert colour. |
