### Linking the chart title to a cell

You can create dynamic text within a chart by linking the text element to a worksheet cell.

[![](https://www.andypope.info/Site_Images/excel_green.png)Excel 2013](https://www.andypope.info/tips/tip001.htm#Excel2013 "Excel 2013 version") [![](https://www.andypope.info/Site_Images/Excel2007.png)Excel 2007/2010](https://www.andypope.info/tips/tip001.htm#Excel2010 "Excel 2010 version") [![](https://www.andypope.info/Site_Images/Excel2003.png)Excel 2003](https://www.andypope.info/tips/tip001.htm#Excel2003 "Excel 2003 version")

  
  
 

**Excel 2013**  

If your chart does not already have the title displayed you can use Add Chart Element > Chart Title to insert one.  
  

![](https://www.andypope.info/tips/tip001h.png)

  
To create the link select the chart title and using the formula bar enter the cell address of the cell you want to link to.  
You can also enter the = sign and then use your mouse to select the cell and the address will automatically be entered.  
  

![](https://www.andypope.info/tips/tip001i.png)

  
Now when the contents of B1 changes the charts title will automatically update..  
  
The above technic can be applied to other text elements within a chart.

*   Primary horizontal axis title
*   Primary vertical axis title
*   Secondary horizontal axis title
*   Secondary vertical axis title
*   Datalabels
*   Value axis Display Units label
*   Depth axis title

**Excel 2007 or 2010**  

If your chart does not already have the title displayed you can use Chart Tools > Layout > Labels > Chart Title to add one.  
  

![](https://www.andypope.info/tips/tip001d.png)

  
To create the link select the chart title and using the formula bar enter the cell address of the cell you want to link to.  
You can also enter the = sign and then use your mouse to select the cell and the address will automatically be entered.  
  

![](https://www.andypope.info/tips/tip001e.png)

  
Now when the contents of B1 changes the charts title will automatically update..  
  
The above technic can be applied to other text elements within a chart.

*   Primary horizontal axis title
*   Primary vertical axis title
*   Secondary horizontal axis title
*   Secondary vertical axis title
*   Datalabels
*   Value axis Display Units label
*   Depth axis title

**Excel 2003**  

If your chart does not already have the title displayed you can use Chart Options dialog enter chart title.  

![](https://www.andypope.info/tips/tip001b.gif)

  
To create the link select the chart title and using the formula bar enter the cell address of the cell you want to link to.  
You can also enter the = sign and then use your mouse to select the cell and the address will automatically be entered.  

![](https://www.andypope.info/tips/tip001c.gif)

**Linking Shapes**  

  
As well as chart text elements it is also possible to link a shape to a cell using the same technique.  
  

![](https://www.andypope.info/tips/tip001f.png)

  
Linked shapes automatically pickup the formatting of the cell when the link is first made.  
 

![](https://www.andypope.info/tips/tip001g.png)
