Excel Functions
===============

Published [by Giles Male](https://www.fullstackmodeller.com/blog/author/giles-male)

Jan 1, 2021 2:35:00 PM

The Excel functions list.

Below are the key Excel functions for financial modelling:
----------------------------------------------------------

#### [IF](https://www.fullstackmodeller.com/blog/how-to-use-the-if-function)
   Returns one value for a TRUE result, and another for a FALSE result

### [EOMONTH](https://www.fullstackmodeller.com/blog/how-to-use-the-eomonth-function)
  Returns the last day of the month after adding a specified number of months

### [MAX](https://www.fullstackmodeller.com/blog/how-to-use-the-max-function)
  Returns the largest value from a range of numbers

### [MIN](https://www.fullstackmodeller.com/blog/how-to-use-the-min-function)
  Returns the smallest value from a range of numbers

### [INDEX](https://www.fullstackmodeller.com/blog/how-to-use-the-index-function)
 Returns a value from a range based on the position specified

### [SUMIFS](https://www.fullstackmodeller.com/blog/how-to-use-the-sumifs-function)
 Sum rows that meet specific criteria

### [XLOOKUP](https://www.fullstackmodeller.com/blog/how-to-use-the-xlookup-function)
 _Look up values from another table_

_Read more about all of Excel's functions on the Microsoft support page [here](https://support.microsoft.com/en-us/office/excel-functions-alphabetical-b3944572-255d-4efb-bb96-c6d90033e188)
