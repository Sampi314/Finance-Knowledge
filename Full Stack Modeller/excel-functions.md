Guide To Key Excel Functions
============================

There are over 450 functions in Excel and this number is constantly growing as the Microsoft team expand the Excel function library.

We have filtered this list of Excel functions down for you to provide you with the key functions that you need to know as a modeller.

|     |     |
| --- | --- |
| [SUMIFS](https://www.fullstackmodeller.com/blog/how-to-use-the-sumifs-function) | The [SUMIFS](https://www.fullstackmodeller.com/blog/how-to-use-the-sumifs-function)<br> function returns the total value for rows in a dataset that meet the criteria that you have defined. [SUMIFS](https://www.fullstackmodeller.com/blog/how-to-use-the-sumifs-function)<br> is an extremely useful function for obtaining the values to use in your reports. |
| [XLOOKUP](https://www.fullstackmodeller.com/blog/how-to-use-the-xlookup-function) | In modelling we use the [XLOOKUP](https://www.fullstackmodeller.com/blog/how-to-use-the-xlookup-function)<br> function to map data from one table into another based on a unique reference. |
| [IF](https://www.fullstackmodeller.com/blog/how-to-use-the-if-function) | The [IF](https://www.fullstackmodeller.com/blog/how-to-use-the-if-function)<br> function performs a logical test and then returns the value you have provided based on a TRUE or FALSE result. In financial modelling it is often used to model business logic based on 1, 0 flags. |
| [IFS](https://www.fullstackmodeller.com/blog/the-ifs-function) | The [IFS](https://www.fullstackmodeller.com/blog/the-ifs-function)<br> function performs a series of logical tests and then returns the value you have provided based on a TRUE or FALSE result. The [IFS](https://www.fullstackmodeller.com/blog/the-ifs-function)<br> function is an upgrade of the [IF](https://www.fullstackmodeller.com/blog/how-to-use-the-if-function)<br> function. It allows financial modellers to test multiple arguments without needing a complicated nested [IF](https://www.fullstackmodeller.com/blog/how-to-use-the-if-function)<br> formula. |
| [INDEX](https://www.fullstackmodeller.com/blog/how-to-use-the-index-function) | We use the [INDEX](https://www.fullstackmodeller.com/blog/how-to-use-the-index-function)<br> function to return a value from a range based on the position specified. In scenario modelling we can use [INDEX](https://www.fullstackmodeller.com/blog/how-to-use-the-index-function)<br> to pull the values for the selected scenario. [INDEX](https://www.fullstackmodeller.com/blog/how-to-use-the-index-function)<br> is often combined with the MATCH function to to map data from one table into another based on a unique reference. |
| [EOMONTH](https://www.fullstackmodeller.com/blog/how-to-use-the-eomonth-function) | The [EOMONTH](https://www.fullstackmodeller.com/blog/how-to-use-the-eomonth-function)<br> function returns the date value for the last day of the month plus or minus the number of months you have specified. |
| [TEXTSPLIT](https://www.fullstackmodeller.com/blog/how-to-use-the-textsplit-function) | The [TEXTSPLIT](https://www.fullstackmodeller.com/blog/how-to-use-the-textsplit-function)<br> function splits values from a cell across rows and/or columns based on a delimiter. The function makes use of Excel' dynamic array functionality to "spill" the results aross rows and columns. |
| [TEXTJOIN](https://www.fullstackmodeller.com/blog/how-to-use-the-textjoin-function) | The [TEXTJOIN](https://www.fullstackmodeller.com/blog/how-to-use-the-textjoin-function)<br> function concatenates (or joins) values together from cells or ranges of cell. |
| [TEXTBEFORE](https://www.fullstackmodeller.com/blog/how-to-use-the-textbefore-function) | The [TEXTBEFORE](https://www.fullstackmodeller.com/blog/how-to-use-the-textbefore-function)<br> function returns the text from a text string that occurs before a given delimiter (character or string). |
| [TEXTAFTER](https://www.fullstackmodeller.com/blog/how-to-use-the-textafter-function) | The [TEXTAFTER](https://www.fullstackmodeller.com/blog/how-to-use-the-textafter-function)<br> function returns the text from a text string that occurs after a given delimiter (character or string). |
| [SWITCH](https://www.fullstackmodeller.com/blog/the-switch-function) | The [SWITCH](https://www.fullstackmodeller.com/blog/the-switch-function)<br> function evaluates an expression against a given value and returns the corresponding result value that you have provided. |
| [MIN](https://www.fullstackmodeller.com/blog/how-to-use-the-min-function) | The [MIN](https://www.fullstackmodeller.com/blog/how-to-use-the-min-function)<br> function returns the smallest value in the range of numbers provided. In financial modelling the [MIN](https://www.fullstackmodeller.com/blog/how-to-use-the-min-function)<br> function is very useful for ensuring that the result of a calculation does not go above a given threshold. |
| [MAX](https://www.fullstackmodeller.com/blog/how-to-use-the-max-function) | The [MAX](https://www.fullstackmodeller.com/blog/how-to-use-the-max-function)<br> function returns the largest value in the range of numbers provided. In financial modelling the [MAX](https://www.fullstackmodeller.com/blog/how-to-use-the-max-function)<br> function is very useful for ensuring that the result of a calculation does not go below a given threshold. |
