Responsive \[2/6\] | Forvis Mazars Financial Modelling

We expect our models to be used live and support dynamic situations such as transactions, presentations and decision making.  Use the right functions and functionality to keep the model nimble and structure to give fast feedback to the user.

*   Keep inputs close together for quick access 
*   Key outputs shown near inputs for fast feedback 
*   Transparent display of scenarios 
*   Signal alerts for the user 
*   Control buttons prominent and easy to access 
*   Minimise calculation macros 
*   Fast calculation through optimal function choice 
*   Build in signal checks about optimisation 
*   Choose a sensible tolerance for optimality. 
*   Keep the model flexible to help explore the options 
*   Give the user lots of feedback to help guide the inputs
