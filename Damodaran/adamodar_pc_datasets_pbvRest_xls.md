Variables & FAQ
---------------

|     |     |     |
| --- | --- | --- |
| End Game | To estimate ratios of market values (equity and firm) to book value (equity and firm), with companion variables (ROE and ROC) |     |
|     |     |     |
| Variable | Explanation | Why? |
| Number of firms | Number of firms in the industry grouping. | Law of large numbers? |
| PBV | Aggregated market capitalization divided by aggregated book value of equity, across all firms in the group. | Measure of market value of equity, relative to accounting value of equity |
| ROE | Aggregate net income divided by aggregated book value of equity, across companies in group. | Measure of returns earned on projects, to equity investors |
| EV/ Invested Capital | Aggregated enterprise value divided by aggregated invested capital, across all firms in the group. (EV = Market Cap + Debt - Cash, Invested Capital = BV of equity + Debt - Cash) | Measure of market value of firm, relative to accounting value of firm |
| ROIC | Aggregate after-tax operating income divided by aggregated invested capital, across companies in group. | Measure of returns earned on projects, to all claim holders |
