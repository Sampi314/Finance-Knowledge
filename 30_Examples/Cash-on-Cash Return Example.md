# Cash-on-Cash Return Example

## Setup

Calculating the first-year yield for a commercial real estate property investment.

## Assumptions

- **Potential Gross Income**: $100,000
- **Vacancy and Credit Loss**: $25,000
- **Operating Expenses**: $30,000
- **Mortgage Payments**: $20,000
- **Equity Invested**: $200,000

## Walk-through

1. **Calculate Effective Gross Income (EGI)**
   - $100,000 - $25,000 = $75,000
2. **Calculate Net Operating Income (NOI)**
   - $75,000 - $30,000 = $45,000
3. **Calculate Annual Pre-Tax Cash Flow**
   - Subtract debt service from NOI: $45,000 - $20,000 = $25,000
4. **Calculate Cash-on-Cash Return**
   - Divide pre-tax cash flow by equity invested: $25,000 / $200,000 = 12.5%

## Result

The property yields a 12.5% cash-on-cash return in the first year.

## What this teaches

- Factoring in debt payments shows the true short-term return on actual cash deployed.
- Cash-on-cash return represents a one-year snapshot of performance.
