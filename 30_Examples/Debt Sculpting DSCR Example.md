# Debt Sculpting DSCR Example

## Setup

We are calculating the sculpted principal repayment for a project in a given period based on its available cash flow.

## Assumptions

- Cash Flow Available for Debt Service (CFADS): $150
- Target Debt Service Coverage Ratio (DSCR): 1.5x
- Initial Debt Balance: $800
- Interest Rate: 10%

## Walk-through

1. Calculate the maximum Debt Service the asset can support:
   Debt Service = CFADS / DSCR
   Debt Service = $150 / 1.5 = $100
2. Calculate the Interest Expense for the period:
   Interest Expense = Initial Debt Balance * Interest Rate
   Interest Expense = $800 * 10% = $80
3. Calculate the Sculpted Principal Repayment:
   Principal Repayment = Debt Service - Interest Expense
   Principal Repayment = $100 - $80 = $20

## Result

The sculpted principal repayment in this period is $20.

## What this teaches

- The required principal repayment changes based on the available cash flows and the interest expense.
- The principal repayment will increase each year as the cash flow grows and the interest expense falls.
