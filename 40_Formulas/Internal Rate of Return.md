# Internal Rate of Return

$$ \sum_{t=0}^{T} \frac{CF_t}{(1+r)^t} = 0 $$

**Variables**

- $CF_t$ — Cash flow at time t
- $r$ — Internal Rate of Return (IRR)
- $T$ — Total number of periods
- $t$ — Time period

**Notes**

The internal rate of return is the rate `r` at which the Net Present Value (NPV) of all future cash inflows and outflows for a project is zero.
