# Present Value

$$PV = \frac{FV}{(1 + r)^n}$$

**Variables**

- $PV$ — Present value of the future cash flow
- $FV$ — Future value (the cash flow to be received in the future)
- $r$ — The discount rate or interest rate per period
- $n$ — The number of periods

**Notes**

This is the standard discounting formula, which is simply the algebraic rearrangement of the Future Value formula. It calculates what a future sum of money is worth today given a specific rate of return.
