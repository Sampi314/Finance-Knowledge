# Future Value

$$FV = PV \times (1 + r)^n$$

**Variables**

- $FV$ — Future value of the investment
- $PV$ — Present value (the initial amount invested)
- $r$ — The discount rate or interest rate per period
- $n$ — The number of periods

**Notes**

This is the standard compounding formula. If interest is compounded more frequently than annually, you must adjust $r$ and $n$ accordingly (e.g., for semi-annual compounding, use $r/2$ and $n \times 2$).
