# Power Query

> **TL;DR.** Power Query (also known as Get & Transform) is an Excel add-in that allows users to easily search, access, combine, and refine data from multiple sources.

## When you'd use this

You would use Power Query when you need to clean up mountains of uncleansed data, or when you find yourself spending hours each week copying and pasting data from one source to another. It helps automate data shaping and mind-numbing ETL (Extract, Transform, Load) activities.

## The 30-second version

Power Query, previously known as "Data Explorer" and rebranded as "Get & Transform" in Excel 2016, enhances self-service Business Intelligence in Excel. It enables users to discover, combine, and refine data for better analysis. Instead of manually cleaning data, you can build custom data sets quickly by downloading data tables into Excel in seconds. For Excel 2010 and 2013, it was available as a downloadable add-in, while in Excel 2016 and later, it is built-in on the 'Data' tab.

## Pitfalls and gotchas

- **Version Confusion**: Microsoft has renamed Power Query to "Get & Transform" in Excel 2016 and later. It was also previously known as "Data Explorer".
- **Feature Restrictions**: In older versions like Excel 2013, there was a "Power Query Premium" version with all features, and a "Power Query Public" version that lacked support for certain data sources like Active Directory, DB2, Oracle, and Salesforce.
