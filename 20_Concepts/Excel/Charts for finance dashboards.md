# Charts for finance dashboards

> **TL;DR.** Well-designed charts and dashboards turn raw data into information, enabling easier and more effective critical decision-making.

## When you'd use this

Charts and dashboards are essential when conveying complex financial models or datasets to stakeholders. They help highlight key messages without losing the audience in large tables of data.

## The 30-second version

Every Excel chart contains standard elements like a Chart Area, Plot Area, a Title, and at least one data series. By tweaking these components—such as fonts, colors, titles, and gridlines—you can significantly improve readability. Modifying scales, gaps between columns, or applying gradient fills allows for customization that ensures charts are both informative and visually appealing.

## The full explanation

Charts consist of standard pieces, but elements can vary based on chart type. Key parts of a standard chart include the Chart Area and Plot Area, both of which can be moved and resized. Additionally, elements like legends can be placed inside or outside the chart area.

You can format any element by right-clicking on it (e.g., "Format Plot Area"). There, properties like Fill, Border, Effects, and Text Options can be modified.

To make a chart stand out ("pop"), you might:
- Change fonts and sizes for titles and axes.
- Apply colors to the Plot Area, borders, or gridlines.
- Relocate the legend and apply shadows.
- Modify units and scales on axes.
- Reduce gaps between data columns.

## Pitfalls and gotchas

- Losing the key message by overloading the dashboard with too much data ("losing sight of the proverbial trees hidden in the corporate forest of data").
- Forgetting that not all chart elements can be moved (though changes can usually be undone with `CTRL + Z`).
