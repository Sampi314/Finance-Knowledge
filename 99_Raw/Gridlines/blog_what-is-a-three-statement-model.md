What is a Three Statement Model?
================================

*   July 18, 2022

A Three statement model is a basic setup for financial modelling.

**This type of model includes three different statements:**
-----------------------------------------------------------

*   Income statement
*   Balance sheet
*   Cash flow

These are all linked with Excel formulas.

The objective is to set it up so all the accounts are connected and a set of assumptions can drive changes in the entire model.

It’s essential to have a fundamental understanding of how to link the 3 financial statements.

This requires a solid foundation of accounting, finance, and Excel skills.
