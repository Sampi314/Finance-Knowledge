**Watch Video – How to Quickly Find Hyperlinks in Excel**

If you have a lot of [hyperlinks](https://trumpexcel.com/hyperlinks/)
 and these are scattered in the worksheet, it could be a herculean task to find these. Doing it manually can take you forever.

In this tutorial, you’ll learn how to quickly:

*   Find all the hyperlinks in Excel.
*   Find all the hyperlinks linked to a specific text.
*   Delete All Hyperlinks at One Go.

### Find All Hyperlinks in Excel

Suppose you have a dataset as shown below:

![Find and Remove Hyperlinks in Excel - Numbers](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20142%20333'%3E%3C/svg%3E)

Using [Find and Replace](https://trumpexcel.com/find-and-replace-in-excel/)
 feature, you can quickly find all the hyperlinks and give you a list of cells that have it.

Here are the steps:

1.  Go to Home –> Editing –> Find & Select –> Replace. This will open the Find and Replace dialog box (you can also use the [keyboard shortcut](https://trumpexcel.com/excel-keyboard-shortcuts/)
     **Control + H**).![Find and Remove Hyperlinks in Excel - Replace click](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20202%20190'%3E%3C/svg%3E)
2.  In the Find and Replace dialog box, within the Replace tab, click on Options button. This would show you an additional set of options available to you.![Find and Remove Hyperlinks in Excel - Options](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20449%20202'%3E%3C/svg%3E)
3.  In the Find what options, click on the small downward arrow on the Format button and select Choose Format From Cell.![Find and Remove Hyperlinks in Excel - Numbers choose format](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20564%20251'%3E%3C/svg%3E)
4.  Select any of the cells that has a hyperlink in it. This will populate the preview pane (on the left of the Format button) with the hyperlink format.![Find and Remove Hyperlinks in Excel - find preview](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20459%20249'%3E%3C/svg%3E)
5.  Click on Find All button. This will find all the cell with hyperlinks in it and show you a list as shown below.![Find and Remove Hyperlinks in Excel - number find all](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20471%20382'%3E%3C/svg%3E)

You now have a list of all the hyperlinks. You can select each of these cells individually or hold the control key and select multiple cells from the list.

Also read: [Extract URL from Hyperlinks in Excel](https://trumpexcel.com/extract-url-from-hyperlinks-excel/)

### Find All Hyperlinks Linked to Specific Text in Excel

Suppose you have a dataset as shown below and you only want to find hyperlinks from the cells that have the text ‘Company A’ in it.

![Find and Remove Hyperlinks in Excel - Company Data](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20164%20364'%3E%3C/svg%3E)

Here are the steps to do this:

1.  Go to Home –> Editing –> Find & Select –> Replace. This will open the Find and Replace dialog box (you can also use the keyboard shortcut **Control + H**).![Find and Remove Hyperlinks in Excel - Replace click](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20202%20190'%3E%3C/svg%3E)
2.  In the Find and Replace dialog box, within the Replace tab, click on Options button. This would show you an additional set of options available to you.![Find and Remove Hyperlinks in Excel - Options](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20449%20202'%3E%3C/svg%3E)
3.  In the Find what field, enter ‘Company A’.![Find and Remove Hyperlinks in Excel - find What field](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20456%20248'%3E%3C/svg%3E)
4.  In the Find what options, click on the small downward arrow on the Format button and select Choose Format From Cell.![Find and Remove Hyperlinks in Excel - Company A Choose Format](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20564%20248'%3E%3C/svg%3E)
5.  Select any of the cells that have the text ‘Company A’ and the hyperlink in it. This will populate the preview pane (on the left of the Format button) with the hyperlink format.![Find and Remove Hyperlinks in Excel - Company A Preview](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20460%20249'%3E%3C/svg%3E)
6.  Click on Find All button. This will find all the cells that have the text ‘Company A’ and also have the hyperlinks as shown below.![Find and Remove Hyperlinks in Excel - Company A Find List](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20473%20410'%3E%3C/svg%3E)

You now have a list of all the hyperlinks. You can select each of these cells individually or hold the control key and select multiple cells from the list.

### Delete All Hyperlinks at One Go

This one is easy.

To delete all hyperlinks at one go:

*   Select the entire worksheet (click on the tilted triangle at the top left of the sheet or use the keyboard shortcut Control + A).![Find and Remove Hyperlinks in Excel - Select All](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20262%20170'%3E%3C/svg%3E)
*   Right-click anywhere in the worksheet and select the ‘Remove Hyperlinks’ options.![Find and Remove Hyperlinks in Excel - Remove Hyperlinks](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20308%20410'%3E%3C/svg%3E)

This would instantly remove all the hyperlinks in the worksheet.
