In various fields like statistics, mathematics, and engineering, the sigma symbol (Σ or σ) is frequently used, often representing the sum of a set of terms or as a symbol for standard deviation in statistics.

There are two types of sigma symbols:

*   Σ – Greek Capital Letter Sigma
*   σ – Greek Small Letter Sigma

In this article, I’ll guide you through several straightforward methods to insert the Sigma symbol into your Excel documents.

### Copy and Paste the Sigma Symbol into Excel

The easiest way to get the sigma symbol in Excel is by copying it from a source and pasting it into your spreadsheet.

Below are the sigma symbols for you to copy.

**Greek Capital Letter Sigma**

Σ

**Greek Small Letter Sigma**

σ

Use Control + C to copy this symbol, then go to the desired cell in Excel where you want this symbol, and use Control + V to paste it.

### Keyboard Shortcut to Insert the Sigma Symbol in Excel

Below are the [keyboard shortcuts](https://trumpexcel.com/excel-keyboard-shortcuts/)
 to get the Sigma symbols in Excel:

**Greek Capital Letter Sigma** (Σ)

ALT + 228

**Greek Small Letter Sigma** (σ)

ALT + 229

Follow these steps to use this shortcut:

1.  Select the cell where you want the sigma symbol.
2.  Hold down the ALT key.
3.  While holding the ALT key, type 2, 2, and 8 on the numeric keypad (for inserting the capital letter sigma symbol).
4.  Release the ALT key.

Once you release the ALT key, the sigma symbol will appear in the selected cell.

Note: To use this shortcut, you need to use the Numeric keypad on your keyboard (it doesn’t work with a regular keyboard without a numeric keypad). Also, the Num Lock must be enabled.

**Pro Tip:** In MS Word, you can insert the sigma symbol by typing **03A3** and then pressing ALT + X to get the capital sigma symbol (Σ), or typing **03C3** and then pressing ALT + X to get the small sigma symbol (σ)

### Formula to Insert the Sigma Symbol

The sigma symbol can also be inserted using an Excel formula.

**Formula to get the Greek Capital Letter Sigma (Σ)**

\=UNICHAR(931)

![UNICHAR Formula to insert Capital Sigma symbol](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20492%20192'%3E%3C/svg%3E)

**Formula to get the Greek Small Letter Sigma (σ)**

\=UNICHAR(963)

![UNICHAR Formula to insert small Sigma symbol](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20501%20192'%3E%3C/svg%3E)

After the sigma symbol appears in the cell, you can [remove the formula](https://trumpexcel.com/remove-formulas-keep-data-excel/)
 to leave only the static symbol.

### Insert Sigma Symbol Using the Symbol Dialog Box

Another method is using the built-in Symbol dialog box in Excel:

1.  Select the cell where you want the sigma symbol.
2.  Click on the ‘Insert’ tab in the ribbon.

![Click on the insert tab in the ribbon](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20490%20223'%3E%3C/svg%3E)

3.  Click on the ‘Symbol’ icon to open the Symbol dialog box.

![Click on the symbol icon in Excel Ribbon](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20487%20223'%3E%3C/svg%3E)

4.  In the Subset dropdown, select ‘Greek and Coptic’.

![Click on the Greek and Coptic subset in the symbol dialog box](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20668%20512'%3E%3C/svg%3E)

5.  Locate the sigma symbol (Σ or σ) and double-click on it (or select it and click on the Insert button).

![Locate and double click on the Sigma symbol character](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20668%20512'%3E%3C/svg%3E)

This will insert the sigma symbol into the cell.

**Shortcut:** ALT + N + U can be used to open the Symbol dialog box quickly.

**Pro Tip:** To find the sigma symbol easily in the Symbol dialog box, enter 03C3 (for Σ) 03C3 (for σ) in the Character code field.

### Using the Character Map

Windows Character Map is a useful tool for inserting the sigma symbol:

1.  Search for ‘Character Map’ in the Windows search bar.
2.  Click on the Character Map App option.

![Open the character map app in windows](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20600%20383'%3E%3C/svg%3E)

3.  Check the ‘Advanced View’ options, which would make more options visible.
4.  In the Search field, type ‘sigma’ and hit enter.

![Search for sigma in character map app](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20555%20624'%3E%3C/svg%3E)

4.  Locate the sigma symbol, double-click it, then click ‘Copy’. This copies the sigma symbol to the clipboard.

![](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20555%20624'%3E%3C/svg%3E)

5.  Paste the symbol in Excel or any other application.

This method is not limited to Excel only and can be used across applications on Windows.

Also read: [Insert Pi Symbol (π) in Excel](https://trumpexcel.com/pi-symbol-excel/)

### Using the Equation Feature in Excel

When working with Greek symbols, you can also use this nifty Equation feature trick to get the symbol quickly.

Let me show you how it works:

1.  Click the Insert tab in the ribbon
2.  Click on the Equations icon.

![Click on the equation icon in the ribbon](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20522%20223'%3E%3C/svg%3E)

3.  In the Equations box that is inserted in the worksheet, remove what is already there and type the following text:

**For capital sigma symbol (Σ):**

/Sigma

**For small sigma symbol (σ):**

/sigma

4.  After typing the text, press the space bar key.

![Sigma symbol using equation text box](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20425%20227'%3E%3C/svg%3E)

When you hit the space bar key, the text would automatically (or should I say automagically) be converted into the sigma symbol.

You can now copy and use this symbol in any cell in Excel.
