Flash Fill is an amazing tool that was added in Excel 2013 and is available in all version after that.

You can use Flash Fill in Excel to make data entry easy. It’s a smart tool that tries to identify patterns based on your data entry and does that for you.

Some simple examples of using Flash Fill could be to get the first name from the full name, get name initials, format phone numbers, etc.

Using Flash Fill in Excel – Examples
------------------------------------

Here are some examples where flash fill can really save time by identifying patterns and making data entry easy.

### Example 1 – Get the First Name from Full Name

Suppose you have a dataset as shown below:

![Using Flash fill in Excel - Dataset](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20353%20201'%3E%3C/svg%3E)

Until Excel Flash Fill came into existence, the way to get the first name from the full name was to use [Text to Columns](https://trumpexcel.com/excel-text-to-columns-examples/)
 or use [Excel functions](https://trumpexcel.com/excel-functions/)
.

But now, you can instantly get the results using the new tool on the block – Flash Fill in Excel.

Here is how you can use Flash Fill in Excel to get the first name:

*   Select cell adjacent to the cell that has the data (B2 in this case).![Using Flash fill in Excel - Select first cell](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20353%20199'%3E%3C/svg%3E)
*   Enter the data (first name in this case).![Using Flash fill in Excel - Enter name](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20353%20200'%3E%3C/svg%3E)
*   Select the cell right below the one where you entered data (B3 in this case).![Using Flash fill in Excel - Select second cell](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20354%20200'%3E%3C/svg%3E)
*   Go to the Data tab and within Data Tools group, click on Flash Fill.![Using Flash fill in Excel - Excel Flash Fill](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20555%20126'%3E%3C/svg%3E)

Excel Flash Fill would automatically fill all the cells with the first name from the dataset.

![Using Flash fill in Excel - result](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20374%20200'%3E%3C/svg%3E)

It’s done in a flash (and hence the name Excel Flash Fill).

The keyboard shortcut to use Flash Fill in Excel is **Control + E**.

Note: Flash Fill is still not yet available in [Excel for Mac versions](https://answers.microsoft.com/en-us/msoffice/forum/msoffice_excel-mso_mac/flash-fill-for-mac/b7f9732e-c7a7-4147-96be-993c97459ff1)
.

### Example 2 – Get the Initials from the Name

Since Flash Fill works by identifying patterns, you can use it to get the [first name and last name](https://trumpexcel.com/separate-first-and-last-name-excel/)
 initials.

Suppose you have a dataset as shown below:![Using Flash fill in Excel - example 2 dataset](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20422%20201'%3E%3C/svg%3E)

Here are the steps use Flash Fill in Excel to get the initials:

*   Select cell adjacent to the cell that has the data (B2 in this case).
*   Enter the first name initial in the select cell.
*   Select the cell right below the one where you entered data (B3 in this case).
*   Go to the Data tab and within Data Tools group, click on Flash Fill.

Excel Flash Fill would automatically fill all the cells with the first name initials.

![Using Flash fill in Excel - initials](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20443%20200'%3E%3C/svg%3E)

### Example 3 – Concatenate Names in a Different Format

While you can use Flash Fill in Excel to split data (into first name or initials as seen in Example 1 and 2 above), you can also use it to [merge cells](https://trumpexcel.com/how-to-merge-cells-in-excel/)
.

Suppose you have a dataset as shown below:

![Using Flash fill in Excel - example 3 dataset](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20465%20200'%3E%3C/svg%3E)

The intent here is to combine the names in the following format – Last Name, First Name. For example, Tony and Stark would combine to give Stark, Tony.

Here is how you can use Flash Fill in Excel to combine names:

*   Select cell adjacent to the cell that has the data (B2 in this case).
*   Enter the cell with the combined name in the desired format.
*   Select the cell right below the one where you entered data (B3 in this case).
*   Go to the Data tab and within Data Tools group, click on Flash Fill.

Excel Flash Fill would automatically fill all the cells where the names are combined in the specified format.

![Using Flash fill in Excel - example 3 result](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20466%20203'%3E%3C/svg%3E)

### Example 4 – Get User Names from Email Addresses

Suppose you have a dataset as shown below:

![Using Flash fill in Excel - example 4 dataset](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20514%20199'%3E%3C/svg%3E)

The intent here is to get the username from the email id (for example, to get ‘tonystark’ from tonystark@ironman.com).

Here is how you can use Flash Fill in Excel to get the user names from email ids:

*   Select cell adjacent to the cell that has the data (B2 in this case).
*   Enter the cell with the username from the email id from the cell on the left.
*   Select the cell right below the one where you entered data (B3 in this case).
*   Go to the Data tab and within Data Tools group, click on Flash Fill.

Excel Flash Fill would automatically fill all the cells where the user names from the respective email ids.

![Using Flash fill in Excel - example 4 result](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20533%20201'%3E%3C/svg%3E)

### Example 6 – Extract Specific Data Points (such as City from Address)

You can use Flash Fill in Excel to extract a part from a data set.

Suppose you have a dataset as shown below:

![Using Flash fill in Excel - example 6 data](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20661%20178'%3E%3C/svg%3E)

The intent here is to get the state code from each address.

Here are the steps to get the state code from address using Flash Fill in Excel:

*   Select cell adjacent to the cell that has the phone number (B2 in this case).
*   In the cell, enter the state code from the address on the left.
*   Select the cell right below the one where you entered data (B3 in this case).
*   Go to the Data tab and within Data Tools group, click on Flash Fill.

This would instantly extract all the state codes in column B.

![Using Flash fill in Excel - example 6 result](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20610%20178'%3E%3C/svg%3E)

Similarly, you can extract pin codes as well.

Limitations of Flash Fill in Excel
----------------------------------

While Flash Fill is an amazing tool, it’s important to know it’s limitations before you use it on large and important datasets:

*   The result of Flash Fill is static. This means that if you change the original data set, the Flash Fill result would not change.
*   Flash Fill ignores cell with a space character in it. If you’re using Flash Fill, make sure the cells where the result would come don’t have any space character in it. If it does, Excel Flash Fill would simply ignore it.
*   In some cases, Flash Fill may falter in identifying the pattern giving incorrect results. For example, in the case of extracting the middle names, if there are names without any middle name, Flash Fill would give the incorrect results (as shown below).![Using Flash fill in Excel - incorrect](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20371%20200'%3E%3C/svg%3E)

Excel Flash Fill Error
----------------------

If Flash Fill is not able to recognize a pattern, it may show an error as shown below:

![Excel Flash Fill - Error](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20600%20111'%3E%3C/svg%3E)

In such case, you can fill a couple of more cells so that Flash Fill can recognize a pattern, or dump the idea of using Flash Fill and use Excel functions instead.
