As the world goes crazy (including me) over the new season of [Game of Thrones](https://en.wikipedia.org/wiki/Game_of_Thrones)
, I thought it would be a good idea to visualize its data by creating an [Excel Dashboard](https://trumpexcel.com/creating-excel-dashboard/)
.

Using my substandard secondary skills, I was somehow able to find enough data points to create an awesome Excel dashboard. There are many wonderful people/forums/wikis that helped me compile this dataset _(credits at the end)._

**Here is how the final Dashboard looks like:**

**Data Points used to create this dashboard:**

*   How many characters are Dead vs. Alive
*   How many Dead vs. Alive in prominent families (such as Baratheon, Lannister)
*   Major characters with their total screen time, dead or alive status, and the person’s name who portrayed the character
*   Episode titles with rating and viewership (and the functionality to sort based on it)
*   Character’s Profile – picture, allegiance, and how he/she died (if dead)

**What Excel Stuff makes this dashboard?**

*   [Scroll bar](https://trumpexcel.com/create-a-scroll-bar-in-excel/)
     to go through the list of characters.
*   [Radio button](https://trumpexcel.com/insert-use-radio-button-in-excel/)
     to sort the data.
*   [Check Boxes](https://trumpexcel.com/insert-checkbox-in-excel/)
     to show selected data only.
*   [Picture Look-up technique](https://trumpexcel.com/picture-lookup/)
    .
*   VBA to get the profile when double-clicking on character’s name.
*   Some back-end [Excel functions](https://trumpexcel.com/excel-functions/)
     to make it all work.

###### How to play with this Game of Thrones Dashboard?

*   Use check-boxes to get the list of characters with screen time, status, and the name of the person portraying the character (in the selected season(s)).![Games of Thrones Dashboard - Check Box](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20560%20284'%3E%3C/svg%3E)
*   Double Click on characters name to get the profile, which shows character’s picture, allegiance, and how the character died (if he/she did).![Game of Thrones - Character Profile - 2](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20100%20224'%3E%3C/svg%3E)
*   Use radio buttons to sort the episode list based on episode number, rating, or viewership.![Games of Thrones Dashboard - Episode List Sort](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20468%20276'%3E%3C/svg%3E)

##### Where did the data come from?

This was the hard part. I have used multiple sources to collate the data.

*   Screen time – Posted in a [forum](https://asoiaf.westeros.org/index.php?/topic/112748-a-game-of-screentime-season-4/)
     by a user “Sellsword”.
*   Character data and Images – From [Game of Thrones Wiki.](http://gameofthrones.wikia.com/wiki/Game_of_Thrones_Wiki)
    
*   Episode Data – [IMDB.](http://www.imdb.com/title/tt0944947/)
    
*   Images for Dead/Alive – [BandCollectables](http://www.bandbcollectables.co.uk/)
     website.
