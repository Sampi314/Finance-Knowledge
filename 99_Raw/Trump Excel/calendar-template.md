Monthly Excel Calendar Template
-------------------------------

Below is the demo of the monthly calendar template (the download file has been updated for 2023):

![Monthly Excel Calendar Template 2019 and 2020](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20724%20484'%3E%3C/svg%3E)

You can specify the year in cell B1 and use the [drop-down list](https://trumpexcel.com/excel-drop-down-list/)
 in cell B2 to specify the month.

Based on it, the calendar automatically updates.

You can also specify the holidays in the backend, and it automatically gets highlighted in the calendar (as yellow boxes).

You can also print this monthly calendar and write notes, plans, or reminders below it. Alternatively, you can write down the notes in Excel and then take a print.

Yearly Excel Calendar Template
------------------------------

Similar to the monthly calendar, there is also a yearly calendar (in a separate tab).

It’s a dynamic template, so you can enter the year number in cell B1, and it will automatically update the calendar for the specified year.

![Yearly Excel Calendar Template Demo 2019 & 2020](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20668%20468'%3E%3C/svg%3E)

You can specify the holidays in the template, and those days are automatically highlighted in the calendar.

The calendar has been made to fit in a single sheet when you print it. There is also some additional space below the calendar where you can write important notes.

**[Click here](https://www.dropbox.com/s/rozwb9rg0divsrc/Excel-Calendar-Template-2022.xlsx?dl=1)
** to download the Excel Calendar Template.

Print out of the calendar would look as shown below:

![Yearly Excel Calendar Template 2019 & 2020 with Notes](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20564%20680'%3E%3C/svg%3E)

Note that you can specify the holidays in the Holidays tab in the template.

Make sure that the date is entered in a format that Excel can understand as a date. For example, you can use the dd-mm-yyyy, or dd mm yyyy, or dd/mm/yyyy formats.

### What goes into making this Excel Calendar Template?

For the Excel enthusiasts, here is what it took to make this calendar template in Excel:

*   [**Excel Formulas**](https://trumpexcel.com/excel-functions/)
    : This template used functions such as DATE, WEEKDAY, DAY, ISNUMBER, COLUMNS, and [VLOOKUP](https://trumpexcel.com/excel-vlookup-function/)
    .
*   [**Conditional Formatting**](https://trumpexcel.com/excel-conditional-formatting/)
    : It is used to highlight the holidays in the calendar.
*   [**Drop-Down List**](https://trumpexcel.com/excel-drop-down-list/)
    : It is used to give an option to select the month in the monthly calendar.
*   **Custom Number Formatting**: It is used to show the numbers as months. For example, in the yearly calendar, The text January is actually number 1. Select cell B3 in the yearly calendar, and you will see that the formula bar shows 1. This number is shown as the [month name](https://trumpexcel.com/get-month-name-from-date-excel/)
     using custom number formatting.
