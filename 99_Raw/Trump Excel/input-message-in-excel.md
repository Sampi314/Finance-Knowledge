Input Message in Excel can be used to show a message when a cell is selected. It is particularly useful if you want to show some instructions to the user when he/she selects a cell. Examples could be to display form filling instructions or data entry instructions.

_Can’t this also be done using comments in cells?? – Well Yes!_

Yes, you can use cell comments as well. The only drawback with comments is that it is displayed only when you hover the mouse over the cell.

On the other hand, you can use Input Message in Excel to display a message whenever a cell is selected (using either mouse or keyboard).

Something as shown below:

![Input Message in Excel - Display Message when a cell is selected](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20326%20215'%3E%3C/svg%3E)

##### **How to Use Input Message in Excel**

1.  Select any cell and then go to Data tab –> Data Validation.![Input Message in Excel - Data Validation](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20653%20132'%3E%3C/svg%3E)
2.  In Data Validation dialogue box, select Input Message tab.![Input Message in Excel - Select Input Message](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20403%20324'%3E%3C/svg%3E)

1.  Ensure that “Show input message when cell is selected” check box is selected.![Input Message in Excel - check show input message](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20404%20325'%3E%3C/svg%3E)
2.  In the Input message tab, enter Title (max 32 characters, optional) and Input Message (max 256 characters).![Input Message in Excel - enter Title and Message](data:image/svg+xml,%3Csvg%20xmlns='http://www.w3.org/2000/svg'%20viewBox='0%200%20401%20321'%3E%3C/svg%3E)
3.  Click OK.

Now whenever you either click on the cell or select it using the keyboard, it would display the message.

_**Caution**_: If you move the message box from its position, then all the message box will be shown at that position only. So it is safe not to move the message box.
