### Excel links – after a long time

After a long time, I have a few more Excel links to share. Check out these for useful Excel tips, information & ideas.

**[Neat compilation of Excel keyboard shortcuts](http://exceljet.net/keyboard-shortcuts)
**  

Dave & Lisa (his wife) created this beautiful, searchable, PC & Mac compatible list of Excel keyboard shortcuts. Check it out to learn something useful. \[[More keyboard shortcuts](http://chandoo.org/wp/2010/02/22/complete-list-of-excel-shortcuts/)\
\]

**[Create small multiples (panel charts) automatically \[Excel workbook\]](https://img.chandoo.org/vba/Small%20multiples%205.xlsm)
**  

James sent me this beautiful Excel VBA file using which you can generate [small multiples chart](http://chandoo.org/wp/2011/11/16/small-multiples-charts-in-excel/)
 from a large, cluttered chart. You just enter all the data, create one big chart, tell how you want your small multiples and press a button – bingo, your small multiples are ready.

**[Making pretty annual report in Excel](http://darkhorseanalytics.com/blog/pretty-counts/)
**  

Joey at Darkhorse analytics discusses how we can create a beautiful annual report using Excel alone and why it is important to make your work look pretty as well. Check out the downloadable file and see it yourself. PS: You need install a font if you want to see it perfectly.

**[Building waffle charts in Excel](http://www.datapigtechnologies.com/blog/index.php/building-waffle-charts-in-excel/)
**  

Mike shows us how to create waffle charts, which are nothing but square pie charts. Very interesting technique. (You can [create waffles with conditional formatting](http://chandoo.org/wp/2008/07/10/partition-charts-visualization-fun/)
 too.).

**[Summing data across multiple sheets with multiple criteria](http://blogs.office.com/b/microsoft-excel/archive/2013/05/01/summing-data-across-multiple-criteria-on-multiple-worksheets.aspx)
**  

Liam shows us how to calculate sum of all values across multiple sheets that meet a given criteria using INDIRECT formulas, tables and Excel trickery.
