**If you work with multiple Excel workbooks everyday, then here is a handy tip.**

![Excel workspace feature - what is it and how to use it?](https://img.chandoo.org/q/excel-workspace-feature-how-to-use-it.png)

1.  ![Save Workspace button - Excel ribbon](https://img.chandoo.org/q/save-workspace-button-excel-ribbon.png)Open all the files you frequently use together.
2.  Go to View ribbon and click on Save Workspace
3.  Give a name to your workspace
4.  Now, whenever you need to open all these files together, just double click on workspace file in your explorer.
5.  You are done!

_**Workspace saves not just file collection, but also window layouts.**_ That means, if you like to arrange your Excel windows in a specific fashion, you can save this arrangement too.

Bonus tip: [Use specific filter criteria all the time? Try custom views](http://chandoo.org/wp/2008/12/17/custom-views-excel/ "Custom Views in Excel – Save filter and header settings for quick reference")

### Do you use workspace feature?

I find workspace feature to be very useful when I need to work with multiple files, especially in cases like file comparison and consolidation. I can save the file layout as a workspace and re-open it whenever I want to resume the work.
