Ok, enough self-promotion, let us take a look at this week’s excel links and articles from around web.

**[Understand Stock Charts in Excel](http://peltiertech.com/WordPress/2009/01/23/stock-charts-in-excel-2007/)
**

In this excellent tutorial, Jon explains stock charts in excel and how you can use them. Read it if you have never used a stock chart or don’t know what they are.

**[Choose the Right Chart Using this Flow Chart](http://www.flickr.com/photos/amit-agarwal/3196386402/sizes/l/)
**

This flow chart is very useful for deciding on the right type of chart for your data and visualization needs. Also checkout [Juice Analytics’ Chart Chooser](http://www.juiceanalytics.com/chartchooser/)

**[Quickly Create Named Ranges using F4 Key](http://blog.contextures.com/archives/2009/01/22/quickly-create-named-ranges-in-excel/)
**

Debra suggests using F4 key to creating named ranges in a snap. Very useful.

**[Print Multiple Areas from Several Sheets by Creating a Consolidation Sheet](http://excel.tips.net/Pages/T002845_Printing_Multiple_Selections.html)
**

Often you have a large workbook with several sheets and you want to print one consolidated worksheet with all the relevant areas from other sheets. You can create a consolidated sheet and use paste special to paste links to source ranges.

**[Get a List of Filenames in a Directory to Excel using Browser](http://spreadsheetpage.com/index.php/tip/getting_a_list_of_files_names_another_method)
**

You might need this trick if you use excel to manage lists of files (I do this all the time, when I am uploading a bunch of images to my site and need to create IMG tags)
