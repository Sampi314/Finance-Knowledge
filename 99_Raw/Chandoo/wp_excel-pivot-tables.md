What is a Pivot Table?
----------------------

**Excel Pivot Tables help you take a table (or list) of data and then create a report from it, _instantly_.**

For eg. you can take a bunch of sales data and then create a report on region-wise sales performance by Product.

![Excel Pivot Tables: Tutorial](https://chandoo.org/wp/wp-content/uploads/2009/08/excel-pivot-tables.png)  

How to Create a Pivot Table in Excel?
-------------------------------------

![Insert Pivot Table - Demo](https://img.chandoo.org/guides/pivot/insert-pivot-table.png)Creating a pivot table in Excel is very simple. Just follow these steps.

1.  Arrange your data in a table like fashion. Make sure there are no blank rows.
2.  Select your data and goto Insert > Pivot from Ribbon (or press ALT+DP)

**[Read this tutorial to learn how to create pivot reports](http://chandoo.org/wp/2009/08/19/excel-pivot-tables-tutorial/ "Excel Pivot Tables Tutorial : What is a Pivot Table and How to Make one")
**.

Top 5 Tips on Pivot Tables
--------------------------

[![Pivot Table Tricks](https://img.chandoo.org/p/pivot-table-tricks-excel.png)](http://chandoo.org/wp/2010/01/27/pivot-table-tricks/)
You can do a lot of reporting & analysis using Pivot Tables. Keep these 5 tips in mind next time you are using Pivot Tables. Click on the links to view the tip in detail.

1.  [Drill down pivot tables](http://chandoo.org/wp/2010/01/27/pivot-table-tricks/#drilldown)
    : You can drill-down and get details by just double clicking on a value.
2.  [Change Summary from Total](http://chandoo.org/wp/2010/01/27/pivot-table-tricks/#summary_type)
    : You can change summaries in Pivot Reports from Total to Count, Average or something else very easily. Just use Value Field settings.
3.  [Slice & Dice Pivots](http://chandoo.org/wp/2010/01/27/pivot-table-tricks/#slice_dice)
    : You can move anything to anywhere in pivot reports and Excel would instantly change the report layout and calculations.
4.  [Difference from last month](http://chandoo.org/wp/2010/01/27/pivot-table-tricks/#diff_from)
    : You can easily display the difference from last month by changing value field settings.
5.  [Calculated Fields in Pivots](http://chandoo.org/wp/2010/01/27/pivot-table-tricks/#calculated_fields)
    : You can make custom calculations in Pivot Reports by adding adding calculated fields.

Advanced Pivot Table Techniques
-------------------------------

There are many things we can do with Pivot Tables to analyze data or do complex reporting. Go thru these advanced pivot table techniques to learn more.  

### Create Monthly, Quarterly or Yearly reports from Daily Data:

Using “Group” feature of Excel, you can easily create monthly or quarterly reports from daily data. Or, create show summaries by hour from a transaction data etc. The possibilities are many.  
[![Group by Quarter & Month in Pivot Tables](https://chandoo.org/img/p/pivot-table-group-by-quarter-month.png "Group by Quarter & Month in Pivot Tables")](http://chandoo.org/wp/2009/11/17/group-dates-in-pivot-tables/ "Grouping Dates in Pivot Tables")

[**Learn more about grouping dates in pivot reports**](http://chandoo.org/wp/2009/11/17/group-dates-in-pivot-tables/ "Grouping Dates in Pivot Tables")
.  

### Use Report Filters to Create Many Reports from Same Data

We can use _report filters_ feature of Excel Pivot Tables to create multiple reports from same data in just a few clicks.

[**Learn more about pivot table report filters**](http://chandoo.org/wp/2011/04/20/pivot-table-report-filters/ "What are Pivot Table Report Filters and How to use them?")
.  

### Use Slicers to create interactive dashboards in Excel

In Excel 2010, Microsoft has introduced a feature called as **_slicers_**. Think of slicers as filters, only more visual. So, if you add a slicer for **_department_**, you would see the list of all departments in box. You can just click on “marketing” to see the report for that department alone. We can use this feature to create an interactive dashboard in Excel, like below.

[![Using slicers to create a dynamic dashboard in Excel](https://chandoo.org/img/dashboards/dynamic-excel-dashboard-using-pivot-slicers-demo.png)](http://chandoo.org/wp/2010/12/08/dynamic-dashboard-video-tutorial/)

**[Learn more about Slicers & using them to create dashboards](http://chandoo.org/wp/2010/12/08/dynamic-dashboard-video-tutorial/)
**.
