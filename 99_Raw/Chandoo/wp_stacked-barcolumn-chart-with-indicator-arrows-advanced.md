Last week I wrote a post on how to develop a Stacked Bar/Column Chart with an indicator arrow which you can read here: [http://chandoo.org/wp/2016/09/12/stacked-bar-and-indicator-arrow-chart-tutorial/](http://chandoo.org/wp/2016/09/12/stacked-bar-and-indicator-arrow-chart-tutorial/)

[![sba01](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA01-1.png)](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA01-1.png)

This week I have taken that technique and added some high-performance steroids.

I have allowed for three (High, Medium and Low) Indicator arrows, which can be in any order

[![sba04](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA04-1.png)](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA04-1.png)
 [![sba03](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA03.png)](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA03.png)
 [![sba02](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA02-1.png)](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA02-1.png)

I am not going to go through how to construct it, as it is exactly the same as last weeks post, except that it needs a few helper cells.

[![sba06](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA06-1.png)](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA06-1.png)

It can be applied to both Column Charts as above or Bar Charts as below:

[![sba08](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA08-1.png)](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA08-1.png)

The other thing to remember is that you can use any Shape as the indicators.

[![sba07](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA07-1.png)](https://chandoo.org/wp/wp-content/uploads/2016/09/SBA07-1.png)

Please [download the sample file here](https://chandoo.org/wp/wp-content/uploads/2016/09/Column-chart-with-Multiple-Indicators.xlsx)
, pull it apart and see what you think.
