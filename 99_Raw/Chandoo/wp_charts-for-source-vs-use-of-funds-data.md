Let’s say you manage a fund or charity. You get money from various places and you use that money for various reasons. How do you tell the story of **source vs. uses of funds?** In this post, let’s review 14 charting options.

Source vs. Use of funds – Sample Data for this problem
------------------------------------------------------

Let’s say you have 3 sources and 3 uses, as shown below. For all these charts, we will use the same data, but may add some simple calculations or change the layout to get the charts.

![data-for-source-vs-uses](https://chandoo.org/wp/wp-content/uploads/2019/04/data-for-source-vs-uses.png)

As I made quite a few alternatives, I broke them in to two kinds of charts.

*   **Simple charts:** These are easy to make, easy to read and scale well when you have lots of sources & uses.
*   **Exotic charts:** These are somewhat tricky but fun to make, look exotic and raise a few eyebrows in the meeting room.

Simple Charts for Source vs. Use of funds
-----------------------------------------

### #1 – Waterfall chart

![](https://chandoo.org/wp/wp-content/uploads/2019/04/14-waterfall-chart-default.png)

Waterfall chart is perfect for this kind of data. Simply layout your data like below (you can [use SUBTOTAL](https://chandoo.org/wp/subtotal-formula-excel/)
 or SUM formulas to calculate intermediate totals), select everything and insert waterfall chart from Insert ribbon. Mark total columns “as total” and you are done.

![](https://chandoo.org/wp/wp-content/uploads/2019/04/data-for-waterfall-chart-2.png)

### #2 – Stacked column chart

![](https://chandoo.org/wp/wp-content/uploads/2019/04/01-stacked-column-charts.png)

![Data for stacked column chart - source vs. use of funds](https://chandoo.org/wp/wp-content/uploads/2019/04/data-stacked-charts.png)

Set up your data in two columns as shown aside and create stacked column chart. Format to your taste. Simple and effective.

### #3 Couple of Pies

![](https://chandoo.org/wp/wp-content/uploads/2019/04/02-couple-of-pie-charts.png)

Use the same data format as “stacked column chart” but instead make two pie charts. If you have just a few options for sources and uses, Pie charts work great.

### #4 – Mirrored bar chart

![](https://chandoo.org/wp/wp-content/uploads/2019/04/03-mirrored-stacked-bar-chart.png)

As you can see, sources and uses are two sides of the same coin, so we can go with a mirror metaphor like above. To create this chart, you need data in the format shown below. Once you have the data, just select it and insert a stacked bar chart. Format as you see fit and the mirror is ready.

![data format for mirrored stacked chart.](https://chandoo.org/wp/wp-content/uploads/2019/04/data-mirrored-bar-chart.png)

### #5 – Dollar background – stacked chart

![](https://chandoo.org/wp/wp-content/uploads/2019/04/04-stacked-bar-with-background.png)

This is same is option #2, but with a fancy dollar bill for plot area background and gap width for columns set to 0. When you make the fill **_colors for series semi-transparent_**, you get this effect.

### #6 – Touching stacked bars…

![](https://chandoo.org/wp/wp-content/uploads/2019/04/05-touching-stacked-bars.png)

Again, same format of data as “stacked chart #2”, but we make a stacked bar chart and then set gap width to 0 to get this effect. This works very well and makes it easy to compare.

### #7 – Donuts

![](https://chandoo.org/wp/wp-content/uploads/2019/04/06-donut-chart.png)

While I am not a fan of donuts or pies, they work great when you have small set of data like this. Set up your data like #2 stacked chart and instead make donut chart. Format and you get this.

### #8 – Half a pie each…

![](https://chandoo.org/wp/wp-content/uploads/2019/04/07-mirroed-pie-chart.png)

![](https://chandoo.org/wp/wp-content/uploads/2019/04/data-for-pie-chart.png)

This works very well **if source total = use total**. Just set up your data like this and make a normal pie chart. Format sources and uses in two colors like above and _**draw a line in the middle**_. You get above chart.

### #9 – Sunburst chart

![](https://chandoo.org/wp/wp-content/uploads/2019/04/08-sunburst-chart.png)

![](https://chandoo.org/wp/wp-content/uploads/2019/04/data-treemap-sunburst-charts.png)

This is essentially similar to donuts (option #7) but we need data in a different structure (see aside). Sunburst charts work very well even when you have lots of items in each category or even sub-categories. Keep in mind though, sunburst charts are new to Excel 2016, so if you have older versions, you can’t get these.

### #10 – Treemap chart

![](https://chandoo.org/wp/wp-content/uploads/2019/04/09-treemap.png)

Same as sunburst charts, treemaps are a new chart in Excel 2016. They can work well when you have many items or even sub-categories.

Exotic charts for Source vs. Use of funds
-----------------------------------------

### #11 – Hourglass chart

![](https://chandoo.org/wp/wp-content/uploads/2019/04/10-funnelish-chart.png)

**_The idea is Sources flow to Uses_**. So we make an hour glass shape with regular bar chart (with invisible series on left to push the items to center).

To create this, set up your data like below.

![](https://chandoo.org/wp/wp-content/uploads/2019/04/data-funnel-chart.png)

Now, simply create a regular stacked bar chart. Select Gap series, make it invisible (fill color > none), format source and uses series. You are done.

### #12 – Butterfly chart

![](https://chandoo.org/wp/wp-content/uploads/2019/04/11-buttery-fly-chart.png)

Sources and Uses as wings of butterfly. This is really an area chart, with an invisible series at the bottom to create the effect of butterfly wings. To make this chart, set up your data like below:

![data for butterfly chart](https://chandoo.org/wp/wp-content/uploads/2019/04/data-butterfly-chart.png)

Now, Make an area chart. Move gap series to top (so it is really at the bottom when laid out), then make it invisible. See this GIF to understand how it all works to make the butterfly wings.

![How to make butterfly chart - demo](https://chandoo.org/wp/wp-content/uploads/2019/04/butterfly-chart-howto-excel.gif)

### #13 – Waterfall lines

![](https://chandoo.org/wp/wp-content/uploads/2019/04/13-waterfall-line-chart.png)

Instead of waterfall bars, we make this with lines. Unfortunately, this step chart is the trickiest one to create as we use XY plot with error bars to get the steps. The technique has too many steps, so I am not explaining them here, but if you are curious, [see this tutorial](https://chandoo.org/wp/story-of-change/)
.

### #14 – Cube chart

![](https://chandoo.org/wp/wp-content/uploads/2019/04/12-cube-chart.png)

Here the idea is Sources and Uses are two faces of a cube. Again, this is actually an area chart, but invisible gap series at bottom and a top layer. To make this, set up your data like below (remember, the horizontal axis has to be dates, else you will not get Cube effect).

![](https://chandoo.org/wp/wp-content/uploads/2019/04/data-cube-chart.png)

To be honest, I am not a fan of cube chart, as it is tricky to label things and won’t work for more items. But I wanted to give it a try.
