Lookups for Data Analysis
=========================

What is this course?
--------------------

Do you want to know how to use various lookup formulas in Excel? **_from beginner to advanced level?_** This course is for you. 

**You will learn:**

*   Overview of new ways to lookup – XLOOKUP, VLOOKUP, HLOOKUP, INDEX+MATCH & FILTER
*   How to use these functions alone or combine with others?
*   Advanced lookup scenarios in business analysis  
    *   2 way lookups (both row & column conditions)
    *   Lookup and return all matching results
    *   Lookup with multiple criteria
    *   Using derived columns in lookups
    *   Combining two tables with lookups
    *   Data extraction with lookups
*   Dealing with 8 common errors in Lookups
