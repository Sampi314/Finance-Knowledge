Pivot tables are powerful. Use these 6 tricks to save time when working with them.

![Excel Pivot Table Tricks to save your time and make you awesome](https://chandoo.org/wp/wp-content/uploads/2020/07/Excel-pivot-tables-time-saving-tricks.jpg)

New to Pivot Tables? Check out this intro Are you new to Pivot Tables or just used them a few times? Then check out this excellent [getting started with Pivot Tables guide](https://chandoo.org/wp/excel-pivot-tables-tutorial/)
.

1 - Double click to see details
-------------------------------

Ever wondered what numbers add up to the total in a pivot table value area? Simply double click to investigate. Excel will show all underlying values. 

**Extra tip:** Don’t forget to delete such detail worksheets as they can create duplicate data.

![pivot tricks - 1 - drill to details](https://chandoo.org/wp/wp-content/uploads/2020/07/pivot-tricks-1-drill-to-details.gif)

2 - Format value fields quickly
-------------------------------

The default value field formatting is useless. Often, we need to change the values to **currency** or **accounting** formats. You can do this quickly by right clicking on the values instead of going to Value Field Settings > Number Formats route.

See this image to understand the process.

![pivot trick 2 - format quickly](https://chandoo.org/wp/wp-content/uploads/2020/07/pivot-trick-2-format-quickly.png)

🙏 Thanks to Srinivas who taught me this trick on my YouTube channel ([link](https://www.youtube.com/watch?v=Kg7aZBtAWd0&lc=UgyDHX-9LAWKxTRnWgx4AaABAg)
)

3 - Include filtered items in subtotals
---------------------------------------

Let’s say you have a pivot table with Sales by product & region. You filter out some products. The **grand totals change.** This can be inaccurate.

If you want to to include _**filtered items in the totals**_ you can use the option in “Subtotals” menu in ribbon. Oh wait a sec, that is grayed out 🤔🤔🤔

![pivot tricks 3 - include filtered values in totals - disabled](https://chandoo.org/wp/wp-content/uploads/2020/07/pivot-tricks-3-include-filtered-values-in-totals-disabled.png)

Turns out you can ungray it. All you need is to use **data model.** Click on “Add to data model” in the insert pivot screen you are able to use this feature. 

![](https://chandoo.org/wp/wp-content/uploads/2020/07/pivot-tricks-3-include-filtered-values-in-totals-enabled.png)

🙏 Thanks to Ryan who taught me this tip on my YouTube channel ([link](https://www.youtube.com/watch?v=S9_COxQ-_8E&lc=UgxkoyCamDI3V2KlWxN4AaABAg)
)

4 - Consistent pivot charts with a simple trick
-----------------------------------------------

Let’s say you have a pivot chart linked to a slicer showing sales by product in _selected region._ As you don’t sell all products in all regions, the chart looks jumpy with inconsistent axis.

_**Something like this:**_

![pivot tricks 4 - inconsistent axis](https://chandoo.org/wp/wp-content/uploads/2020/07/pivot-tricks-4-inconsistent-axis.gif)

You can make the axis consistent by formatting the row label to show “items with no data”. To do this, select the row labels area of the pivot, right click and go to “Field settings”. Now from display area, enable “Items with no data” option.

![pivot tricks 4 - items with no data option](https://chandoo.org/wp/wp-content/uploads/2020/07/pivot-tricks-4-items-with-no-data-option.png)

Your chart looks much better.

![pivot tricks 4 - consistent axis](https://chandoo.org/wp/wp-content/uploads/2020/07/pivot-tricks-4-consistent-axis.gif)

5 - Repeat row labels
---------------------

Tabular form is my favorite layout for pivot. It looks clear and easy on eyes. 

Do you know that you can enable “Repeat item labels” option to make the tabular layout even better.

![pivot tricks 5 - repeat item labels](https://chandoo.org/wp/wp-content/uploads/2020/07/pivot-tricks-5-repeat-item-labels.png)

**Bonus tip:** When you repeat item labels, you can use Pivot Tables in other formulas (like SUMIFS) easily. 

6 - Multiple pages with Report Filters
--------------------------------------

You got a handy little pivot report that details the performance of a region. Now you want to make similar reports for rest of the 16 regions too.

You can use Report Filters to quickly create all versions of this report in one click. Here is a demo of how report filters work.

![Report Fitters in Excel - used to create multiple versions of a report - Demo](https://img.chandoo.org/pivot/pivot-table-report-filters-demo.gif)
