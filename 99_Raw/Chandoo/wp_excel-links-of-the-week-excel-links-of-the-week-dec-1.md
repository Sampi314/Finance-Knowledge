Excel Links of the Week \[Dec 1\]
=================================

 - 0 comments

  

Every Monday PHD shares some of the best excel related articles around the web so that you are up to date with discussions and new ways to be productive. Here is a list of links recommended for this week.

[**Clean up data using magic and copy-paste**](http://cleanupdata.com/)

Unclean data is probably the most time consuming part of any analytical work. I have spent countless hours cleaning few hundred rows of data only to spend another 5 minutes to get the insights. The cleanupdata.com promises to automate the drudgery. All you have to do is paste the unclean data and tell the website how you want to process it and it would do the dirty work.

[**Learn Excel 2007 through video**](http://www.free-training-tutorial.com/)

If you are an excel beginner and would like to see some videos to learn basic concepts like writing formulas, using auto-fill, the above site is a great place to start.

[**Make a chart and win Tufte books**](http://flowingdata.com/2008/12/01/contest-win-two-edward-tufte-books-enter-now/)

Nathan at FlowingData is giving away 2 Edward Tufte’s books: [The Visual Display of Quantitative Information, 2nd edition](http://www.amazon.com/gp/product/0961392142?ie=UTF8&tag=poinhairdilb-20&linkCode=as2&camp=1789&creative=9325&creativeASIN=0961392142)
![](https://www.assoc-amazon.com/e/ir?t=poinhairdilb-20&l=as2&o=1&a=0961392142) and [Visual Explanations](http://www.amazon.com/gp/product/0961392126?ie=UTF8&tag=poinhairdilb-20&linkCode=as2&camp=1789&creative=9325&creativeASIN=0961392126)
![](https://www.assoc-amazon.com/e/ir?t=poinhairdilb-20&l=as2&o=1&a=0961392126). All you need to do is make a good chart of US Immigration data that is shown in his post.

[**Filter data by Months using Advanced Filters and VBA**](http://www.dailydoseofexcel.com/archives/2008/11/26/autofiltering-on-months/)

DailydoseofExcel presents a cool and simple vba script that would help you filter data by month. (you can also do this by using advanced filter and setting month start and end date as criteria)

[**What is a OLAP cube?**](http://blog.xlcubed.com/the-basic-strucure-of-a-cube/)

XLCubed blog explains the basics of OLAP (online analytical processing) and cubes. This is useful if you have never or rarely used pivot tables and need to understand when to use them.

[**Subscribe to PHD and stay upto date on all things excel and charting**](http://feeds.feedburner.com/PointyHairedDilbert)

Have a fantastic week ahead. 🙂
