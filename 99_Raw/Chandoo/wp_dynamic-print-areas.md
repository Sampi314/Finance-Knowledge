Yesterday, you learned about [Print Areas](http://chandoo.org/wp/2015/08/23/print-areas/)
 – a time & paper saving feature of Excel. While print areas are great, you can only set up one print area per sheet. _**What if you want to print either report or data based on user selection?**_ 

In such cases, you can set up _**dynamic print areas.**_ 

That is right. See below demo to understand how it looks. Read on to learn how to set up dynamic print areas.

![24-dynamic-print-areas-demo](https://chandoo.org/wp/wp-content/uploads/2015/08/24-dynamic-print-areas-demo.gif)

### Set up dynamic print areas – tutorial:

1.  In your spreadsheet, select any range of cells and [create a print area](http://chandoo.org/wp/2015/08/23/print-areas/)
     (click on the link for instructions).
2.  Now, identify the all ranges that may be printed. To keep it simple, let’s say there are 2 such ranges. Give them names like _report_ and _data._ 
3.  Set up a selection mechanism thru form controls so that user can pick what they want to print. Something like this:![print-preferences-form-controls](https://chandoo.org/wp/wp-content/uploads/2015/08/print-preferences-form-controls.png)  
    Link the option buttons to a cell, say **M8**.
4.  Go to Formulas ribbon tab and click on Name Manager.
5.  You can see the Print\_Area named range, pointing to the range you originally selected.![regular-print-area-named-range](https://chandoo.org/wp/wp-content/uploads/2015/08/regular-print-area-named-range.png)
6.  Edit the range and write a [CHOOSE formula](http://chandoo.org/wp/2014/07/16/introduction-to-choose-function/)
     to return either _report_ or _data_ range based on M8 (form control) value. Like this:  
    \=CHOOSE($M$8, report, data)![dynamic-print-area-named-range](https://chandoo.org/wp/wp-content/uploads/2015/08/dynamic-print-area-named-range.png)

That is all. Your print area is now dynamic.

### Download dynamic print areas demo workbook

[Please click here to download dynamic print areas workbook](https://chandoo.org/wp/wp-content/uploads/2015/08/dynamic-print-areas.xlsx)
. Examine the formula for Print\_Area named range to understand better.

### Other creative ways to use print areas:

*   Your users can see something but print an alternative range (may be B/W compatible), with print areas.
*   Your print areas can depend on latest date (for example, print coupon before expiry date and print alternative material after expiry)
*   Your users can print reports at various levels – summary or detailed with dynamic print areas.
