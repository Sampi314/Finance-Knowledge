Focus on Details = Satisfied Customers
======================================

[Random](https://chandoo.org/wp/category/uncategorized/)
 - 0 comments

  

It is amazing how attention to smaller details can change the face of a service organization. There are several things that are done just the way they are since they were first done.

Ok, thats a complex statement, for example, when you go to a website and it has some data/animation to load, all you see is some lousy 584.5495049.54895o4% of 489234893248bytes loaded type of message. Now, as an eager customer waiting to see behind this status bar what would those numbers tell you? nothing, probably if you are math inclined, you start calculating how much more time it would take. But all of us have been using Microsoft products since slice bread that we know instantly these times/numbers do not make sense. so how about putting a welcome change? [CNN-IBN](http://ibnlive.com/)
 did just this. And instead of the lousy status displayer you have this funny message (below). I caputured 3 frames, but to see the effect read any article on [IBNLive.com](http://ibnlive.com/)
.
