### How does this Work?

This workbook is just a clone of [Suicides vs. Murders visualization](http://chandoo.org/wp/2011/09/09/suicides-murders-interactive-chart/)
 with different data. The only change is that, here we used [LARGE](http://chandoo.org/excel-formulas/large.html)
 and [MATCH formulas](http://chandoo.org/wp/2010/11/02/how-to-lookup-values-to-left/)
 instead of [COUNTIF formula](http://chandoo.org/excel-formulas/countif.html)
 to sort the list.

Learn more about this technique is from [KPI Dashboard Sorting](http://chandoo.org/wp/2008/08/27/excel-kpi-dashboard-sort-2/)
 article.
