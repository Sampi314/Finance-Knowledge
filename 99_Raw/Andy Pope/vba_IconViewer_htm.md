Icon Viewer
-----------

![](https://www.andypope.info/vba/IconViewer.png)

  
The viewer displays all the available msoID images listed in the worksheet.  
You can use the Search box to reduce the icons displayed.  
If you double click an icon the msoId will to outputted to the worksheet when the form is closed, along with the image.  
  
Depending on the Excel version you are using some images may not be available and will not be display  
You can add new msoId names to the list as and when they become known.
