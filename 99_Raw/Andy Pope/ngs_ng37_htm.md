##### Create a radar chart based on the range A2:A9

![](https://www.andypope.info/ngs/ng37b.gif)

 

##### Use the Source data dialog to add an extra data series

![](https://www.andypope.info/ngs/ng37c.gif)

##### Change the new data series to an xy scatter chart and then set the x values to the range C2:C8 and the y values to D2:D8.

##### Format the secondary y axis to have a minimum value of -3 and a maximum of +3  
Format the secondary x axis to have a minimum value of -1 and a maximum of -1

![](https://www.andypope.info/ngs/ng37e.gif)

##### Format the data points to none. Apply data labels, displaying the y value.  
Change the alignment of the data labels to Left.

##### The horizontal lines are positive X error bars using a fixed value of 1.

![](https://www.andypope.info/ngs/ng37f.gif)

##### Finally finish off the formatting of the error bars and the value axis of the radar chart, removing the grid lines.

##### ![](https://www.andypope.info/ngs/ng37g.gif)
