##### Use the chart wizard to create a line chart based on A1:G9  
![](https://www.andypope.info/charts/lablast_1.gif)

##### Select the 4th data series 'Label Last 1' and format series to have border and marker None. Also enabled Data Label Value. Repeat for 5th and 6th data series.  
![](https://www.andypope.info/charts/lablast_2.gif)

##### Format the plot area as required. Here I have removed the border, major gridlines and changed the plotarea colour.  
![](https://www.andypope.info/charts/lablast_3.gif)

##### Finally remove the additional data series from the legend.  
Select the legend and then select the data series. Make sure the selection include both marker and text otherwise the data will be deleted.  
![](https://www.andypope.info/charts/lablast_4.gif)
