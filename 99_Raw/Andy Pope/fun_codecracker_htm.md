### Code Cracker

 

|     |     |     |
| --- | --- | --- |
|     |     |     |
|     |     | ![](https://www.andypope.info/fun/Codecracker.gif) |
|     | ##### This is a variation on the Mastermind game. See if you can crack the 4 colour code.  <br>  <br>Drag & drop the coloured pegs on to the board.  <br>If you have already placed a peg and want to change your mind simply drag & drop another coloured peg on it.  <br>You can do this as much as you like until you are ready to check your code against the hidden one.<br><br>##### Each black score peg means you have correctly match the colour and position in the code.  <br>The white score peg means you have match the colour but not the location.  <br>Obviously the position of the scoring pegs has no relation to the position of your code, that would be too easy!  <br>  <br>Due to the use of raised events in the code this will only work with xl2000 and greater. |     |
| ##### I must mention [Andrew Engwirda](http://blog.livedoor.jp/andrewe/archives/50105543.html)<br> who's own version of the Mastermind game inspired, _or should that be distracted_, me into putting this game together. Thanks Andrew! |     |
