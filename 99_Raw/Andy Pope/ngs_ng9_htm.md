|     | Re: Venn Diagrams<br>----------------- |
|     | ##### The [workbook](https://www.andypope.info/ngs/venn.zip)<br> contains 3 examples of  a venn diagram.<br><br>##### The circles were constructed using auto-shapes. These were then taking into MS Paint where sectors were flood-filled with  colour. The venn image was then used as the picture fill for the plot area. A data series is used to plot the labels in the correct positions. These labels can be changed by simply modifying the text in the linked worksheet cells. If necessary the text position within a sector can by changed simply by altering the X and or Y values.  <br>These examples are all about presentation and do not decode data in anyway.  <br>![](https://www.andypope.info/ngs/venn2.GIF)<br><br>![](https://www.andypope.info/ngs/venn3.GIF)<br><br>![](https://www.andypope.info/ngs/venn4.GIF)<br><br>##### ![](https://www.andypope.info/ngs/venn4e.gif) Ellipse venn provided by Lilly Rosier |
