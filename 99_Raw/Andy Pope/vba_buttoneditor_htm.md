### Button Editor

  

![](https://www.andypope.info/vba/buted.gif)
--------------------------------------------

 

##### This [add-in](https://www.andypope.info/vba/ButtonEditor.zip)
 allows you to edit button images.

##### To install the add-in use the Tools > Add-ins menu and then browse for the ButtonEditor.xla file. This will add a new menu item to the Tools menu.

##### Once the editor is open you can use the tools (pencil, line, oval, rectangle, floodfill) to draw an image.  
  
You can also load a 16x16 pixel bitmap direct from file.  
  
Or you can copy an existing button. For users of XL2000 and later this can be done whilst the editor is open. For XL97 you should copy the image prior to opening the editor.
