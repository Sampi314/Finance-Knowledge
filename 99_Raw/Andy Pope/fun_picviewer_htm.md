### Picture viewer

| --- | --- |
|     | [![Picture viewing](https://www.andypope.info/fun/picview.gif)](https://www.andypope.info/fun/picviewer.zip)<br><br>------------------------------------------------------------------------------------------------------------- |
|     | ##### This workbook contains a very simple database of employees for a fictitious company.  <br>Using a few formula and a modified XY scatter chart you can make images appear when there corresponding name is selected from the list. The example contains no macro code.  <br>  <br>I have previously solved this type of problem using named pictures and macro code to either  <br>A) move the pictures into or out of view from a location way off screen or even another worksheet.  <br>  <br>B) change the z-order of a picture  <br>  <br>Details of setup and usage are stored within the [download](https://www.andypope.info/fun/picviewer.zip)<br>. |
