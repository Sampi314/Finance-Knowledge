### Selecting chart elements using the cursor keys

You can use the cursor keys to cycle through the elements in a chart. The up and down keys move through the main elements whilst the left and right can be used to move through the items in a element.

![](https://www.andypope.info/tips/tip003a.gif)

Starting with a chart that has its chart area selected a press of the up arrow would then select....

![](https://www.andypope.info/tips/tip003b.gif)  
Plot area

![](https://www.andypope.info/tips/tip003g.gif)  
Legend

![](https://www.andypope.info/tips/tip003c.gif)  
Value axis

![](https://www.andypope.info/tips/tip003d.gif)  
Chart Title

![](https://www.andypope.info/tips/tip003e.gif)  
Major gridlines

![](https://www.andypope.info/tips/tip003f.gif)  
Data Series

  
With an element selected use of the right arrow would then select an item within that element...

![](https://www.andypope.info/tips/tip003h.gif)  
From Legend to Legend

![](https://www.andypope.info/tips/tip003j.gif)  
From Data Series to Data Point

![](https://www.andypope.info/tips/tip003i.gif)  
From Value axis to Category axis

Some items within an element even have sub items which using the left and right arrows will allow you to select ...

![](https://www.andypope.info/tips/tip003k.gif)  
From Legend Entry to Legend Marker
