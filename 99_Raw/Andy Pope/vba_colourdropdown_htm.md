### Coloured Dropdown control

[![](https://www.andypope.info/vba/colourdrop.gif)](https://www.andypope.info/vba/colourdropdown.zip)
  
The download contains a demonstration and overview of a class that will generate a coloured dropdown list on a userform.  
  
All the code is available for view.  
  
[![](https://www.andypope.info/site_images/nav_download.png)](https://www.andypope.info/vba/colourdropdown.zip)
  
  
Following feedback I have created a download that will function within Excel97.  
This uses public sub routines to replace Events.  
  
Excel 97 version  
[![](https://www.andypope.info/site_images/nav_download.png)](https://www.andypope.info/vba/colourdropdown97.zip)
