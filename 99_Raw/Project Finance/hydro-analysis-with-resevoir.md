# Hydro Project Finance Models

**Source:** https://edbodmer.com/hydro-analysis-with-resevoir

---

This page includes some hydro project finance models.  Hydro includes a lot of construction risk and issues of the value of re-financing. When projects are developed with high IRR’s and when the projects use straight line depreciation a lot of distortions can occur.  I hope some of this issues are illustrated below.
