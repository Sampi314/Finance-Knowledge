# Downloading Files with Macros

**Source:** https://edbodmer.com/downloading-files-with-macros

---

.

A lot of the files on this website have macros. It seems that these days when you try to download a file with a macro, you are not allowed to and you get a pink message. To resolve this you can do a couple of things. The first is to go to internet options in the from the windows search bar and then add this website to your secured websites.

Other options include copying the file to another folder with a new name and then add the file to that folder. This is a pain if you want to download a bunch of files.

.

.
