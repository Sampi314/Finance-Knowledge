# Example with Saudi Stocks

**Source:** https://edbodmer.com/example-with-saudi-stocks

---

This page illustrates how to use the database using a few stocks from Saudi Arabia. These stocks have code numbers from the Tadawul index. You can see that when you enter the stocks, you need to put in the country code for the market watch data. For the Yahoo ticker use .SR at the end. Thanks to Abdullah for the suggestion.
