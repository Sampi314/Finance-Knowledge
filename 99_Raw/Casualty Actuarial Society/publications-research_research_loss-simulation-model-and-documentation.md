Loss Simulation Model and Documentation
=======================================

CAS Loss Simulator 2.0 is an open source R package (cascsim.zip), helping actuaries perform transaction data fitting and claim-level loss reserving. It comes with an Excel Front End to assist users with an easy to use interface.

1.  [**How to Run Loss Simulator 2.0**](https://www.casact.org/sites/default/files/2023-03/HowToRunCASSimulator2.docx)
     (Read this first; advanced users download product from step 2)
2.  [**Loss Simulator 2.0 Excel**](https://www.casact.org/sites/default/files/2023-03/cascsim_R4_2023.xlsm)
     (You only need this file to run)
3.  [**R Source Code**](https://www.reserveprism.com/Simulator2/src/RSourceCode.zip)
     (DIY code for advanced R users)
4.  [**Loss Simulator 2.0 Architecture**](https://www.reserveprism.com/Simulator2/doc/Simulator2Architecture.pdf)
    (PDF)
5.  [**Loss Simulation 2.0 Methodology**](https://www.reserveprism.com/Simulator2/doc/Simulator2Methodology.docx)
    
6.  [**R Package Example**](https://www.reserveprism.com/Simulator2/doc/Simulator2RExample.docx)
     (Run R package directly)
7.  [**Validation Report**](https://www.reserveprism.com/Simulator2/doc/Simulator2ValidationReport.docx)
    
8.  [**Download a testing claim data**](https://www.reserveprism.com/Simulator2/testingclaim/test2_claim.csv)
