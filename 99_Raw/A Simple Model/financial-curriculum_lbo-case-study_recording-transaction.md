Recording the Transaction
=========================

In this course you will learn how to update the balance sheet for a control equity buyout. The process will include developing a framework for the balance sheet adjustments, and then building a sources and uses table to populate the balance sheet adjustments.

[![this image is related to 1](https://portal.asimplemodel.com//Assets/Files/c7a97780-3lbo-solution-curriculum-title-pic_purchase-acct_1024_tinypng.png)](https://portal.asimplemodel.com/Subscribe.aspx)

In this video you will learn how to build a sources and uses table for an LBO model. In an LBO model the sources and uses table is a convenient way to track the sources and uses of cash required to close a transaction.
