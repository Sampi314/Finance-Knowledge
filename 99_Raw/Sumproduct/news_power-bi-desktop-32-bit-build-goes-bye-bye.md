Power BI Desktop 32-bit Build Goes Bye Bye
==========================================

![](https://sumproduct.com/wp-content/uploads/2025/06/image-14.png)

As part of Microsoft’s ongoing efforts to enhance the performance and security of Power BI, the company has announced the deprecation of the 32-bit build of Power BI Desktop.  Starting in August 2025, there will no longer be any 32-bit builds for Power BI Desktop supported.

The main headlines are as follows:

*   **Deprecation timeline:** the 32-bit build will be deprecated in the August 2025 release.  Customers using the 32-bit build are encouraged to transition to the 64-bit build to continue receiving updates and support
*   **Benefits of transitioning to 64-bit:** the 64-bit build provides enhanced performance, better security and support for the latest features and improvements in Power BI Desktop.

It is recommended all users to make the switch to the 64-bit build to take full advantage of these benefits.  It can still work in an otherwise 32-bit environment.
