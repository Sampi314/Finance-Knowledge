Course Overview
---------------

#### **Overview:**

This course highlights key mistakes and design aspects to avoid when creating PivotTables, giving insight on the design approaches to take to ensure that your PivotTables are not only robust, but efficiently highlight trends in data.  You will also learn what sort of Tables to create depending upon the type of data you are working with, plus how to format and set value fields, apply filters, grouping and slicers, and be able to create PivotCharts / develop an interactive dashboard.  

#### **Delivered:**

Online, enrolment for 12 months.  

#### **Course prerequisites:**

While there are no entry requirements a basic understanding of Microsoft Excel is recommended.
