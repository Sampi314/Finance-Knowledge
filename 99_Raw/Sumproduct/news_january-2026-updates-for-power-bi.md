January 2026 Updates for Power BI
=================================

![](https://sumproduct.com/wp-content/uploads/2026/01/image-88.png)

2026 sees more updates, important announcements and some big changes, especially around feature retirements, and the latest in AI / Copilot advancements.  New features include new reporting tools, revised data modelling options, enhanced resources for developers, broader global standards support and upgraded visualisations.

There is more though and the full list of updates is as follows:

**_General_**

*   Deprecation of Power BI Q&A
*   SQL Server Reporting Services Report Viewer Webpart Support discontinuation
*   The Power BI application is accessible within Teams on Microsoft 365 Operated by 21Vianet

**_Copilot and AI_**

*   Attach reference to Copilot chat in the Power BI mobile application
*   Updating ‘Prepped for AI’ setting name to ‘Approved for Copilot’
*   Standalone Copilot access on Power BI Home

**_Reporting_**

*   Field parameters persist hierarchy level setting
*   Format pane improvements
*   Introducing Markers in Azure Maps visual
*   Modern visual ToolTips (Generally Available)
*   Column sizing update in Tables and Matrices

**_Modelling_**

*   Updates on editing Semantic Models in Power BI Service
*   Chinese coded character set (GB18030-2022) support

**_Developers and APIs_**

*   Power BI enhanced report format (PBIR) as default format

**_Visualisations_**

*   Funnel chart by Powerviz.
