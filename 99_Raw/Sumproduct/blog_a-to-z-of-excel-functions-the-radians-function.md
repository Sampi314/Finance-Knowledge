**The RADIANS function**

This function appears to have cornered the market regarding converting degrees into radians.

![](<Base64-Image-Removed>)

The **RADIANS** function employs the following syntax to operate:

**RADIANS(angle)**

The **RADIANS** function has the following argument:

*   **angle:** this is required and represents the **angle** in degrees that you wish to convert.
