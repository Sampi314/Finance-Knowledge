Power Query: Power Query Online – Part 4
========================================

30 August 2023

_Welcome to our Power Query blog. Today, I continue looking at Power Query Online._

In the current series, I am looking at Power Query Online, which I have accessed from Power Apps:

![](https://sumproduct.com/wp-content/uploads/2025/05/0be10eb79a96a80d7daf33f297448cb7.jpg)

[Last week](https://sumproduct.com/blog/power-query-power-query-online-part-3/)
, from the views I could choose, I was looking in detail at ‘Diagram view’:

![](https://sumproduct.com/wp-content/uploads/2025/05/246675c90b965dd9103be2f85b4bbf50.jpg)

I noted that the plus (+) icon on the expanded view of the queries provides me with a searchable menu, which allows me to choose any functionality:

![](https://sumproduct.com/wp-content/uploads/2025/05/abd8f21fc681f8f86fbb52ea8dcb51d8.jpg)

There is also a search feature is also available from the main screen, allowing me to find any functionality without searching through the tabs. Here however, the functionality is arranged _mostly_ alphabetically (more on this in a moment) within ‘Actions’ and ‘Get data’ sections:

![](https://sumproduct.com/wp-content/uploads/2025/05/fb67996dfb18d0c9cb824bee85ff49c4.jpg)

Note also, that if the functionality is available on the tabs, the location is given:

![](https://sumproduct.com/wp-content/uploads/2025/05/122b016e5aec0158d6d4bd6e863772be.jpg)

If there are more options than can be shown on the dropdown, I have the option to view more options using ‘See more results for…’:

![](https://sumproduct.com/wp-content/uploads/2025/05/1a6207984bea783dc2034b69087440ed.jpg)

When I click on this, I can access a scrollbar:

![](https://sumproduct.com/wp-content/uploads/2025/05/872e44f207de7121e27865a8850be1c0.jpg)

Note one interesting feature – any commands that are all in lowercase appear at the end and not alphabetically:

![](https://sumproduct.com/wp-content/uploads/2025/05/59041b5fd56f1fe3114985bcc3cd3348.jpg)

As in the ‘lowercase’ example, if the function appears on more than one tab, or more than once on a tab, each method of access appears. This does bloat the list of commands, and is arguably not necessary. Having said that, being able to find the functionality in this way is a great addition to Power Query, and would be useful in the desktop version too.

I will look at more functionality in Power Query Online next time.

Come back next time for more ways to use Power Query!
