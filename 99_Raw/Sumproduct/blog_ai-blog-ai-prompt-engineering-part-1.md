__Welcome back to our AI blog. Today, we will talk about prompt engineering, which can help you effectively interact with AI tools and get the expected results.__

**_What is prompting?_**

**_Quick Review_**

The reason why effective prompt engineering is important is that AI language models, including ChatGPT, Gemini, Claude and so on, generate responses based upon the input prompts they received.  The more accurate, detailed and well-structured prompts are the more precise and relevant responses will be.

But how can we identify if a prompt is effective?  How can we improve our input to get the expected output?  First of all, we need to understand what kind of elements make a good prompt.

**_Five-Step Framework_**

![](https://sumproduct.com/wp-content/uploads/2025/05/image-37.png)

**Task: Clearly define what you want AI to do**

For example, you can ask ChatGPT**: “**_Give me suggestions to improve my oral English_**”.** 

You will get a long response that provides practical suggestions on daily speaking, listening and repeating, expanding vocabulary, _etc_.

![](<Base64-Image-Removed>)

The output looks acceptable, but it is still very general, so you will probably require a more specific, tailored response.  You may elevate the output by using two \[2\] more elements.  The first of these is a persona, which means assigning a role to the AI model to instruct it to behave as a particular character or expert.

For example, you can edit the previous prompt to act as an English teacher to make an oral English improvement plan for you.

![](<Base64-Image-Removed>)

We can see that the results are much more specific than before, with finer granularity and a clearer breakdown into sections.
