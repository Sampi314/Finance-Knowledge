December 2022 Updates for Power BI
==================================

14 December 2022

![](<Base64-Image-Removed>)

The latest updates see Microsoft add a slew of DAX functions (well, three anyway!), improve slicer type formatting and bring Metrics to the Windows application. The complete list for this month is as follows:

**_Reporting_**

*   Slicer type formatting moved to Format pane

**_Modelling_**

*   Making it easier to do comparison calculations (introducing the **INDEX**, **OFFSET** and **WINDOW DAX** functions)

**_Data connectivity and preparation_**

*   Anaplan (connector update)
*   Azure Databricks, Databricks (connector update)
*   CData Connect Cloud (new connector)
*   Cosmos DB V2 (new connector)
*   Dremio Cloud (connector update)
*   Google BigQuery (Azure AD) (new connector)

**_Service_**

*   Power BI org app Multiple Audiences will be Generally Available next year
*   Announcing the deprecation of ‘Getting Started’ in the Expanded View of Power BI Service
*   Upcoming changes to the ‘Get Data’ experience in Power BI Service

**_Mobile_**

*   Track your Metrics on the Windows app

**_Developers_**

*   Dynamically setting data chunk size

**_Visualisations_**

*   New visuals in AppSource
*   Update on Charticulator status.
