Report View Copilot Chat Pane Upgrade Required
==============================================

![](https://sumproduct.com/wp-content/uploads/2025/06/image-95-1024x609.png)

To continue using the Report view Copilot chat pane, Microsoft has announced that you will need to upgrade to the February 2025 version.  It has been stated that the ‘Report view copilot chat’ pane will probably not work as expected for versions before February 2025 after 30 April 2025.

In the meantime, please remember we offer training in Power BI which you can find out more about [here](https://sumproduct.com/courses/)
.  If you wish to catch up on past articles, you can find all of our past Power BI blogs [here](https://www.sumproduct.com/thought/power-bi-tips-blog-series)
.
