New Features for Excel September 2025
=====================================

![](<Base64-Image-Removed>)

The big announcement this month is Agent Mode in Excel, which brings expert-level automation, refining results with AI so you can focus on guiding the outcome.  The recently discussed [**COPILOT**](https://sumproduct.com/news/introducing-excels-new-copilot-function/)
 function is also rolling out to Excel for the web users soon through Microsoft’s Frontier program.  And then there is formula completion with Copilot in Excel, making it easier than ever to work with formulae.  And don’t forget the new Python Initialization Editor now available to Insider users on Windows!

Here is this month’s full list:

**_Excel for the web_**

*   Agent Mode in Excel (Frontier)
*   **COPILOT** function (Frontier)

**_Excel for Windows and the web_**

*   Formula completion with Copilot in Excel

**_Excel for Windows_**

*   Python in Excel: editable initialisation pane (Insiders)

**_Excel for iOS_**

*   Copilot-powered file previews (Insiders).
