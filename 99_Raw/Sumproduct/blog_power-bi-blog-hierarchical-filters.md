Power BI Blog: Hierarchical Filters
===================================

21 May 2020

_Welcome back to this week’s edition of the Power BI blog series. This week,_ _we look at_ _a new Preview feature in Power BI, Hierarchical Filters._

Microsoft has introduced a new Preview feature to Power BI that has been long requested: ‘Hierarchical Filters’. Let’s take a look at this dataset, for example:

![](<Base64-Image-Removed>)

We’ll load this dataset into Power BI and create a hierarchy for ‘**Product Type**‘ and ‘**Sub-Category**‘:

![](<Base64-Image-Removed>)

Now we can create a slicer with the ‘**Product Type Hierarchy**‘ field:

![](<Base64-Image-Removed>)

Notice that even though we have a hierarchy in this field we can only see the ‘**Product Type**‘ field. Instead, let’s enable the Preview feature and see what happens.

![](<Base64-Image-Removed>)

After we have enabled it, the slicer changes to display the different hierarchical values within each ‘**Product Type**‘:

![](<Base64-Image-Removed>)

We now have the ability to select the ‘**Sub-Category**‘ within each ‘**Product Type**‘. We can even select a combination between two different ‘**Sub-Category**‘ values in two separate ‘**Product Type**‘ fields, _viz._

![](<Base64-Image-Removed>)
