**The LEN Function**

The **LEN** function returns the number of characters in a text string. It employs the following syntax to operate:

**LEN(text)**

The **LEN** function has the following argument:

*   **text:** this is required and represents the text string whose length you wish to ascertain.

It should be further noted that:

*   spaces count as characters in **text**.
