AI Blog: Agent Mode in Excel Part 1
===================================

____Welcome back to our AI blog series!  After Copilot in Excel, Microsoft has quietly introduced another game-changing feature — Agent Mode in Excel.____

If Copilot in Excel (introduced in a previous [blog](https://sumproduct.com/blog/ai-blog-copilot-in-excel-smart-suggestions-and-automation/)
) feels like a smart assistant who answers your questions within seconds, Agent Mode will probably seem more like a colleague who can take your brief, automate repetitive work, create new sheets and return with a bunch of outputs.  All you need to do is describe the outcome you want.

![](<Base64-Image-Removed>)

Agent Mode is an experimental capability surfaced through **Excel Labs** that lets Excel act on multi-step instructions.  Some commentators have gone as far to say that Agent Mode is what Copilot should have been.

Instead of asking a single question such as, “What’s the average by region?”, you can outline a mini project:  
_“Clean this dataset, identify key patterns, create visuals, and summarise the results on a dashboard.”_

In the process, the agent interprets your intent, reads the data and performs a sequence of actions including creating new tabs, inserting formulae and assembling charts.  The experience is less chatty and more outcome oriented.  The mindset shifts from _“Please answer this one question”_ to _“Please get this done”._

![](<Base64-Image-Removed>)

**_What is the difference between Agent Mode and Copilot?_**

Whilst Copilot provides some interactive guidance such as answering questions, generating formulae and suggesting next steps, Agent Mode is aimed at execution over a chain of steps instead.  In other words, Copilot usually works within the current sheet and returns an answer or suggestion, whereas Agent Mode can create structure: new worksheets, visuals, PivotTables, dashboards, and summary tabs.

Copilot helps you think faster and explore ideas more deeply, whilst Agent Mode works like an AI analyst that takes a defined scope and delivers complete outputs.  Copilot focuses on quick insights and smart suggestions; Agent Mode operationalises (great word!) the process end-to-end.

![](<Base64-Image-Removed>)

**_Why Agent Mode matters_**

Analysts tend to dislike small tasks such as importing, cleaning, reshaping, calculating, visualising and formatting, especially when deadlines are tight.  Agent Mode helps to compress those steps.  For routine workflows (_e.g_. month-end sales analysis), assigning most of repetitive tasks to an agent can save hours each week and lead to fewer errors compared with manual work.

**_What you need before you start_**

To access this Excel feature, you need to have a Microsoft 365 subscription and a Copilot licence.  At the time of writing, Agent Mode is currently only available in Excel for the web.

![](<Base64-Image-Removed>)

To enable it, open **Home** -> **Add-ins** -> search “**Excel Labs**”.  Under experimental features, turn on **Agent.**

![](<Base64-Image-Removed>)

**_How to use it_**

To begin with, you can try a short prompt instead of a complicated task such as:

_“Use the data in Source Data tab to profile sales, find the top growth products line and build a one-page summary with charts.”_

From there, the agent typically creates working tabs (for cleaning, grouping or calculations), a summary tab with visuals.  It’s not perfect (no AI is), but the behaviour is consistent enough that you can plan around it.  

**_Current limitations_**

Agent Mode is still new, so it’s not perfect yet:

*   if your work uses many linked sheets or Power Query, the agent may not handle it well
*   sometimes it creates too many sheets or charts, so you might need to tidy up
*   as Microsoft 365 introduced, Agent Mode doesn’t currently support a web search, enterprise search, file grounding or integration with external data sources
*   not everyone can access Agent Mode right now, but Microsoft is rolling it out gradually, just like Copilot.

**_Word to the Wise_**

Agent Mode is Excel’s next step beyond Copilot.  It turns your prompts into projects, handling multiple actions and generating end-to-end outputs.  While it still has limits, its potential to automate structured workflows is immense.

If Copilot helps us _think_ faster, Agent Mode helps us _work_ faster.
