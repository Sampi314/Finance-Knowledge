3 March 2023

_Welcome back to our Charts and Dashboards blog series. This week, we offer a tip when creating a Line chart._

We have some simple sales data:

![](https://sumproduct.com/wp-content/uploads/2025/05/1bb07bf52b72f87a3c95c410faf9143f.jpg)

We are going to create a Line chart to view the trend of the **Total** by **Month**.

To do this, we select the **Month** column, and then whilst holding down **CRTL**, we select the **Total** column. Then, on the Insert tab, we select the dropdown next to the Line chart icon:

![](https://sumproduct.com/wp-content/uploads/2025/05/aa8860804972f49d4911295bb5caf408.jpg)

We choose the first ‘2-D Line’ and create the chart.

![](https://sumproduct.com/wp-content/uploads/2025/05/08d46a367980ddc39166b3cb654f8ff2.jpg)

Notice how the line doesn’t reach the edge of the chart?

![](https://sumproduct.com/wp-content/uploads/2025/05/abad1475c56ad4862a5b91a4fa4c1657.jpg)

We can change this. If we select the horizontal axis and double-click, the ‘Format Axis’ pane appears:

![](https://sumproduct.com/wp-content/uploads/2025/05/667e720a27f0db734f04291edccb54b0.jpg)

If we choose the ‘Axis Options’, we can change the ‘Axis Position’ from ‘Between tick marks’ to ‘On tick marks’.

![](https://sumproduct.com/wp-content/uploads/2025/05/b80a5fe89db9ab70caad67ba4ffabf8d.jpg)

This removes the gap, and makes the chart look neater:

![](https://sumproduct.com/wp-content/uploads/2025/05/5ca96e1ea22be79ecf19cbfd78883c32.jpg)

Next time, we’ll look at how we can create this chart for our sales results:

![](https://sumproduct.com/wp-content/uploads/2025/05/bb0fd451e15f9f58a7136f0dea0d92bd.jpg)

_That’s it for this week, come back next week for more Charts and Dashboards tips._
