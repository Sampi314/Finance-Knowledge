New Features for Excel August 2025
==================================

![](https://sumproduct.com/wp-content/uploads/2025/08/image1-14.png)

August sees the formal introduction of the new **COPILOT** function in Microsoft Excel for Windows and Excel for Mac, bringing the power of large language models (LLMs) directly into the grid and makes it theoretically easier to analyse text, generate content and work more efficiently.  The **COPILOT** function is now rolling out to Beta Channel users with a Microsoft 365 Copilot license.

Other features rolling out this month include ‘Explain this formula’ with Copilot in Excel, analyse images with Python in Excel, and more.  Here is this month’s full list:

**_Excel for Windows and Mac_**

*   **COPILOT** function (Insiders)

**_Excel for Windows and the web_**

*   Explain formulae on the grid with Copilot in Excel

**_Excel for Windows, Mac and the Web_**

*   Analyse images with Python in Excel (Insiders)

**_Excel for Windows_**

*   New pen tools and Draw tab customisation (Insiders)

**_Excel for web_**

*   Edit data label text in Excel for the web.
