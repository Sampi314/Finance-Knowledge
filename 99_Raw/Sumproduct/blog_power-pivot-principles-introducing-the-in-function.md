Power Pivot Principles: Introducing the IN Function
===================================================

27 November 2018

_Welcome back to our Power Pivot blog. Today, we discuss the IN function, one of the several logical functions in Power Pivot._

The **IN** function is a logical function that returns with TRUE for the values in the row of data that match the input value.

For example, we can use the **IN** function together with the **CALCULATE** function (you can read about the **CALCULATE** function [here](http://https//www.sumproduct.com/blog/article/power-pivot-principles/ppp-introducing-calculate)
), to create a filtered measure that sums the sales for products that are classified as ‘Tanning Products’ and ‘Skin Care’.

![](https://sumproduct.com/wp-content/uploads/2025/05/e774d10cbbb9450fc45efbe51abdf434-527.jpg)

The results are somewhat similar to filtered measures (read more about filters [here](https://www.sumproduct.com/blog/article/power-pivot-principles/ppp-filters)
