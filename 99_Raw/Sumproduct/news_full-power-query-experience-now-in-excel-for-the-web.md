Full Power Query Experience Now in Excel for the Web
====================================================

Microsoft has some great news for Excel for the web users: apparently, the “full Power Query user experience” (including crashing?)is now Generally Available.  This includes the import wizard and Power Query Editor.

There are some restrictions on this.  According to the official literature:

*   Viewing and refreshing queries is available to all Microsoft 365 subscribers
*   The “full Power Query experience” is available to all Microsoft 365 Subscribers with Business or Enterprise plans.

So not quite “the full experience” then – _yet._

At the time of writing, security seems a little too tight with credential challenges happening frequently, but it is a step in the right direction.

![](<Base64-Image-Removed>)

You can import data into Excel using Power Query from a wide variety of data sources, _e.g._ an Excel Workbook, Text / CSV, XML, JSON, SQL Server Database, SharePoint Online List, OData, Blank Table and a Blank Query.

You can read more about this in our February newsletter (subscribe now if you don’t already).  We have already been testing this all out. Check out our first impressions in this [Power Query blog](https://sumproduct.com/blog/power-query-now-in-excel-for-the-web-properly/)
