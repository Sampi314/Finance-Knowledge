Define New Measures in DAX Query View Quick Queries
===================================================

14 November 2024

This one is short and sweet!

Creating measures in DAX query view has become a little simpler recently.The ‘Quick queries’ option available from the context menu of tables, columns or other items in the Data pane, now includes ‘Define new measure’, _viz_.

![](https://sumproduct.com/wp-content/uploads/2025/05/6fcfb69c2b7837f739dbfcb602e1db05.jpg)

This will create a new query table with the syntax started for you to create a query-scoped measure **DAX** formula ready for you to add your own **DAX** formula and then run when ready.
