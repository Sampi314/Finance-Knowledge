Charts and Dashboards: Learning our Lines – Part 3
==================================================

17 March 2023

_Welcome back to our Charts and Dashboards blog series. This week, we offer a tip to enhance a chart that shows the deviation from the average._

We have some simple sales data:

![](https://sumproduct.com/wp-content/uploads/2025/05/1a63728d60230125bb6652017d1e6eaf.jpg)

In [Part 1](https://sumproduct.com/blog/charts-and-dashboards-learning-our-lines-part-1/)
, we created a Line chart to view the trend of the **Total** by **Month**. We looked at how to remove the gap between the end of the line and the ends of the horizontal axis:

![](https://sumproduct.com/wp-content/uploads/2025/05/0e8d9a4d321f3d2ad04cce1773a5fe12.jpg)![](https://sumproduct.com/wp-content/uploads/2025/05/b5825c321df1c05686cbc63a0c5e3d13.jpg)

[Last time](https://sumproduct.com/blog/charts-and-dashboards-learning-our-lines-part-2/)
, we created this chart:

![](https://sumproduct.com/wp-content/uploads/2025/05/f41f543154f68df7d8e63a8f85e37382.jpg)

This time, we’ll look at how we can create a variation of this chart to highlight the negative and positive deviation:

![](https://sumproduct.com/wp-content/uploads/2025/05/eb5bb6030a049ef7f22c0bd8d3cef804.jpg)

We start with the data again, but this time, in addition to the columns we added [last week](https://sumproduct.com/blog/charts-and-dashboards-learning-our-lines-part-2/)
, we create an extra column:

![](https://sumproduct.com/wp-content/uploads/2025/05/05cae44482975f1ff418eb66b535f559.jpg)

**Average2** is a duplicate of **Average**. We have arranged them in this order to make them easier to select for our chart, as we will show later.

We then select the **Month** column, and the four \[4\] new columns, and choose a Line chart, as we did [last week](https://sumproduct.com/blog/charts-and-dashboards-learning-our-lines-part-2/)
:

![](https://sumproduct.com/wp-content/uploads/2025/05/2947d2195c8d00d9ba2b8dae02390e4b.jpg)

We can select the chart and change the chart type by using the option on the ‘Chart Design’ tab:

![](https://sumproduct.com/wp-content/uploads/2025/05/7fc0ac994138a0c5be321a12b0feef16.jpg)

We change to a Combo chart:

![](<Base64-Image-Removed>)

We then change **Average** to a Line, and **Max**, **Average2** and **Min** to Area charts. This means that we have a Line chart _and_ an Area chart for the average values.

![](<Base64-Image-Removed>)

Next, we select the **Max** data series and double-click to access the ‘Format Data Series’ pane:

![](<Base64-Image-Removed>)

We change the Fill to ‘Solid fill’ and choose a green colour:

![](<Base64-Image-Removed>)

We can then use the dropdown next to ‘Series Options’ to select each of the other series.

![](<Base64-Image-Removed>)

We change each Area series to ‘Solid Fill’ and change the colour of **Average2** to red, and set the Transparency to 70%. This is how we will highlight the negative deviation. We change the colour of **Min** to white:

![](<Base64-Image-Removed>)

Note that we needed to select the data in the correct order to get the correct overlapping here. We change **Average** to a black solid line and reduce the Width:

![](<Base64-Image-Removed>)

We also apply the fix we looked at in [Part 1](https://sumproduct.com/blog/charts-and-dashboards-learning-our-lines-part-1/)
 to make sure our data reaches the edges of our chart by formatting the horizontal axis so that the ‘Axis position’ is ‘On tick marks’:

![](<Base64-Image-Removed>)

We can then delete the legend and change the title:

![](<Base64-Image-Removed>)

Our chart is complete, and we should probably check what is happening in March and November!

_That’s it for this week, come back next week for more Charts and Dashboards tips._
