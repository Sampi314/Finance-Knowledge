12 November 2021

_Welcome back to our Charts and Dashboards blog series. This week, I return to Sparklines._

The basics of Sparklines were covered in our [Charts and Dashboards Blog: Sparklines](https://sumproduct.com/blog/charts-and-dashboards-sparklines/)
. But let’s revisit. Consider the following:

![](https://sumproduct.com/wp-content/uploads/2025/05/e774d10cbbb9450fc45efbe51abdf434-109.jpg)

I have some simple sales figures, and I have used an example of each type of Sparkline.

*   **F4** is a Line
*   **F5** is a Column
*   **F6** is a Win / Loss.

In each case, I have selected the cell I want the Sparkline to appear in, and then from the ‘Insert’ tab I chose the relevant chart type from the Sparklines section:

![](https://sumproduct.com/wp-content/uploads/2025/05/f32e5a15e2cf9c3e4d2d058458ce054d-147.jpg)

I can then specify the range and location in the dialog that appears:

![](https://sumproduct.com/wp-content/uploads/2025/05/f1140ff857fc3b6f5f97a6a24f4a6fc7-110.jpg)

Each type of Sparkline is dynamic in several respects:

*   if the underlying data set changes, then the Sparkline will reflect this
*   if the cell size containing the Sparkline is changed, then the Sparkline remains in proportion with the cell.

To demonstrate, I change one of the values for Newbie and widen column **F**. This will change the Sparklines:

![](https://sumproduct.com/wp-content/uploads/2025/05/72aa864d2854c6fefb1083fba0ab5792-108.jpg)

The Win/Loss Sparkline only shows an increase or decrease, so for Newbie, -5 is represented by the same thickness of bar as -3. Changing the values for Newbie only changes the Sparkline if the sign of the number changes (or becomes zero \[0\], as in this example).

If I were to change the data range a Sparkline is based upon, I can select the Sparkline and choose the Sparkline tab where I have an ‘Edit Data’ dropdown.

![](<Base64-Image-Removed>)

For this example, I choose the ‘Edit Single Sparkline’s Data’ option:

![](<Base64-Image-Removed>)

There are other options on the Sparkline tab that can help me when I have missing data. To show this, I will remove the value for March for Mary:

![](<Base64-Image-Removed>)

The Line Sparkline does not look so good now. If I select the Sparkline, and choose the ‘Edit Data’ option as I did earlier, there is an option for dealing with ‘Hidden & Empty Cells’:

![](<Base64-Image-Removed>)

This gives me a dialog:

![](<Base64-Image-Removed>)

I would like to ‘Connect the data point with line’, which is only available for Line Sparklines.

![](<Base64-Image-Removed>)

The Sparkline for Mary looks much better now. I’ll look at some more ways to use Sparklines next time.

That’s it for this week. Come back next week for more Charts and Dashboards tips.
