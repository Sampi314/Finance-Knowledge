Final Friday Fix: November 2025 Challenge
=========================================

__On the final Friday of each month, we set an Excel / Power Pivot / Power Query / Power BI problem for you to puzzle over for the weekend.  On the Monday, we publish a solution.  If you think there is an alternative answer, feel free to email us.  We’ll feel free to ignore you.__

**_The Challenge_**

After last month’s very basic attempt at a clock in a spreadsheet, we thought we would extend the idea this time.  This month’s challenge is to create a working clock in Excel like the following:

![](<Base64-Image-Removed>)

Your clock must:

*   display the input time using two \[2\] hands
*   have the hour hand as the shorter of the two hands pointing to the hour between numbers similar to how a normal analogue clock would work
*   have the minute hand as the longer of the two hands pointing exactly to the minute position
*   have the hour and minute hands rotate just like a real clock
*   have the numbers 1 to 12 arranged in a circular clock face. 

The data area of your spreadsheet should include a Time cell that is an input and is formatted as hh:mm:ss, _viz._

![](<Base64-Image-Removed>)

You can download the original question file [here](https://sumproduct.com/wp-content/uploads/2025/11/SP-Excel-Clock-Challenge.xlsx)
.

As always, there are some requirements:

*   the formula needs to be within just one \[1\] column (no “helper” cells)
*   you must ensure the solution is dynamic so that it updates when new placeholders or replacement values are added
*   no VBA or Power Query is allowed; this is purely a formula challenge.

__Sounds easy?  Then why not have a go?  We’ll publish one solution in Monday’s blog.  Have a great weekend in the meantime!__
