New Features for Excel February 2026
====================================

![](<Base64-Image-Removed>)

_Image: Vecteezy.com_

Updates are a bit thin on the ground this month.  And what there is seems AI-oriented, rather than Excel itself.  Are we starting to witness the death of Excel by 1,000 cuts?

The latest updates see the availability for Agent Mode in Excel expanded, as well that you can now query modern Excel workbooks (like .xlsx, .xlsb, .xlsm, .ods) stored locally on your device using Microsoft 365 Copilot Chat, on both Windows and Mac.

Given all the confusing ways you can access Copilot, the so-called “fragmentation” is being addressed.  The editing capabilities that App Skills provided will now be integrated into Copilot Chat and Agent Mode in Excel, which became Generally Available earlier this year.

Here’s the full list of both items this month:

**_Excel for Windows and Mac_**

*   Agent Mode in Excel expanded availability
*   Query your local Excel files with Copilot Chat.
