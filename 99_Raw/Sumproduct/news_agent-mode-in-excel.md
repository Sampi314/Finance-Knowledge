Agent Mode in Excel
===================

![](<Base64-Image-Removed>)

Agent mode is a new feature, currently exclusive to Excel for the web via the Excel Labs add-in, that allows you to work alongside Copilot whilst editing your workbook.

![](<Base64-Image-Removed>)

Let’s generate some sample data and see what Agent mode can do.

![](<Base64-Image-Removed>)

![](<Base64-Image-Removed>)

You may notice that Agent mode takes a considerable amount of time to perform some tasks, however you can still make changes to your workbook whilst it’s working away in the background and even check what Agent mode is up to:

![](<Base64-Image-Removed>)

In this example it took Agent mode 46 seconds of reasoning to create the dashboard:

![](<Base64-Image-Removed>)

As well as creating the dashboard, Agent mode provides a summary of what it has done and how it’s done it:

![](<Base64-Image-Removed>)

Whilst what Agent mode has created in this instance isn’t perfect, it’s a good start and we can always ask it to iterate on this and improve it:

![](<Base64-Image-Removed>)

It’s still not quite right, but you get the picture!  We can even use Agent mode to adjust the formatting and styles used:

![](<Base64-Image-Removed>)

That looks much better.  Interestingly, in performing this step it seems to have noticed that the x-axis of the “Sales by Year” chart was still incorrectly labelled despite me not mentioning it in the most recent prompt!  It appears as though Agent mode will perform additional validation during each step to try and ensure that nothing has been broken.

Creating dashboards is helpful and all, but what else can it do?

With a few prompts, iterations, and a lot of reasoning time (with no manual adjustments!) it managed to create this:

![](<Base64-Image-Removed>)

That’s much more inspirational (to me at least)!  I can certainly see the advantage.
