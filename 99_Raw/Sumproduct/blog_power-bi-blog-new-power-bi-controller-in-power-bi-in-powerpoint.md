Power BI Blog: New Power BI Controller in Power BI in PowerPoint
================================================================

_Welcome back to this week’s edition of the Power BI blog series._  _This week, we look at the new Power BI controller in Power BI in PowerPoint._

![](https://sumproduct.com/wp-content/uploads/2025/10/image-165.png)

You may now embed interactive reports and visuals into PowerPoint, supporting data-driven presentations across organisations.  Many of us have noticed there is a requirement for a centralised mechanism to manage and update several Power BI add-ins across multiple slides.  The Power BI Controller addresses this need by allowing you to execute bulk operations from a single interface.

The Power BI Controller is a task pane add-in that serves as a central command centre for all Power BI add-ins within a presentation.  To get this add-in go to ‘Add-ins’ in the PowerPoint Ribbon and search for ‘Microsoft Power BI Controller’.  Then, select ‘Add’ to insert the add-in onto your Ribbon.  Once you select it, a new panel will open with the list of commands that you can apply.  Pick the command you wish to run and click ‘Apply’.  This will trigger the discovery phase in which the controller will look for all Power BI content in the presentation and next the selected command will be executed on the discovered Power BI add-ins.
