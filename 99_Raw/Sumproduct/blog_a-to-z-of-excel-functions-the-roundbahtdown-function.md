_Welcome back to our regular A to Z of Excel Functions blog. Today we look at the **ROUNDBAHTDOWN** function._

**The ROUNDBAHTDOWN function**

![](<Base64-Image-Removed>)

The **ROUNDBAHTDOWN** function is an interesting function. It certainly exists:

![](<Base64-Image-Removed>)

However, Microsoft states that this function is “reserved for internal use” and belongs to a “Hidden” category of functions. Great.

Just in case you were wondering, it has the following syntax – not that it will help you much:

**ROUNDBAHTDOWN(number)**

The **ROUNDBAHTDOWN** function has the following arguments:

*   **number**: this is required.

Given that it always seems to generate an _#VALUE!_ error, there are no useful examples.
