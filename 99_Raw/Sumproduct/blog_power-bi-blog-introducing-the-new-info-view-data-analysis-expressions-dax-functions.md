Power BI Blog: Introducing The New INFO.VIEW Data Analysis eXpressions (DAX) Functions
======================================================================================

14 November 2024

_Welcome back to this week’s edition of the Power BI blog series. This week, we provide an overview of four new **INFO.VIEW DAX** functions._

DAX query view introduced new **DAX** functions to get metadata about your semantic model with the **INFO DAX** functions, and now four \[4\] of these functions are also available as **INFO.VIEW DAX** functions, which convert IDs to friendly names, and can be used in calculated tables of the semantic model in addition to being able to run them in DAX query view.

![](https://sumproduct.com/wp-content/uploads/2025/05/b2e335f2255450f9494221b0d3f3180e-1.jpg)

Adding them as calculated tables ensures your model self-documents and stays current with all your latest changes automatically. Not only can these be used in calculated tables, but they will show the name or value of a column previously only displaying an ID.

**INFO.VIEW.TABLES()** shows information about the tables in your model, including what storage mode each table is in. You can also quickly identify tables marked as a date table by the Data Category of Time.

![](https://sumproduct.com/wp-content/uploads/2025/05/462330cd6d01b97d7c20e2e9da9606e4-1.jpg)

![](https://sumproduct.com/wp-content/uploads/2025/05/d888e8075ffd2bacdfe907a704d6c6ed-1.jpg)

**INFO.VIEW.RELATIONSHIPS()** shows information about all the relationships in your model, including a relationship column giving a quick summary of the to and from columns with direction and cardinality.

![](https://sumproduct.com/wp-content/uploads/2025/05/e72daaca9c77dbed33e4deada8a59437-1.jpg)

![](https://sumproduct.com/wp-content/uploads/2025/05/cecce984139b540355029244685c1bff-1.jpg)

**INFO.VIEW.MEASURES()** shows information about the model measures, including if it’s in a valid or error state.

![](https://sumproduct.com/wp-content/uploads/2025/05/eb8f35fa2da8ce44b502a13a46d6bee4-1.jpg)

![](https://sumproduct.com/wp-content/uploads/2025/05/1ac2d20e372dd5b78c5be4d036e9ed33-1.jpg)

**INFO.VIEW.COLUMNS()** shows information about the columns, including data category and data type.

![](https://sumproduct.com/wp-content/uploads/2025/05/75cb860124058b97376616204636ab95-1.jpg)

![](https://sumproduct.com/wp-content/uploads/2025/05/c34cb9a1ec17cf423a78f94c7227e091-1.jpg)
