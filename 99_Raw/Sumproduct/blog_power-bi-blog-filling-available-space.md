Power BI Blog: Filling Available Space
======================================

_Welcome back to this week’s edition of the Power BI blog series._  _This week, we look at how Power BI will now automatically expand table columns to fill the available space._

![](https://sumproduct.com/wp-content/uploads/2025/10/image-3.jpg)

There is now a new option to table formatting that makes your visuals look cleaner and more balanced automatically.  With ‘Grow to fit’, any extra space in the visual is evenly distributed across existing columns when the total column width is smaller than the visual container.  No more manual resizing to fill the gap!

If your table isn’t using the full visual width, you can manually resize columns or use Auto-size width, but now there’s an improved solution.  For example, this table doesn’t fully use the available space:

![](https://sumproduct.com/wp-content/uploads/2025/10/image-163.png)

If you navigate to the Format pane under **Column headers -> Options**, toggle ‘Auto-size width’ to ‘On’ and set ‘Resize behavior’ _(sic)_ to ‘Grow to fit’,

![](https://sumproduct.com/wp-content/uploads/2025/10/image-164.png)

you will see that your table expands to make full use of the available space.  Indeed, the additional available space will be distributed across the columns:

![](https://sumproduct.com/wp-content/uploads/2025/10/image-4.jpg)

It’s a subtle upgrade that makes your data layouts cleaner, more balanced and easier to read.  You should note that this will not interfere with any adjustments you make to column widths: once a column width is manually adjusted, it will no longer be expanded automatically.
