Power Query Editor in Excel for Mac
===================================

1 September 2022

It’s great to announce that this feature is now available to Insiders using either the Current Channel Preview or Beta Channel, running Version 16.64 (Build 22072501) or later of Excel for Mac.

When Power Query was first released in Excel for Mac, there was hope that it wouldn’t take long for the features to mirror their Window counterparts. It’s taking time, but the capabilities are improving. Beginning with the ability to refresh data two years ago, importing data was introduced this time last year, so this does seem to be an annual event! As you may imagine, Mac users have desperately requested the ability to transform data using the Query Editor, which would provide users with the full Power Query experience in Excel for Mac.

Well… it’s getting there! You may now clean and shape your data with hundreds of transformations available in Power Query Editor in Excel for Mac.

To access, on the Data tab, click the Get Data (Power Query) button.

![](https://sumproduct.com/wp-content/uploads/2025/05/4326cdcbb6b79cf77834515f66e24bd3.jpg)

Click Launch Power Query Editor to open the Query Editor.

![](https://sumproduct.com/wp-content/uploads/2025/05/bca0c66164a9a8f64d8906c792fa2bde.jpg)

You can shape and transform your data using the Query Editor similarly to Excel for Windows. When you’re done, click the ‘Close & Load’ button on the Home tab.

![](https://sumproduct.com/wp-content/uploads/2025/05/a79a75284e8fdf396ccf0240da05912c.jpg)

The newly imported data appears in a new sheet.

Supported data sources include:

*   text, CSV, XLSX, XML and JSON files
*   SharePoint, SharePoint Lists, SharePoint Folders and OData
*   local tables, Tables and ranges
*   Microsoft SQL Server.

You may also access the Query Editor from the data import flow by clicking the Get Data (Power Query) button, choosing a data source, and clicking the Next button.
