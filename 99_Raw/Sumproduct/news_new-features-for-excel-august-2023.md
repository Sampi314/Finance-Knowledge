**_Excel for the web_**

*   Interactive PivotCharts
*   Simplified Sharing across Microsoft 365
*   Access and Explore Office Add-ins

**_Excel for Windows_**

*   Python in Excel (Insiders Beta)
*   Stale Value Formatting (Insiders Beta)
*   Monospaced Font in Formula bar (Insiders Beta)
*   Power BI Connected Tables
*   Improved Save As / Discard Dialog
*   Simplified Sharing across Microsoft 365
*   Access and Explore Office Add-Ins (Insiders)
