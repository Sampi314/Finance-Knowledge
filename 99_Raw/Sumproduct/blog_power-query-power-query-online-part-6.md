Power Query: Power Query Online – Part 6
========================================

13 September 2023

_Welcome to our Power Query blog. Today, I look at the help available in Power Query Online._

In the current series, I am looking at Power Query Online, which I have accessed from Power Apps:

![](https://sumproduct.com/wp-content/uploads/2025/05/7e3e74a265cf2d377c7a37ea40973f00.jpg)

[Last week](https://sumproduct.com/blog/power-query-power-query-online-part-5/)
, I looked at the shortcuts available in Power Query Online:

![](https://sumproduct.com/wp-content/uploads/2025/05/bb51f84330d96e071d2cd1ca24fee1a1.jpg)

In addition to the ‘Keyboard shortcuts’, there are other resources available on the ‘Help’ tab:

![](https://sumproduct.com/wp-content/uploads/2025/05/3008fb3e4710fb7616a86adf8e4a8562.jpg)

‘Power Query documentation’ takes me to the Microsoft website in a new window, so that I can browse the available pages for Power Query on the Power Platform, or download everything as a PDF:

![](https://sumproduct.com/wp-content/uploads/2025/05/1f3052976d6e1daff8baa6c0ee2c7246.jpg)

I will cover some of the topics here later in the series. Now, I looked at shortcuts last week, so the next option is ‘Support’:

![](https://sumproduct.com/wp-content/uploads/2025/05/92499753f63e637b11332fb0dfce6bd4.jpg)

This takes me to the Power Apps support page, where I can see if there are any issues, upgrade my support plan from the included ‘Subscription support’ and search for my particular problem:

![](https://sumproduct.com/wp-content/uploads/2025/05/58e7fccd0e308beccc15d23e68baaf7c.jpg)

I can also view ‘Frequently asked questions’:

![](https://sumproduct.com/wp-content/uploads/2025/05/0c483633d0e4212990785cfa15271542.jpg)

I can raise a request if I can’t find the answers I need:

![](https://sumproduct.com/wp-content/uploads/2025/05/4454ef07a40b505ebc7c8ede992bc394.jpg)

The other tabs allow me to further explore the Power Apps. Back in Power Query Online, the next option is ‘Power Query Website’:

![](https://sumproduct.com/wp-content/uploads/2025/05/c722ffe5e41ed923e4fb79229f2da893.jpg)

This takes me to the Microsoft pages that are about Power Query in all its flavours:

![](https://sumproduct.com/wp-content/uploads/2025/05/21bab8a5c99eec56c294ff9c26121921.jpg)

Moving on to the ‘Community’ section, we have the ‘Power Query Blog’:

![](<Base64-Image-Removed>)

Sadly for me, this doesn’t take you to the SumProduct blog; instead, you are taken to the Microsoft Power Query Team Blog. Below is the introduction page, which explains the purpose of the blog:

![](<Base64-Image-Removed>)

The next ‘Community’ option is the ‘Power Query Community’:

![](<Base64-Image-Removed>)

This takes you to the Power BI Community page. Yes, you read that correctly. It is, however, where the Power Query forum now resides, allowing you to see the questions and responses from other users of all abilities.

![](<Base64-Image-Removed>)

Finally, in the ‘Community’ section, we have ‘Submit an idea’:

![](<Base64-Image-Removed>)

This virtual suggestion box allows everyone to ask for changes to not only Power Query Online, but also the other Power Apps:

![](<Base64-Image-Removed>)

You may also vote for other suggestions. The suggestions are shown in ‘Most votes’ order by default, but you can use the filters and change the sort order to find any like-minded suggestions:

![](<Base64-Image-Removed>)

The last item on the ‘Help’ tab in Power Query Online, is the ‘Send feedback’ option:

![](<Base64-Image-Removed>)

This is a more basic and general response to the way Power Query Online is functioning:

![](<Base64-Image-Removed>)

The resources on the ‘Help’ tab are a welcome addition, and it would be good to see their desktop equivalents on Power Query desktop for Excel. Currently, we have a version of ‘Send feedback’:

![](<Base64-Image-Removed>)

It would however, be useful to have access to other online resources.

Come back next time for more ways to use Power Query!
