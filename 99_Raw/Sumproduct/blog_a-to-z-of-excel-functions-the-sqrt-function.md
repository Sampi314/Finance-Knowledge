A to Z of Excel Functions: The SQRT Function
============================================

_Welcome back to our regular A to Z of Excel Functions blog.  Today we look at the **SQRT** function._

**The SQRT function**

![](<Base64-Image-Removed>)

This function returns the positive square root of the number used as the argument.  The square root (![](<Base64-Image-Removed>)) of a number is the value which when multiplied by itself generates the number cited, _e.g._

> ![](<Base64-Image-Removed>) = 2 since 2 x 2 = 4

> ![](<Base64-Image-Removed>) = -2 since -2 x -2 = 4

> ![](<Base64-Image-Removed>) = ±3i since +3i x +3i = -3i x -3i = -9

The last example is known as a complex (imaginary) solution.  This function does not report such values, and the absolute value (_i.e._ non-negative) number should be used first and then multiplied by _i_.

The **SQRT** function employs the following syntax to operate:

> **SQRT(number)**

The **SQRT** function has the following argument:

*   **number:** this is required and represents the **number** for which you want the **positive** square root.

It should be further noted that:

*   if the **number** is negative, **SQRT** returns the _#NUM!_ error.
