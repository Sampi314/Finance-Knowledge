Charts and Dashboards: Custom Axes – Part 2

Charts and Dashboards: Custom Axes – Part 2
===========================================

18 November 2022

_Welcome back to our Charts and Dashboards blog series._ _This week, we’re going to continue looking at one way to customise our axes as we format the axes we created last week._

Consider the following data:

![](https://sumproduct.com/wp-content/uploads/2025/05/a0bb6413c15f3bd0e16016da6cb3f643.jpg)

These are **X** and **Y** values to be plotted on a Scatter chart. Sounds simple enough, right?

![](https://sumproduct.com/wp-content/uploads/2025/05/33cf748959b8d11939a99f4afd61c055.jpg)

There we have it. But already we can see some issues. It’s hard to discern the exact value of each point as they sit between the gridlines. Let’s create custom axis to make these values easier to determine.

[Last week](https://sumproduct.com/blog/charts-and-dashboards-custom-axes-part-1/)
, we created two new data series, one to act as the **x**\-axis of our chart and one to act as the **y**\-axis.

![](https://sumproduct.com/wp-content/uploads/2025/05/47ae82b9118dd286d6cdfc9bb22346ad.jpg)

We plotted them on our chart:

![](https://sumproduct.com/wp-content/uploads/2025/05/4c54c274de92eccf430074bff1fced7b.jpg)

After changing the colour and adding data labels, we were left with an issue, which we will solve this week:

![](https://sumproduct.com/wp-content/uploads/2025/05/2ead861873ca0699c0d5bb7190c8bb49.jpg)

When we have added the data labels (this time below the data series) they are all displaying zero \[0\]. This is because the data labels are displaying the **y**\-values. We can format them to amend this:

![](https://sumproduct.com/wp-content/uploads/2025/05/5620b2b515bd07977e066d14ccad39ce.jpg)

We want to remove the existing gridlines and hide the numbers on the default axis. Removing the gridlines is as simple as clicking them and pressing the **DELETE** key, but hiding the numbers will require us to adjust the number formatting:

![](https://sumproduct.com/wp-content/uploads/2025/05/5b02d231b190db043c5a2af450c38714.jpg)

This will hide the numbers, whilst ensuring that the data labels remain aligned. Finally, having removed the default gridlines, we want to add custom gridlines that align with our custom axis. To do this, we will use ‘Error Bars’. We can add ‘Error Bars’ that we can customise to our **x**\-axis data by selecting ‘More Options…’:

![](https://sumproduct.com/wp-content/uploads/2025/05/fe97a7cfcad49554bbad3189855987fd.jpg)

This has added error bars in both the **x** and **y**\-directions:

![](<Base64-Image-Removed>)

Next time, we will format the ‘Error Bars’ to create out custom gridlines.

That’s it for this week. Come back next week for more Charts and Dashboards tips.
