Final Friday Fix: December 2019 Challenge
=========================================

27 December 2019

_On the final Friday of each month, we’re going to set an Excel challenge for you to puzzle over so that you can get your “Excel fix”. Challenge your office colleagues to see who can solve the puzzle quickest (if you’re not already on Christmas break). There’s no prizes at this stage, you’re playing for bragging rights only (or first pick of the boxes under the tree)!_

This month’s Final Friday Fix is about creating bullet charts in Excel. Imagine that we just received the data for budgeted costs and actual costs for three divisions from 2014 to 2018. In this challenge, we want to create a single chart that has multiple bullet charts within it. It will also display the different target ranges for each period for each data series.

Considering that this is a very niche chart with no globally agreed upon name, for the purpose of this challenge we are going to call it the Bullet Time Series chart (patent pending!):

![](<Base64-Image-Removed>)

No add-ins have been downloaded: this was created just using Excel.

**The Challenge**

Create a single chart in Excel that looks like the Bullet Time Series chart _(pictured above)_. Here is the sample [file](https://sumproduct.com/assets/blog-pictures/2019/challenges-fff-mmm/dec-2019-fff/final-friday-fix-dec-2019-starter-file.xlsm)
 that you can use.

_Sound easy? Have a go. We’ll publish one solution in Monday’s blog. Have a great weekend!_
