# Productivity pack update

**Source:** https://www.financialmodellinghandbook.org/productivity-pack-updates

---

Hi there. It's been a little while since I posted here. Life hey.

We have been working hard to update the Productivity Pack macros. The latest version (34c) is [available here.](https://www.financialmodellinghandbook.org/financial-modelling-productivity-pack/)

We are seeing more and more conflicts creeping up when the macros are used with different configurations of Excel version, operating system, and users' existing add-ins. We are, therefore, going to gauge what issues come up with this updated version and may share a trimmed-back version with just core functionality that uses simpler code and will cause fewer conflicts.

So please do let us know how you get on with this version!

Now, I'm going to press on with the PF book!

Thanks again for following.

Best

Kenny
