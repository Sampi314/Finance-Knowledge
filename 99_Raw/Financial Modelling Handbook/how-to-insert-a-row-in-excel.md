# Q&A: How to insert a copied row in Excel

**Source:** https://www.financialmodellinghandbook.org/how-to-insert-a-row-in-excel

---

Hi Angus.

Once you have copied (or cut) the row that you want to insert, select the row below where you want to put it. Then use Control+plus ("+") to insert the copied row.

Note that the plus sign is often on the shift position of "=". If this is true then the keystroke will be Ctrl+Shift+=. That depends on your keyboard set up though!

Hope this answers your question.
