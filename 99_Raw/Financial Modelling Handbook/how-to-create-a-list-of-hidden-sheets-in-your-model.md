# How to create a list of hidden sheets in your model

**Source:** https://www.financialmodellinghandbook.org/how-to-create-a-list-of-hidden-sheets-in-your-model

---

We've updated the Used Ranges report from the [Productivity Pack](https://www.financialmodellinghandbook.org/financial-modelling-productivity-pack/)
 to include and highlight hidden sheets.

You can access the report through the Model build utility dialogue (Ctrl+Shift+6). Press R to run the Used Ranges report.

The report lists every sheet in the model, and the cell ranges that Excel thinks are being used.

Any hidden sheets in your model are included in the report:

![](https://www.financialmodellinghandbook.org/content/images/2022/11/1-13.jpg)

See also how to use this report to tidy up the Used Ranges in your model.
