# Project Finance Modelling Handbook

**Source:** https://www.financialmodellinghandbook.org/project-finance-modelling-handbook-contents

---

I'm writing my next book, the Project Finance Modelling Handbook. As I did with the Financial Modelling Handbook, I'm publishing drafts of each chapter online as I write so that I can get as much feedback and input from fellow professionals as possible.

The Project Finance Modelling Handbook will build on the core model structuring and model-build skills shared in the Financial Modelling Handbook.

I look forward to your comments.

_Last updated: October 1st 2024_
