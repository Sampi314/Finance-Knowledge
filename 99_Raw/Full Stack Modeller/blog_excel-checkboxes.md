What are Checkboxes in Excel?
-----------------------------

![Checkboxes in Excel](https://www.fullstackmodeller.com/hs-fs/hubfs/Capture.jpg?width=1677&height=966&name=Capture.jpg)

Checkboxes are boxes that you can insert directly into a cell in Excel and then tick or untick using your mouse.

A ticked box has the value TRUE.

An unticked box has the value FALSE.

How to use Checkboxes in Excel
------------------------------

![How to use Checkboxes in Excel](https://www.fullstackmodeller.com/hs-fs/hubfs/CheckboxesVideo1.gif?width=1779&height=969&name=CheckboxesVideo1.gif)

Inserting Checkboxes into your Excel spreadsheet is now incredibly simple using the recently released Checkbox feature.

Simply select the cell, or cells, that you would like to add a checkbox into, go to the Insert tab on the ribbon in your Excel spreadsheet and click on Checkbox.

A checkbox will then be added into the selected cell(s).

Checkboxes have the cell value of either TRUE (ticked) or FALSE (unticked). You can reference this TRUE or FALSE value in your formulas.

For example, you can use the [IF function](https://www.fullstackmodeller.com/blog/how-to-use-the-if-function)
 to reference the TRUE or FALSE in the cell and return "Yes" or "No".

The example shown above this uses the formula:

\=IF(C5 , "Yes", "No")

In plain English, this is: If the value of cell C5 is "TRUE" then return "Yes" otherwise return "No".

How to use conditional formatting with Checkboxes in Excel
----------------------------------------------------------

![CheckboxesVideo2](https://www.fullstackmodeller.com/hs-fs/hubfs/CheckboxesVideo2.gif?width=1737&height=945&name=CheckboxesVideo2.gif)

As the value of a checkbox in Excel is either TRUE or FALSE we can easily apply conditional formatting over the checkboxes to change the background colour and strikethrough based on the value being "TRUE".

How to use Checkboxes with Excel Tables and Dynamic Arrays
----------------------------------------------------------

![Using Excel Checkboxes with Excel Tabels and Dynamic Arrays](https://www.fullstackmodeller.com/hs-fs/hubfs/CheckboxesVideo3.gif?width=1785&height=972&name=CheckboxesVideo3.gif)

We can also put checkboxes into [Excel Tables](https://www.fullstackmodeller.com/blog/use-excel-tables)
 and reference them using Dynamic Array formulas such as FILTER.

Note that when the Excel Table extends the Checkboxes also automatically extend down the rows.

Read more about the Excel Checkboxes feature on the Microsoft support page [HERE](https://techcommunity.microsoft.com/t5/excel-blog/introducing-checkboxes-in-excel/ba-p/4173561)
