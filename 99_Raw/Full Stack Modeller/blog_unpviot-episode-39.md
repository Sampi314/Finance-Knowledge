New features in Excel and PowerBI
---------------------------------

The Unpivot team have a chat through some recent new features in Excel and PowerBI:    

*   •    Visual Calcs in Dax (PowerBI) – User Defined Functions (UDFs) with Scalable Vector Graphics (SVGs)
*   •    Excel Agent mode will now be linked to CoPilot so no need for Labs add-in
*   •    Formula by example now in Excel desktop
*   •    Bing Map being deprecated in PowerBI – need to turn them into Azure maps
*   •    PowerBI controller in PowerPoint
*   •    Spill feature for Pivot Tables to replace the annoying overlap message (in beta insider version)

Question of the Week:   What are the most common data problems and how can we solve them
----------------------------------------------------------------------------------------

*   •   Free form text from an input form
*   •   Date formatting
*   •   Wrong, missing data, needs to be merged / split
*   •   Finding the data in the first place
*   •   Missing dates / other values
*   •   Lack of or lack of clarity of data definitions across departments
*   •   Disparate, messy data

**Some options for** **fixing them:** 

*   •  AI, Python & R
*   •  =COPILOT()
*   •  PowerQuery
*   •  Push back up the data flow to resolve at source and user behaviours etc
*   •  Data Validation at input
