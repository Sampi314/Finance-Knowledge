Summary
-------

The Excel COTH function returns the cotangent of an angle.

Purpose 
--------

Get the hyperbolic cotangent of an angle

Return value 
-------------

The hyperbolic cotangent

Syntax
------

    =COTH(number)

*   _number_ - The hyperbolic angle.

Using the COTH function 
------------------------

The Excel COTH function returns the hyperbolic cotangent of a [hyperbolic angle](https://exceljet.net/glossary/hyperbolic-angle)
. Given 1.00 as input, the function returns 1.313035285 as output.

    =COTH(1) // returns 1.313035285
    

### Explanation

The hyperbolic cotangent is the reciprocal of the [TANH](https://exceljet.net/functions/tanh-function)
 function.

    =1/TANH(x) // equivalent to COTH(x)
    

The plot below shows the COTH function's output in Excel.

![Hyperbolic cotangent plot.](https://exceljet.net/sites/default/files/images/functions/inline/HyperbolicCotangentPlot.png "Hyperbolic cotangent plot.")

**Notes:**

*   Returns a #DIV/0! error when given zero as input.
